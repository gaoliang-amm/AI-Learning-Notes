# 191. 位1的个数

## 🎯 人话翻译

> 给你一个整数，数一数它的二进制表示里面有几个 1（也叫「汉明重量」）。

## 💡 核心思想

> **核心技巧：`n & (n - 1)` 可以消去 n 的二进制表示中最右边的 1。**
>
> 原理：`n - 1` 会把 n 最右边的 1 变成 0，这个 1 右边的所有 0 变成 1。然后 `n & (n - 1)` 就把最右边那个 1 以及它右边所有位都清零了。
>
> 每次执行 `n &= (n - 1)` 消去一个 1，计数加 1，直到 n 变成 0。

用到：[[位运算]]

## 📝 解题步骤

1. **初始化计数器** — `count = 0`
2. **循环直到 n 为 0** — 当 n > 0 时继续
3. **消去最低位的 1** — `n &= (n - 1)`，每次操作消去一个 1
4. **计数加 1** — `count += 1`
5. **返回 count** — 循环次数就是 1 的个数

## 🔢 手动走一遍

输入: `n = 11`，二进制为 `1011`

| 轮次 | n 的二进制 | n 十进制 | n - 1 的二进制 | n & (n-1) 的二进制 | n 更新后 | count | 说明 |
|------|-----------|---------|---------------|-------------------|---------|-------|------|
| 初始 | 1011 | 11 | - | - | 11 | 0 | 初始状态 |
| 1 | 1011 | 11 | 1010 | 1010 | 10 | 1 | 消去最右边的 1（位置 0） |
| 2 | 1010 | 10 | 1001 | 1000 | 8 | 2 | 消去最右边的 1（位置 1） |
| 3 | 1000 | 8 | 0111 | 0000 | 0 | 3 | 消去最右边的 1（位置 3） |

最终 `n = 0`，循环结束，`count = 3`。二进制 `1011` 中确实有 3 个 1。

## 💻 解法

> 利用 `n & (n - 1)` 每次都消去最低位的 1，循环次数等于 1 的个数。

```python
class Solution:
    def hammingWeight(self, n: int) -> int:
        """
        计算无符号整数 n 的二进制表示中 1 的个数。
        使用 n & (n-1) 技巧，每次消去最低位的 1。
        时间复杂度: O(k)，k 为 1 的个数
        空间复杂度: O(1)
        """
        count = 0
        # Python 整数无限大，但题目保证输入为 32 位无符号整数
        while n:
            n &= (n - 1)  # 消除最低位的 1
            count += 1
        return count
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(k) | k 为二进制中 1 的个数，最多 32，最少 0 |
| 空间 | O(1) | 只使用了计数器变量 |

## ⚠ 易错点

- [ ] 不能用 `while n > 0` 处理负数的场景（Python 中负数二进制补码表示 `n - 1` 可能不会终止），但题目保证输入是无符号整数
- [ ] 逐位检查法（`for i in range(32): count += (n >> i) & 1`）能过但不够优雅，面试官想看 `n & (n - 1)` 这个技巧
- [ ] `n & (n - 1)` 在 n = 0 时不会出错（while 条件为假，直接跳过），但不要手动对 0 做 `0 & (-1)` 这种无意义的操作
- [ ] 这个技巧在 [[201. 数字范围按位与]] 和判定一个数是否为 2 的幂（`n & (n - 1) == 0`）中也会用到
- [ ] 注意 `&` 是按位与，`and` 是逻辑与，不要混淆

## 🎤 面试怎么说

```text
这道题用一个非常经典的位运算技巧来解决。

核心操作是 n & (n - 1)，它可以把 n 的二进制表示中最右边的 1 消去。

原理是这样的：n - 1 会把 n 最右边的 1 变成 0，
这个 1 右边的所有 0 都变成 1。
然后 n & (n - 1) 就能精确地把最右边那个 1 清零，
而不影响其他位。

每次执行这个操作，1 的个数就少一个，计数器加一。
当 n 变成 0 时，计数器的值就是答案。

时间复杂度 O(k)，k 是 1 的个数，最多 32。
空间复杂度 O(1)。

这个技巧非常实用，在判断 2 的幂、计算数字范围按位与等问题中都会用到。
```
