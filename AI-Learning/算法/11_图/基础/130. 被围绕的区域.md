# 130. 被围绕的区域

## 🎯 人话翻译

> 二维网格中有 'X' 和 'O'，把被 'X' 四面包围的 'O' 区域全部变成 'X'。但位于边界上的 'O' 区域不算被围绕，保持 'O'。

## 💡 核心思想

> **逆向思维：与其找被围绕的 O，不如先标记不能被围绕的 O（即与边界相连的 O），剩下的 O 就是要翻转的。**
>
> 从四条边的所有 'O' 出发做 DFS/BFS，标记所有与边界相连的 'O' 为临时字符（如 '#'）。标记完成后：
> - '#' -> 'O'（边界可达，恢复）
> - 'O' -> 'X'（被包围，翻转）
>
> 这个思路比正面判断「是否被围绕」简单得多。

用到：[[图]]、[[DFS]]、[[BFS]]、[[Flood Fill]]

## 📝 解题步骤

1. **边界检查** — 空网格直接返回
2. **从四边界的 'O' 出发做 DFS** — 遍历第一行、最后一行、第一列、最后一列的每个格子，如果是 'O' 就启动 DFS
3. **DFS 标记** — DFS 将当前格子标记为 '#'，然后向四个方向递归（只走值为 'O' 的格子）
4. **遍历整个网格，转换字符** — '#' 恢复为 'O'，'O' 翻转为 'X'
5. **原地修改，无需返回**

## 🔢 手动走一遍

输入:
```
X X X X
X O O X
X X O X
X O X X
```

### 步骤一：从边界 DFS

边界  'O' 检查：
- 上边：全部为 'X'，无
- 下边：全部为 'X'，无
- 左边：全部为 'X'，无
- 右边：全部为 'X'，无

边界没有 'O'，所以没有 'O' 被标记为 '#'。

实际上：(3, 1) = 'O'，它被 'X' 完全包围。

### 步骤二：遍历转换

| 位置 | 原值 | 转换后 | 原因 |
|------|------|--------|------|
| (1,1) | O | X | DFS 没走到，被包围 |
| (1,2) | O | X | DFS 没走到，被包围 |
| (2,2) | O | X | DFS 没走到，被包围 |
| (3,1) | O | X | DFS 没走到，被包围 |

最终:
```
X X X X
X X X X
X X X X
X X X X
```

---

另一个例子（边界有 'O'）:
```
X X X X
X O O X
X O O X
X O X X
```

### 步骤一：从边界 DFS

边界检查：
- 最后一行 (3,1) = 'O'，启动 DFS
  - 标记 (3,1) 为 '#'
  - 向上 (2,1) = 'O' -> 标记 '#'，继续
  - 向上 (1,1) = 'O' -> 标记 '#'，向右 (1,2) = 'O' -> 标记 '#'
  - 向下 (2,2) = 'O' -> 标记 '#'

标记后的网格：
```
X X X X
X # # X
X # # X
X # X X
```

### 步骤二：转换
```
X X X X
X O O X
X O O X
X O X X
```
'#' 恢复为 'O'，没有剩余的 'O'（都被标记了）。

## 💻 解法

> 从四边界的 'O' 出发 DFS 标记，未被标记的 'O' 就是被围绕的，翻转为 'X'。

```python
from typing import List


class Solution:
    def solve(self, board: List[List[str]]) -> None:
        # 时间复杂度 O(m*n)，空间复杂度 O(m*n)

        if not board or not board[0]:
            return

        m, n = len(board), len(board[0])

        def dfs(i, j):
            # 边界检查或不是 'O' 则返回
            if i < 0 or i >= m or j < 0 or j >= n or board[i][j] != 'O':
                return

            # 标记为临时字符
            board[i][j] = '#'

            # 四个方向继续搜索
            dfs(i + 1, j)
            dfs(i - 1, j)
            dfs(i, j + 1)
            dfs(i, j - 1)

        # 1. 从四条边界的 'O' 开始 DFS，标记所有边界可达的 'O'
        for i in range(m):
            dfs(i, 0)  # 左边界
            dfs(i, n - 1)  # 右边界

        for j in range(n):
            dfs(0, j)  # 上边界
            dfs(m - 1, j)  # 下边界

        # 2. 遍历整个网格，转换字符
        for i in range(m):
            for j in range(n):
                if board[i][j] == '#':
                    board[i][j] = 'O'  # 边界可达的 O，恢复
                elif board[i][j] == 'O':
                    board[i][j] = 'X'  # 被包围的 O，翻转
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(m x n) | 每个格子最多被访问一次 |
| 空间 | O(m x n) | 递归栈最坏情况（全部是 O 连成一片时，递归深度可达 m*n） |

## ⚠ 易错点

- [ ] **思路不要正向**：不要试图判断每个 'O' 是否被围绕（非常复杂），逆向从边界标记未被围绕的 'O' 才是正解
- [ ] **边界 DFS 的启动位置**：四条边都要检查，不能漏掉。注意四角的格子会被两条边各检查一次（没关系，DFS 有判断不会重复标记）
- [ ] **临时标记的使用**：必须用临时字符（如 '#'）区分「已访问的边界 O」和「未访问的 O」。直接改成 'O' 会丢失信息
- [ ] **DFS 的边界检查**：`i < 0 or i >= m or j < 0 or j >= n` 四个条件缺一不可
- [ ] 如果面试官要求避免递归栈溢出，可以把 DFS 改成 BFS（用队列），空间更可控

与其找被围绕的 O，不如先标记不能被围绕的 O。
