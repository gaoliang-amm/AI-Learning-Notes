# 149. 直线上最多的点数

## 🎯 人话翻译

> 给一堆平面上的点，问最多有多少个点在同一条直线上。

## 💡 核心思想

> **固定一个基准点，计算其他所有点与它的斜率，相同斜率的点就在同一条直线上。**
>
> 对每个点 i，以它作为基准，遍历所有 `j > i` 的点：
> 1. 计算两点间的 `dx` 和 `dy`
> 2. 约分斜率：用 `gcd(dx, dy)` 化简，并统一正负号（保证 `(2,4)` 和 `(1,2)` 算同一斜率）
> 3. 用哈希表统计每种斜率的出现次数
> 4. 处理重复点（`dx == 0 and dy == 0`）：重复点要单独计数，它们在任何直线上
> 5. 处理垂直线（`dx == 0`）：斜率表示为 `(0, 1)`
>
> 答案 = `max(每种斜率的点数) + 重复点数`

用到：[[数学]]、[[哈希表]]、[[最大公约数]]

## 📝 解题步骤

1. **边界处理** — 点数 < 3 直接返回 n
2. **遍历每个点 i 作为基准**：
   - 初始化 `slope_count = defaultdict(int)` 记录斜率频率
   - `duplicate_count = 1`（包含自己）
3. **遍历每个 j > i**：
   - 计算 `dx = points[j][0] - points[i][0]`，`dy = points[j][1] - points[i][1]`
   - 如果 `dx == 0 and dy == 0`：重复点，`duplicate_count += 1`
   - 如果 `dx == 0`：垂直线，斜率记为 `(0, 1)`
   - 否则：用 `gcd` 约分，确保 dx 为正（统一方向），`slope = (dx//gcd, dy//gcd)`
4. **统计** — `slope_count[slope] += 1`
5. **更新最大值** — `max_on_line = max(max_on_line, max(slope_count.values()) + duplicate_count)`，注意 `slope_count` 可能为空（全重复点）
6. **返回全局最大**

## 🔢 手动走一遍

输入: `points = [[1,1], [2,2], [3,3], [4,1], [5,1]]`

5 个点示意图：
```
(1,1) (2,2) (3,3) -- 都在 y=x 上
(4,1) (5,1) -- 都在 y=1 上
```

### 以 (1,1) 为基准

| j | 点 | dx | dy | gcd(2,2)=2 | slope 约分 | slope_count |
|---|-----|----|----|-----------|-----------|-------------|
| 1 | (2,2) | 1 | 1 | gcd(1,1)=1 | (1,1) | {(1,1): 1} |
| 2 | (3,3) | 2 | 2 | gcd(2,2)=2 | (1,1) | {(1,1): 2} |
| 3 | (4,1) | 3 | 0 | - | (1,0) | {(1,1): 2, (1,0): 1} |
| 4 | (5,1) | 4 | 0 | gcd(4,0)=4 | (1,0) | {(1,1): 2, (1,0): 2} |

`max_on_line = max(2, 2) + 1 = 3`（y=x 上有 3 个点：1,1 + 2 个同斜率）

### 以 (2,2) 为基准

| j | 点 | dx | dy | slope | slope_count |
|---|-----|----|----|-------|-------------|
| 2 | (3,3) | 1 | 1 | (1,1) | {(1,1): 1} |
| 3 | (4,1) | 2 | -1 | (2,-1) | {(1,1): 1, (2,-1): 1} |
| 4 | (5,1) | 3 | -1 | (3,-1) | {(1,1): 1, (2,-1): 1, (3,-1): 1} |

`max_on_line = 2`。

最终答案: `max(3, 2, 2, 2, 2) = 3`。

## 💻 解法

> 固定每个点，计算与其他点的斜率并用 gcd 约分，哈希表统计同斜率点数。

```python
from typing import List
from collections import defaultdict
import math


class Solution:
    def maxPoints(self, points: List[List[int]]) -> int:
        """
        计算二维平面上最多有多少个点共线。

        算法核心：固定一个点，计算它与其他所有点连线的斜率，并用哈希表统计斜率出现次数。
        时间复杂度: O(n^2)
        空间复杂度: O(n)
        """
        n = len(points)
        # 点的数量少于 3 时，直接返回点的个数即可
        if n < 3:
            return n

        max_points_on_line = 1

        for i in range(n):
            # 用一个字典来记录从 points[i] 出发的所有斜率及其出现次数
            slope_count = defaultdict(int)
            # 记录与 points[i] 重合的点的数量（包括自身）
            duplicate_count = 1

            for j in range(i + 1, n):
                # 计算两个点之间的坐标差
                dx = points[j][0] - points[i][0]
                dy = points[j][1] - points[i][1]

                # 1. 处理重复点
                if dx == 0 and dy == 0:
                    duplicate_count += 1
                    continue

                # 2. 化简斜率，处理垂直线（dx == 0）的情况
                if dx == 0:
                    # 垂直线，斜率可表示为 (0, 1)
                    slope = (0, 1)
                else:
                    # 普通斜率，约分并统一符号
                    # 确保 dx 为正数，以便唯一表示一条直线
                    if dx < 0:
                        dx = -dx
                        dy = -dy
                    gcd = math.gcd(dx, dy)
                    slope = (dx // gcd, dy // gcd)

                slope_count[slope] += 1

            # 3. 更新全局最大值
            # 当前 base_point i 能达到的最大点数 = 斜率的最大出现次数 + 重复点的数量
            if slope_count:
                max_for_current_point = max(slope_count.values()) + duplicate_count
            else:
                max_for_current_point = duplicate_count

            max_points_on_line = max(max_points_on_line, max_for_current_point)

        return max_points_on_line
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(n^2) | 每个点都要与后面所有点计算斜率，共 n(n-1)/2 次 |
| 空间 | O(n) | 哈希表最多存储 n-1 种斜率（每个其他点一个不同斜率） |

## ⚠ 易错点

- [ ] **斜率的唯一表示**：必须用 gcd 约分 + 统一正负号。`(2,4)` 和 `(1,2)` 是同一斜率；`(-1,2)` 和 `(1,-2)` 也是同一斜率（要确保 dx 为正）
- [ ] **垂直线处理**：`dx == 0` 时斜率为无穷大，不能用除法，需要特殊处理。可以表示为 `(0, 1)`
- [ ] **重复点处理**：`dx == 0 and dy == 0` 表示同一个点，不能算入斜率统计，要单独计数。它们在任何通过基准点的直线上
- [ ] **空 slope_count**：如果所有点都重合（全重复），`slope_count` 为空，此时 `max_for_current_point = duplicate_count`
- [ ] **Python 的 gcd 支持负数**：`math.gcd(-4, 6) = 2`，但为确保一致性，最好先保证 dx 为正再调用 gcd

这道题的核心是对每个点，计算它与其他所有点的斜率，
相同斜率的点就在同一条直线上。

算法流程：
1. 固定一个点 i 作为基准
2. 遍历所有 j > i 的点，计算 (dx, dy)
3. 用 gcd 约分斜率，确保唯一表示
4. 用哈希表统计每种斜率的点数
