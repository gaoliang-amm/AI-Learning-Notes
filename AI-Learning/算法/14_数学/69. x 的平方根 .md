# 69. x 的平方根

## 🎯 人话翻译

> 给你一个非负整数 x，求它的平方根（只保留整数部分，小数舍去）。不能用内置的 `sqrt` 函数。

## 💡 核心思想

> **二分查找：在 [1, x] 范围内搜索最大的整数 `ans` 使得 `ans * ans <= x`。**
>
> 对于 `mid`：
> - 如果 `mid * mid == x`，直接返回 mid
> - 如果 `mid * mid < x`，mid 是一个候选答案，记录后向右搜索
> - 如果 `mid * mid > x`，向左搜索
>
> 另外可以用**牛顿迭代法**：`x_{n+1} = (x_n + x / x_n) / 2`，收敛速度更快，但二分法更容易在面试中写对。

用到：[[数学]]、[[二分查找]]、[[牛顿迭代]]

## 📝 解题步骤

### 二分查找

1. **特例** — `x == 0` 或 `x == 1` 直接返回 x
2. **二分** — `left = 1, right = x, res = 0`
3. **循环** — `while left <= right`：
   - `mid = left + (right - left) // 2`（防溢出写法）
   - 计算 `square = mid * mid`
   - `square == x`：返回 mid
   - `square < x`：记录 `res = mid`，`left = mid + 1`
   - `square > x`：`right = mid - 1`
4. **返回 res** — 循环结束后 res 是最大满足 `res^2 <= x` 的整数

## 🔢 手动走一遍

输入: `x = 8`

| 轮次 | left | right | mid | mid^2 | 与 x 比较 | 操作 |
|------|------|-------|-----|-------|----------|------|
| 初始 | 1 | 8 | - | - | - | res = 0 |
| 1 | 1 | 8 | 4 | 16 | 16 > 8 | right = 3 |
| 2 | 1 | 3 | 2 | 4 | 4 < 8 | res = 2, left = 3 |
| 3 | 3 | 3 | 3 | 9 | 9 > 8 | right = 2 |
| 4 | 3 | 2 | - | - | - | left > right，结束 |

res = 2。验证：`sqrt(8) ≈ 2.828`，整数部分为 2。

输入: `x = 4`

| 轮次 | left | right | mid | mid^2 | 操作 |
|------|------|-------|-----|-------|------|
| 1 | 1 | 4 | 2 | 4 | 4 == 4，直接返回 2 |

## 💻 解法

> 二分查找：搜索最大的整数 ans 使得 ans^2 <= x。

```python
class Solution:
    def mySqrt(self, x: int) -> int:
        """
        计算非负整数x的算术平方根，仅返回整数部分（舍去小数）
        最优解法：二分查找，时间O(log x)，空间O(1)
        """
        # 特殊情况：x=0或x=1时，平方根就是本身，直接返回
        if x == 0 or x == 1:
            return x

        # 二分查找的左右边界：平方根一定在[1, x]之间
        left = 1
        right = x

        # 最终要返回的结果，初始化为0
        res = 0

        # 当左边界 <= 右边界时，持续二分
        while left <= right:
            # 计算中间值mid，用left + (right-left)//2 防止整数溢出（面试必写写法）
            mid = left + (right - left) // 2

            # 计算mid的平方，和x比较
            square = mid * mid

            if square == x:
                # 刚好找到平方根，直接返回mid
                return mid
            elif square < x:
                # mid的平方小于x，说明mid是一个候选答案
                # 继续向右找更大的数，看有没有更接近x的平方根
                res = mid
                left = mid + 1
            else:
                # mid的平方大于x，说明mid太大了，向左缩小范围
                right = mid - 1

        # 循环结束，res就是最大的满足mid^2 <= x的整数，即x的平方根的整数部分
        return res
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(log x) | 二分查找每次将搜索范围减半 |
| 空间 | O(1) | 只用了常数个变量 |

## ⚠ 易错点

- [ ] **mid 的计算**：必须用 `left + (right - left) // 2`，防止 `(left + right)` 溢出（在 C++/Java 中很重要，Python 不需要但面试中应该展示这个意识）
- [ ] **square 溢出**：`mid * mid` 在 `x` 接近 int 最大值时可能溢出（Python 不会，但 C++/Java 需要用 `long long`）
- [ ] **边界条件**：`x == 0` 和 `x == 1` 要提前处理，否则二分逻辑对于 x=0 会出错
- [ ] **right 的初始值**：可以优化为 `right = x // 2 + 1`（因为当 x >= 2 时，sqrt(x) <= x/2 + 1），能更快收敛
- [ ] 不要用 `mid * mid == x` 来精确判断（用 `mid == x // mid` 可避免溢出），但在 Python 中无所谓
