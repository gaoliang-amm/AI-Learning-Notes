# 50. Pow(x, n)

## 🎯 人话翻译

> 自己实现幂函数 pow(x, n)，计算 x 的 n 次方（n 可以是负数）。不能用内置的 `**` 或 `math.pow`。

## 💡 核心思想

> **快速幂算法（Binary Exponentiation / Fast Power）：利用二分思想，将 O(n) 的朴素乘法优化为 O(log n)。**
>
> 核心原理：`x^n = (x^2)^(n/2)` 当 n 为偶数时；`x^n = x * (x^2)^((n-1)/2)` 当 n 为奇数时。
>
> 迭代实现：每次判断 n 的最低位是否为 1（`n & 1`），如果是就把当前的 x 乘入结果。然后 x 平方，n 右移一位（`n >>= 1`），直到 n 为 0。
>
> 负指数处理：`x^(-n) = (1/x)^n`，所以把 x 取倒数，n 变正数即可。

用到：[[数学]]、[[快速幂]]、[[位运算]]、[[二分]]

## 📝 解题步骤

1. **处理特例** — `n == 0` 返回 1（任何数的 0 次方为 1）
2. **处理负指数** — 如果 `n < 0`：`x = 1 / x`，`n = -n`
3. **快速幂迭代** — 初始化 `res = 1`
   - 当 `n > 0` 时循环：
   - 如果 `n & 1 == 1`（n 是奇数），`res *= x`（把当前底数乘入结果）
   - `x *= x`（底数平方）
   - `n >>= 1`（指数右移，相当于 n //= 2）
4. **返回 res**

## 🔢 手动走一遍

输入: `x = 2.0, n = 10`

用二进制快速幂：`2^10 = 2^(1010b) = 2^8 * 2^2 = 256 * 4 = 1024`

### 迭代过程

| 轮次 | n (二进制) | n 的十进制 | n & 1? | res | x (平方后) | 说明 |
|------|-----------|-----------|--------|-----|-----------|------|
| 初始 | 1010 | 10 | - | 1 | 2 | 初始状态 |
| 1 | 1010 | 10 | 0 (偶数) | 1 | 4 | x = 2^2 = 4, n = 5 |
| 2 | 101 | 5 | 1 (奇数) | 4 | 16 | res *= 4, x = 4^2 = 16, n = 2 |
| 3 | 10 | 2 | 0 (偶数) | 4 | 256 | x = 16^2 = 256, n = 1 |
| 4 | 1 | 1 | 1 (奇数) | 1024 | 65536 | res *= 256 = 1024, n = 0 停止 |

最终 `res = 1024`。正确。

输入: `x = 2.0, n = -2`

处理：`x = 1/2 = 0.5, n = 2`，然后快速幂：
| 轮次 | n | n & 1? | res | x |
|------|---|--------|-----|---|
| 初始 | 2 | - | 1 | 0.5 |
| 1 | 2 | 0 | 1 | 0.25 |
| 2 | 1 | 1 | 0.25 | 0.0625 |

`res = 0.25`。`2^(-2) = 1/4 = 0.25`。正确。

## 💻 解法

> 快速幂迭代法：利用二进制分解指数，每次判断最低位决定是否乘入。

```python
class Solution:
    def myPow(self, x: float, n: int) -> float:
        """
        使用快速幂算法（迭代法）计算 x 的 n 次幂。
        时间复杂度: O(log n)，空间复杂度: O(1)
        """
        # 处理指数为 0 的特殊情况，任何数的 0 次幂都是 1
        if n == 0:
            return 1

        # 初始化最终结果为 1
        res = 1

        # 处理负指数：将 x 取倒数，并将 n 变为正数
        # 例如：x^-2 = (1/x)^2
        if n < 0:
            x = 1 / x
            n = -n

        # 核心循环，每次将指数减半，直到指数为 0
        while n:
            # 判断当前指数的最低位是否为 1（即 n 是否为奇数）
            # 注意：Python 中整数可以是任意大，无需显式转换
            if n & 1:
                # 如果为奇数，将当前的底数乘入结果
                res *= x

            # 底数平方：x -> x^2 -> x^4 -> x^8 ...
            x *= x
            # 指数右移一位（相当于 n //= 2），准备处理下一位
            n >>= 1

        return res
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(log n) | 每次 n 右移一位（减半），最多 log n 次循环 |
| 空间 | O(1) | 只用了常数个变量 |

## ⚠ 易错点

- [ ] **负指数处理**：`x^(-n) = (1/x)^n`，先把 x 取倒数再做快速幂，不是最后取倒数
- [ ] **n 的溢出问题**：当 `n = -2^31` 时（Java/C++ 的 Integer.MIN_VALUE），`-n` 会溢出。Python 中整数无限大不会溢出，但面试中要提到这一点
- [ ] **n & 1 判断奇偶**：用 `n & 1` 比 `n % 2 == 1` 在大多数语言中更快，且 `n >>= 1` 比 `n //= 2` 快
- [ ] **n = 0 特例**：必须最早处理，否则后面的逻辑（取负值等）会出错
- [ ] **底数平方的顺序**：先判断奇偶再平方，还是先平方再判断？迭代法中是先判断（当前轮次的 x 是 `x^(2^轮次)`），再平方为下一轮准备

```text
这道题是实现 pow(x, n)，核心是快速幂算法。

朴素做法是循环 n 次做乘法，O(n) 太慢。
快速幂利用二分思想，把指数逐位分解：

x^10 = x^(1010b) = x^8 * x^2

实现上：
1. 初始化 res = 1
2. 循环判断 n 的最低位，如果是 1，就把当前底数乘进 res

```
