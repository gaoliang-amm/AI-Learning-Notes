# 9. 回文数

## 🎯 人话翻译

> 判断一个整数是不是回文数。回文数就是正着读反着读一样（如 121 是，-121 不是，10 也不是）。

## 💡 核心思想

> **反转数字的后半部分，与前半部分比较。**
>
> 朴素方法：把整数转成字符串，用双指针比较。但面试官通常期望**不转字符串**的解法。
>
> 不转字符串的方法：不断取出原数的最后一位，构建反转的后半部分。当原数 <= 反转数时，说明已经处理了一半以上的数字。
>
> 判断条件：
> - 偶数位：如 `1221`，处理后 `x = 12, reversed = 12`，`x == reversed` 返回 True
> - 奇数位：如 `121`，处理后 `x = 1, reversed = 12`，`x == reversed // 10`（去掉中间位）返回 True
>
> 特殊情况：负数一定不是回文数；末位为 0 且非 0 的数也一定不是（如 10）。

用到：[[数学]]、[[双指针]]

## 📝 解题步骤

1. **特殊排除** — 负数返回 False；末尾为 0（非 0）的数返回 False
2. **构建反转后半部分** — `reversed_half = 0`
   - 当 `x > reversed_half` 时循环：
   - `reversed_half = reversed_half * 10 + x % 10`（取出 x 的末位加到反转数上）
   - `x //= 10`（去掉 x 的末位）
3. **判断回文** — 返回 `x == reversed_half`（偶数位）或 `x == reversed_half // 10`（奇数位）

## 🔢 手动走一遍

### 例1: `x = 121`

| 轮次 | x | x % 10 | reversed_half | x > reversed_half? |
|------|---|--------|---------------|-------------------|
| 初始 | 121 | - | 0 | Yes |
| 1 | 12 | 1 | 0*10 + 1 = 1 | Yes (12 > 1) |
| 2 | 1 | 2 | 1*10 + 2 = 12 | No (1 <= 12)，停止 |

判断：`x = 1, reversed_half = 12`
- `x == reversed_half`? `1 == 12`? No
- `x == reversed_half // 10`? `1 == 1`? Yes -> 回文！

### 例2: `x = 1221`

| 轮次 | x | x % 10 | reversed_half | x > reversed_half? |
|------|---|--------|---------------|-------------------|
| 初始 | 1221 | - | 0 | Yes |
| 1 | 122 | 1 | 1 | Yes (122 > 1) |
| 2 | 12 | 2 | 12 | No (12 <= 12)，停止 |

判断：`x = 12, reversed_half = 12`，`x == reversed_half` -> 回文！

### 例3: `x = 10`（边界特例）

`x % 10 == 0 and x != 0` -> 直接返回 False。

### 例4: `x = -121`

负数 -> 直接返回 False。

## 💻 解法

> 不转字符串：反转数字的后半部分，与前半部分比较。

```python
def isPalindrome(x: int) -> bool:
    # 特殊情况排除：负数、末位为0且非0数
    if x < 0 or (x % 10 == 0 and x != 0):
        return False

    reversed_half = 0
    while x > reversed_half:
        # 逐位反转后半部分数字
        reversed_half = reversed_half * 10 + x % 10
        # 原数去掉最后一位
        x = x // 10

    # 偶数位：x == reversed_half；奇数位：去掉中间位后相等
    return x == reversed_half or x == reversed_half // 10
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(log n) | 每次循环 x 除以 10，约 log_10(n) 次，即数字的位数 |
| 空间 | O(1) | 只用了 reversed_half 一个变量 |

## ⚠ 易错点

- [ ] **特殊情况的处理顺序**：先排除负数和末位为 0 的数（0 本身除外），否则后续逻辑会出错
- [ ] **奇数位的处理**：`x == reversed_half // 10` 去掉中间位（如 121 的中间位是 2）。不要忘记这个判断
- [ ] **循环条件**：`x > reversed_half` 不是 `x > 0`。后者会把整个数反转，无法判断
- [ ] **不要转字符串**：面试官通常会要求不转字符串。如果提了字符串做法，要能同时给出数学解法
- [ ] Python 的 `//` 是整除，`%` 是取余，不要和 `/`（浮点除）混淆

## 🎤 面试怎么说

```text
这道题判断一个整数是否是回文数。我先说特殊情况的排除：
1. 负数一定不是回文数
2. 末位是 0 的非零数也一定不是（因为最高位不可能是 0）

核心思路是反转数字的后半部分，和前半部分比较：
- 每次取 x 的个位，加到 reversed_half 的末尾
- x 去掉个位
- 当 x <= reversed_half 时，说明已经处理了一半以上的位数

判断：
- 偶数位（如 1221）：x == reversed_half (12 == 12)
- 奇数位（如 121）：x == reversed_half // 10 (1 == 12 // 10 = 1)

时间复杂度 O(log n)，实际上就是数字的位数。
空间复杂度 O(1)。

这个解法不需要把整数转为字符串，只用了基本的数学操作。
```
