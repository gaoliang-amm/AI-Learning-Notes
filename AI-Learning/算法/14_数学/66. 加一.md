# 66. 加一

## 🎯 人话翻译

> 用一个数组表示一个大整数（如 [1,2,3] 表示 123），把这个整数加一，返回新的数组。

## 💡 核心思想

> **模拟十进制加法：从末位开始加一，处理进位。如果某位加一后不为 10（即不需进位），直接返回。如果遍历完所有位仍然有进位（所有位都是 9），在数组最前面加一个 1。**
>
> 巧妙做法：从后往前遍历，`digits[i] = (digits[i] + 1) % 10`。如果结果不为 0（即不是从 9 变成 10），说明不需要再向前进位，直接返回。如果循环结束都没返回，说明全是 9（如 999 -> 1000），在数组头部插入 1。

用到：[[数学]]、[[数组]]

## 📝 解题步骤

1. **逆序遍历** — 从最后一位（个位）开始向前
2. **当前位加一** — `digits[i] += 1`
3. **取模** — `digits[i] %= 10`
4. **判断是否进位** — 如果 `digits[i] != 0`（说明原来是 0-8，加一后是 1-9，没有进位），直接返回
5. **如果循环结束都没返回** — 说明所有位都是 9（如 99 -> 00），需要在开头插入 1：返回 `[1] + digits`

## 🔢 手动走一遍

### 例1: `digits = [1, 2, 3]`

| i | digits[i] 原值 | digits[i] += 1 | digits[i] %= 10 | digits[i] != 0? | 操作 |
|---|---------------|----------------|-----------------|----------------|------|
| 2 | 3 | 4 | 4 | Yes (4 != 0) | 直接返回 [1, 2, 4] |

### 例2: `digits = [1, 2, 9]`

| i | digits[i] 原值 | digits[i] += 1 | digits[i] %= 10 | digits[i] != 0? | 操作 |
|---|---------------|----------------|-----------------|----------------|------|
| 2 | 9 | 10 | 0 | No (0 == 0) | 继续循环 |
| 1 | 2 | 3 | 3 | Yes (3 != 0) | 直接返回 [1, 3, 0] |

### 例3: `digits = [9, 9, 9]`

| i | digits[i] 原值 | digits[i] += 1 | digits[i] %= 10 | digits[i] != 0? | 操作 |
|---|---------------|----------------|-----------------|----------------|------|
| 2 | 9 | 10 | 0 | No | 继续 |
| 1 | 9 | 10 | 0 | No | 继续 |
| 0 | 9 | 10 | 0 | No | 循环结束，返回 [1] + [0,0,0] = [1, 0, 0, 0] |

## 💻 解法

> 从末位开始加一取模，不需进位就返回；全 9 的情况在头部加 1。

```python
def plusOne(digits):
    n = len(digits)
    # 从最后一位开始遍历
    for i in range(n - 1, -1, -1):
        # 当前位加1
        digits[i] += 1
        # 取余：如果是10则变为0，否则保持不变
        digits[i] %= 10
        # 如果不需要进位（当前位不为0），直接返回结果
        if digits[i] != 0:
            return digits
    # 如果循环结束仍未返回，说明所有位都是9，需要在头部加1
    return [1] + digits
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(n) | 最坏情况全部是 9，需要遍历完所有位。一般情况提前返回 |
| 空间 | O(1) / O(n) | 原地修改 O(1)，全 9 时需要创建新数组 O(n) |

## ⚠ 易错点

- [ ] **循环结束后返回 `[1] + digits`**：不能返回 digits（此时全是 0），需要前面加 1
- [ ] **进位逻辑**：`digits[i] %= 10` 让 10 变成 0，如果原来不是 10（即加一前不是 9），则不变且非 0，可立即返回
- [ ] **直接返回**：`if digits[i] != 0: return digits` 利用了「如果当前位加一后不是 10，说明无需向前进位」的性质
- [ ] **不要用转字符串再转回**：如 `int(''.join(map(str, digits))) + 1` 虽然能过，但面试官期望看到数组层面的操作
- [ ] `[1] + digits` 在 Python 中创建新数组。如果是 C++ 的 vector，用 `digits.insert(digits.begin(), 1)` 也可以

1. 当前位加一
