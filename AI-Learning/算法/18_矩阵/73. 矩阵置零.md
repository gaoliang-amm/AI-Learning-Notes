# 73. 矩阵置零

## 🎯 人话翻译

> 矩阵中如果某个位置是 0，就把这个位置的**整行**和**整列**全部变成 0。要求原地修改。

## 💡 核心思想

> **用第一行和第一列作为标记数组，空间 O(1)。**
>
> 朴素做法：开两个 boolean 数组记录哪些行/列需要置零，再用它们去修改矩阵，但需要 O(m+n) 额外空间。
>
> 优化：用矩阵的第一行和第一列本身来存储"该列/该行是否需要置零"的标记。需要两个额外变量记录第一行和第一列自身是否需要置零（因为它们会被用来存标记）。
>
> 三步走：
> 1. 先检查第一行和第一列是否有 0
> 2. 遍历剩余区域，遇到 0 就在对应第一行/第一列位置打标记
> 3. 根据标记置零（跳过第一行/第一列）
> 4. 根据第一步的结果置零第一行/第一列

用到：[[矩阵]] [[原地算法]]

## 📝 解题步骤

1. **记录第一行/列是否有 0** — `first_row_zero` 和 `first_col_zero`
2. **用第一行/列打标记** — 遍历 `i=1..m-1, j=1..n-1`：若 `matrix[i][j] == 0`，则 `matrix[i][0] = 0` 且 `matrix[0][j] = 0`
3. **根据标记置零内部区域** — 遍历 `i=1..m-1, j=1..n-1`：若 `matrix[i][0] == 0` 或 `matrix[0][j] == 0`，则 `matrix[i][j] = 0`
4. **处理第一行/列** — 若 `first_row_zero`，置零整行；若 `first_col_zero`，置零整列

## 🔢 手动走一遍

输入: `matrix = [[1, 1, 1], [1, 0, 1], [1, 1, 1]]`

**第一步：记录第一行/列是否有 0**

| 检查 | 结果 |
|------|------|
| 第一行有 0 吗？| 否，first_row_zero = False |
| 第一列有 0 吗？| 否，first_col_zero = False |

**第二步：遍历内部打标记**

| i | j | matrix[i][j] | == 0? | matrix[i][0] | matrix[0][j] | 矩阵状态 |
|---|---|-------------|--------|-------------|-------------|----------|
| 1 | 1 | 0 | 是 | 0 | 0 | [1,0,1] / [0,0,1] / [1,1,1] |

标记后矩阵：
```
1 0 1
0 0 1
1 1 1
```
其中 matrix[1][0]=0 表示第 1 行要置零，matrix[0][1]=0 表示第 1 列要置零。

**第三步：根据标记置零内部**

| i | j | matrix[i][0]==0? | matrix[0][j]==0? | 置零? | 矩阵 |
|---|---|-----------------|-----------------|-------|------|
| 1 | 1 | 是 | 是 | 是 | [1,0,1]/[0,0,1]/[1,1,1] (已为0) |
| 1 | 2 | 是 | 否(matrix[0][2]=1) | 是 | [1,0,1]/[0,0,0]/[1,1,1] |
| 2 | 1 | 否(matrix[2][0]=1) | 是 | 是 | [1,0,1]/[0,0,0]/[1,0,1] |
| 2 | 2 | 否 | 否 | 否 | 不变 |

**第四步：处理第一行/列**

first_row_zero = False, first_col_zero = False，不做额外置零。

结果:
```
1 0 1
0 0 0
1 0 1
```

## 💻 解法

> 用第一行和第一列作为标记数组，O(1) 额外空间。

```python
from typing import List


class Solution:
    def setZeroes(self, matrix: List[List[int]]) -> None:
        """
        原地将矩阵中 0 所在的行和列全部置为 0。
        使用第一行和第一列作为标记数组，空间复杂度 O(1)。
        """
        if not matrix or not matrix[0]:
            return

        m, n = len(matrix), len(matrix[0])

        # 1. 记录第一行和第一列是否原本有 0
        first_row_zero = any(matrix[0][j] == 0 for j in range(n))
        first_col_zero = any(matrix[i][0] == 0 for i in range(m))

        # 2. 用第一行和第一列作为标记（记录哪些行/列需要置零）
        for i in range(1, m):
            for j in range(1, n):
                if matrix[i][j] == 0:
                    matrix[i][0] = 0  # 标记第 i 行需要置零
                    matrix[0][j] = 0  # 标记第 j 列需要置零

        # 3. 根据标记将对应位置置零（跳过第一行和第一列）
        for i in range(1, m):
            for j in range(1, n):
                if matrix[i][0] == 0 or matrix[0][j] == 0:
                    matrix[i][j] = 0

        # 4. 处理第一行和第一列
        if first_row_zero:
            for j in range(n):
                matrix[0][j] = 0
        if first_col_zero:
            for i in range(m):
                matrix[i][0] = 0
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(m*n) | 遍历矩阵常数次 |
| 空间 | O(1) | 只用第一行/第一列做标记 + 两个布尔变量 |

## ⚠ 易错点

- [ ] **必须先记录**第一行/列是否有 0，**再打标记**。如果反过来，打标记的过程可能覆盖第一行/列的原始数据，导致信息丢失
- [ ] 第三步置零时跳过第一行和第一列——因为第一行和第一列已经被用来存储标记了，如果现在修改它们，后续的标记信息就丢失了
- [ ] 标记 `matrix[i][0]` 表示第 i 行需要置零，`matrix[0][j]` 表示第 j 列需要置零。映射关系不要搞反
- [ ] 如果需要 O(m+n) 空间的做法更简单，但面试官通常会追问 O(1) 空间的优化
- [ ] 如果不想用第一行/列做标记，也可以用"给要置零的行/列做特殊标记值"的方法，但需要知道值的范围

## 🎤 面试怎么说

```text
这道题的核心挑战是用 O(1) 空间完成。朴素思路是开两个数组记录需要置零的行和列，
但需要 O(m+n) 空间。

优化思路：用矩阵的第一行和第一列本身来做标记。
- matrix[i][0] = 0 表示第 i 行要置零
- matrix[0][j] = 0 表示第 j 列要置零

因为第一行和第一列被拿来做标记，它们原本是否有 0 的信息需要用两个变量提前记录下来。

然后分三步：打标记 -> 根据标记置零内部区域 -> 根据记录置零第一行/列。

时间 O(m*n)，空间 O(1)。
```
