# 54. 螺旋矩阵

## 🎯 人话翻译

> 按顺时针螺旋顺序返回一个 m x n 矩阵的所有元素。

## 💡 核心思想

> **逐层收缩：维护上下左右四个边界，按"左到右 -> 上到下 -> 右到左 -> 下到上"的顺序遍历，每走完一条边就收缩对应边界。**
>
> 边界变量：`top`, `bottom`, `left`, `right`。每次：
> 1. 从左到右遍历上边界，`top++`（上边界下移一行）
> 2. 从上到下遍历右边界，`right--`（右边界左移一列）
> 3. 如果还有行：从右到左遍历下边界，`bottom--`
> 4. 如果还有列：从下到上遍历左边界，`left++`
>
> 步骤 3 和 4 的条件检查是关键——因为步骤 1 和 2 可能已导致边界交叉。

用到：[[矩阵]] [[模拟]]

## 📝 解题步骤

1. **边界初始化** — `top=0, bottom=m-1, left=0, right=n-1`
2. **循环直到边界交叉** — while `top <= bottom and left <= right`：
3. **左到右** — 遍历 `col=left..right`，追加 `matrix[top][col]`，`top++`
4. **上到下** — 遍历 `row=top..bottom`，追加 `matrix[row][right]`，`right--`
5. **右到左（条件：top <= bottom）** — 遍历 `col=right..left`，追加 `matrix[bottom][col]`，`bottom--`
6. **下到上（条件：left <= right）** — 遍历 `row=bottom..top`，追加 `matrix[row][left]`，`left++`
7. **返回结果列表**

## 🔢 手动走一遍

输入: `matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]`

| 步骤 | 边界 (t,b,l,r) | 操作 | 追加 | 更新边界 | res |
|------|---------------|------|------|----------|-----|
| 初始 | (0,2,0,2) | - | - | - | [] |
| 1 左到右 | (0,2,0,2) | 遍历 col=0..2: 1,2,3 | 1,2,3 | top=1 | [1,2,3] |
| 2 上到下 | (1,2,0,2) | 遍历 row=1..2: 6,9 | 6,9 | right=1 | [1,2,3,6,9] |
| 3 右到左 | (1,2,0,1) | top(1)<=bottom(2)? 是，遍历 col=1..0: 8,7 | 8,7 | bottom=1 | [1,2,3,6,9,8,7] |
| 4 下到上 | (1,1,0,1) | left(0)<=right(1)? 是，遍历 row=1..1: 4 | 4 | left=1 | [1,2,3,6,9,8,7,4] |
| 结束 | (1,1,1,1) | top(1)<=bottom(1) 且 left(1)<=right(1)? 是 | 单元素情况（此例不会进入新循环） | - | - |

实际这个 3x3 矩阵在经过第 4 步后，top=1, bottom=1, left=1, right=1，下一轮 while：

| 步骤 | 边界 (t,b,l,r) | 操作 | 追加 | 更新边界 | res |
|------|---------------|------|------|----------|-----|
| 1 左到右 | (1,1,1,1) | 遍历 col=1..1: 5 | 5 | top=2 | [...4,5] |
| 结束 | (2,1,1,1) | top(2) > bottom(1)，退出 | - | - | - |

结果: `[1, 2, 3, 6, 9, 8, 7, 4, 5]`

## 💻 解法

> 四边界收缩法：按顺时针顺序遍历四条边，每走完一条就收缩边界。

```python
from typing import List


class Solution:
    def spiralOrder(self, matrix: List[List[int]]) -> List[int]:
        if not matrix or not matrix[0]:
            return []

        m, n = len(matrix), len(matrix[0])
        top, bottom = 0, m - 1
        left, right = 0, n - 1
        res = []

        while top <= bottom and left <= right:
            # 1. 从左到右遍历上边界
            for col in range(left, right + 1):
                res.append(matrix[top][col])
            top += 1

            # 2. 从上到下遍历右边界
            for row in range(top, bottom + 1):
                res.append(matrix[row][right])
            right -= 1

            # 3. 从右到左遍历下边界（需确保还有行）
            if top <= bottom:
                for col in range(right, left - 1, -1):
                    res.append(matrix[bottom][col])
                bottom -= 1

            # 4. 从下到上遍历左边界（需确保还有列）
            if left <= right:
                for row in range(bottom, top - 1, -1):
                    res.append(matrix[row][left])
                left += 1

        return res
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(m*n) | 每个元素恰好被访问一次 |
| 空间 | O(1) | 不计结果数组，只用了边界变量 |

## ⚠ 易错点

- [ ] 步骤 3 和 4 必须加条件判断——例如只有一行的矩阵，步骤 1 已经遍历了整行，步骤 3 会重复遍历
- [ ] `range(right, left - 1, -1)` 的终止条件是 `left - 1`（不包含 left-1），等价于遍历 `[right, right-1, ..., left]`
- [ ] 同理 `range(bottom, top - 1, -1)` 遍历 `[bottom, bottom-1, ..., top]`
- [ ] 空矩阵或空行矩阵（`[]` 或 `[[]]`）要处理，开头判断 `if not matrix or not matrix[0]`
- [ ] 边界更新的顺序不能乱：遍历完上边界才 `top++`，遍历完右边界才 `right--`，等等

这道题用边界收缩法模拟螺旋遍历。

维护四个边界变量：top、bottom、left、right。
