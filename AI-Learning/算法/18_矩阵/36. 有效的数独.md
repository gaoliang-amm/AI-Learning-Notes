# 36. 有效的数独

## 🎯 人话翻译

> 判断一个 9x9 数独棋盘当前是否合法（每行、每列、每个 3x3 九宫格内 1-9 不重复）。不需要验证是否有解，只验证已填的数字不冲突。

## 💡 核心思想

> **一次遍历 + 三个 9x9 集合数组（行、列、九宫格），用 HashSet 检查重复。**
>
> 关键公式：九宫格编号 `box_idx = (i // 3) * 3 + j // 3`
> - `i // 3` 和 `j // 3` 将 0-8 映射到 0-2（每三个一行/一列）
> - `* 3` 保证第 0 行三个九宫格是 0,1,2，第 1 行三个九宫格是 3,4,5，以此类推

用到：[[哈希表]] [[矩阵]]

## 📝 解题步骤

1. **初始化三个集合数组** — `rows = [set() for _ in range(9)]`，`cols = [set() for _ in range(9)]`，`boxes = [set() for _ in range(9)]`
2. **双重循环遍历** — 遍历每个格子 `(i, j)`
3. **跳过空格** — 若 `board[i][j] == '.'`，continue
4. **计算九宫格编号** — `box_idx = (i // 3) * 3 + j // 3`
5. **检查重复** — 若数字已在 `rows[i]`、`cols[j]` 或 `boxes[box_idx]` 中，返回 False
6. **记录数字** — 将数字加入三个集合
7. **返回 True** — 全部遍历完无冲突

## 🔢 手动走一遍

```

. 9 8 | . . . | . 6 .
```

| i | j | 数字 | rows[i] | cols[j] | box_idx | 有冲突? |
|---|---|--------|---------|---------|---------|----------|
| 0 | 0 | '5' | {5} | {5} | 0 | 否 |
| 0 | 1 | '3' | {5,3} | {3} | 0 | 否 |
| 0 | 4 | '7' | {5,3,7} | {7} | 1 | 否 |
| 1 | 0 | '6' | {6} | {5,6} | 0 | 否 |
| 1 | 3 | '1' | {6,1} | {1} | 1 | 否 |
| 1 | 4 | '9' | {6,1,9} | {7,9} | 1 | 否 |
| 1 | 5 | '5' | {6,1,9,5} | {5} | 2 | 否 |
| 2 | 1 | '9' | {9} | {3,9} | 0 | 否 |
| 2 | 2 | '8' | {9,8} | {8} | 0 | 否 |
| 2 | 7 | '6' | {9,8,6} | {6} | 2 | 否 |

假设 (2,1) 是 '3' 而不是 '9'，则 cols[1] 中已有 '3'（来自 (0,1)），冲突！返回 False。

## 💻 解法

> 一次遍历，用三个集合数组分别检查行、列、九宫格重复。

```python
from typing import List


class Solution:
    def isValidSudoku(self, board: List[List[str]]) -> bool:
        # 9行、9列、9个九宫格，用集合去重
        rows = [set() for _ in range(9)]
        cols = [set() for _ in range(9)]
        boxes = [set() for _ in range(9)]

        for i in range(9):
            for j in range(9):
                c = board[i][j]
                if c == ".":
                    continue
                # 计算当前格子所属九宫格编号
                box_idx = (i // 3) * 3 + j // 3
                # 出现重复，直接返回False
                if c in rows[i] or c in cols[j] or c in boxes[box_idx]:
                    return False
                rows[i].add(c)
                cols[j].add(c)
                boxes[box_idx].add(c)
        return True
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(1) | 棋盘固定 9x9 = 81 个格子，常数时间 |
| 空间 | O(1) | 3 个集合数组最多各存储 9 个元素，常数空间 |

如果棋盘是 n x n，则时间 O(n^2)，空间 O(n^2)。

## ⚠ 易错点

- [ ] 九宫格编号公式 `box_idx = (i // 3) * 3 + j // 3`，不要写成 `i // 3 + j // 3`（那会导致不同行的九宫格编号相同）
- [ ] 只验证已填数字的合法性，**不验证数独是否有解**——这是两种不同的题（后者是 LeetCode 37 解数独）
- [ ] 数字是字符 `'1'` 到 `'9'`（string），不是整数。`'.'` 表示空格要跳过
- [ ] 另一种做法是用位运算（int 的低 9 位记录 1-9 是否出现过），可以节省空间但代码可读性差
