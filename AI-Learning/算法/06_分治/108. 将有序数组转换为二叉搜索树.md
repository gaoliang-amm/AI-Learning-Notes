# 108. 将有序数组转换为二叉搜索树

## 🎯 人话翻译

> 给你一个升序数组，用这些值建一棵平衡的 BST（高度尽可能小，左右子树高度差不超过 1）。

## 💡 核心思想

> **升序数组天然对应 BST 的中序遍历。选区间中点作为根，左半区间递归建左子树，右半区间递归建右子树。**
>
> 为什么选中点？因为中点是区间的最均衡划分，可以保证左右子树节点数尽可能接近，从而保证树的高度平衡。
>
> 分治的本质：把大问题（构建整个 BST）拆成小问题（构建左子树 + 构建右子树），分而治之。
>
> 使用下标 `left/right` 划分区间而不切片数组，避免创建大量新数组——这是 O(n) 空间而不是 O(n log n)。

用到：[[分治]]、[[二叉搜索树]]、[[二叉树]]、[[递归]]、[[中序遍历]]

## 📝 解题步骤

1. **定义递归函数** — `build(left, right)`：在 `[left, right]` 区间上构建 BST，返回根节点
2. **递归出口** — 如果 `left > right`（空区间），返回 `None`
3. **选根** — `mid = (left + right) // 2`，创建值为 `nums[mid]` 的节点
4. **递归建左子树** — `root.left = build(left, mid - 1)`
5. **递归建右子树** — `root.right = build(mid + 1, right)`
6. **返回根** — `return root`

## 🔢 手动走一遍

输入：`nums = [-10, -3, 0, 5, 9]`

递归构建过程：

| 调用层 | 区间 [l,r] | mid | 根值 | 左区间        | 右区间        |
|--------|-----------|-----|------|--------------|--------------|
| 1      | [0, 4]    | 2   | 0    | [0, 1]       | [3, 4]       |
| 2L     | [0, 1]    | 0   | -10  | [0, -1] (空) | [1, 1]       |
| 3L     | [1, 1]    | 1   | -3   | [1, 0] (空)  | [2, 1] (空)  |
| 2R     | [3, 4]    | 3   | 5    | [3, 2] (空)  | [4, 4]       |
| 3R     | [4, 4]    | 4   | 9    | [4, 3] (空)  | [5, 4] (空)  |

最终 BST：

```
        0
       / \
     -10  5
       \   \
       -3   9
```

## 💻 解法

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

class Solution:
    def sortedArrayToBST(self, nums):
        """
        官方最优思路：
        1. 升序数组对应 BST 的中序遍历
        2. 每次选区间中点作为根，保证树高度平衡
        3. 用 left/right 下标划分区间，不切片、不新建数组
        """
        def build(left, right):
            # 递归出口：区间无效，返回空节点
            if left > right:
                return None

            # 选中点作为根节点（防止溢出写法：left + (right - left) // 2）
            mid = (left + right) // 2
            root = TreeNode(nums[mid])

            # 递归构建左子树：[left, mid-1]
            root.left = build(left, mid - 1)
            # 递归构建右子树：[mid+1, right]
            root.right = build(mid + 1, right)

            return root

        # 初始调用：整个数组区间 [0, len(nums)-1]
        return build(0, len(nums) - 1)
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明                                                         |
|------|--------|--------------------------------------------------------------|
| 时间 | O(n)   | 每个元素恰好成为一个节点，访问一次                             |
| 空间 | O(log n)| 递归栈深度为树高，平衡树 O(log n)。用下标不切片，数组本身 O(n) |

## ⚠ 易错点

- [ ] **递归出口是 `left > right`** — 不是 `left >= right`，因为 `mid` 被取走，左右区间可能退化为单元素后左右收缩出现 `left > right`
- [ ] **中点取整** — `(left + right) // 2` 在 Python 中自动向负无穷取整，对于偶数长度区间会取偏左的点，这也是合理的
- [ ] **不要用数组切片** — `nums[:mid]` 会创建新数组（O(n) 复制），用下标划分是 O(1)
- [ ] **BST 不一定唯一** — 升序数组可以构造多棵不同的平衡 BST，取中点是一种方案，所有方案都合法
