# 23. 合并 K 个升序链表

## 🎯 人话翻译

> 给你 k 个已经各自排好序的链表，把它们全部合并成 1 个升序链表。

## 💡 核心思想

> **方法一（优先队列/最小堆）：把 k 个链表的头节点放入最小堆，每次弹出最小值接到结果链表后面，然后把被弹出节点的下一个节点加入堆。**
>
> 方法二（分治归并）：两两合并，类似归并排序的后半段。将 k 个链表分成左右两组，递归合并各组，最后把左右结果二合一。
>
> 方法一更直观，适合面试快速写出。核心思路和"合并两个有序链表"一样，只是从 2 个扩展到了 k 个，需要一个最小堆来高效找到 k 个候选中的最小值。
>
> 注意 Python 的 heapq 不能直接比较 `ListNode`，需要用 `(node.val, index, node)` 做元组，其中 `index` 是链表在 `lists` 中的序号，用于打破平局。

用到：[[分治]]、[[堆]]、[[优先队列]]、[[链表]]、[[归并排序]]

## 📝 解题步骤

**方法一（最小堆）：**

1. **初始化** — `dummy = ListNode(0)`，`cur = dummy`，`heap = []`
2. **所有链表头入堆** — `for i, node in enumerate(lists)`：如果非空，压入 `(node.val, i, node)`
3. **循环弹出最小** — `while heap`：
   - 弹出堆顶 `(val, i, node)`，取值最小
   - 接到结果链表后面：`cur.next = node`，`cur = cur.next`
   - 如果 `node.next` 非空，把下一个节点压入堆：`heapq.heappush(heap, (node.next.val, i, node.next))`
4. **返回 `dummy.next`**

**方法二（分治归并）：**

1. **递归出口** — 如果 `lists` 为空返回 `None`，只有一条返回 `lists[0]`
2. **分** — `mid = len(lists) // 2`，左右两组递归调用
3. **治** — 合并左右的结果（同 [[21. 合并两个有序链表]]）

## 💻 解法一（最小堆，推荐）

```python
from typing import List, Optional
import heapq


class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next


class Solution:
    def mergeKLists(self, lists: List[Optional[ListNode]]) -> Optional[ListNode]:
        # 时间复杂度 O(N log k)，空间复杂度 O(k)

        # 虚拟头节点，简化操作
        dummy = ListNode(0)
        cur = dummy

        # 最小堆，存储 (节点值, 节点索引, 节点)
        # 使用索引避免节点比较（Python 不能直接比较 ListNode）
        heap = []

        # 将所有非空链表的头节点加入堆
        for i, node in enumerate(lists):
            if node:
                heapq.heappush(heap, (node.val, i, node))

        # 循环弹出最小节点
        while heap:
            val, i, node = heapq.heappop(heap)

            # 将当前节点连接到结果链表
            cur.next = node
            cur = cur.next

            # 如果当前节点有下一个节点，加入堆
            if node.next:
                heapq.heappush(heap, (node.next.val, i, node.next))

        return dummy.next
```

## 💻 解法二（分治归并）

```python
class Solution:
    def mergeKLists(self, lists: List[Optional[ListNode]]) -> Optional[ListNode]:
        if not lists:
            return None
        return self._merge_range(lists, 0, len(lists) - 1)

    def _merge_range(self, lists, left, right):
        if left == right:
            return lists[left]
        mid = (left + right) // 2
        l1 = self._merge_range(lists, left, mid)
        l2 = self._merge_range(lists, mid + 1, right)
        return self._merge_two(l1, l2)

    def _merge_two(self, l1, l2):
        dummy = ListNode(0)
        cur = dummy
        while l1 and l2:
            if l1.val < l2.val:
                cur.next = l1
                l1 = l1.next
            else:
                cur.next = l2
                l2 = l2.next
            cur = cur.next
        cur.next = l1 if l1 else l2
        return dummy.next
```

## ⏱ 复杂度

| 方法       | 时间        | 空间    | 说明                                   |
|-----------|------------|---------|----------------------------------------|
| 最小堆     | O(N log k) | O(k)    | N 为总节点数，k 为链表数，堆大小 ≤ k     |
| 分治归并   | O(N log k) | O(log k)| 递归栈深度 log k                        |

## ⚠ 易错点

- [ ] **`heapq` 不能比较 `ListNode`** — 必须用元组 `(val, idx, node)`，其中 `idx` 用于打破值相同的平局
- [ ] **压入堆时检查 `node.next` 非空** — 空节点不能压入堆
- [ ] **结果用 dummy 哨兵** — 避免对头节点的特殊处理
- [ ] **`heappush` 和 `heappop` 注意拼写** — 是 `heapq.heappush` 和 `heapq.heappop`，别写成 `heapq.push`
- [ ] **分治法中空列表处理** — `if not lists: return None`

## 🎤 面试怎么说

```text
这道题合并 k 个有序链表，有两种主流做法：

方法一（推荐的最小堆法）：
把 k 个链表的头节点放入最小堆。
每次弹出堆中最小的节点，接到结果链表后面，
然后把这个节点的下一个节点压入堆。
重复直到堆空。时间 O(N log k)，空间 O(k)。

方法二（分治归并法）：
两两合并——类似归并排序，把 k 个链表不断二分，
直到只剩一个，然后自底向上合并。
时间也是 O(N log k)，空间 O(log k)。

方法一更好写，面试推荐。
关键点：Python 的 heapq 需要 (val, idx, node)
元组来避免节点之间的直接比较。
```
