# 72. 编辑距离

## 🎯 人话翻译

> 给你两个单词，每次可以插入一个字符、删除一个字符、或替换一个字符。问最少几步能把第一个单词变成第二个单词。

## 💡 核心思想

> **双字符串 DP，比较每个字符是否相等，决定操作。**
>
> `dp[i][j]` 表示 `word1` 的前 `i` 个字符变成 `word2` 的前 `j` 个字符需要的最小操作数。
>
> 如果 `word1[i-1] == word2[j-1]`：最后一个字符相等，不需要操作，`dp[i][j] = dp[i-1][j-1]`。
>
> 如果不等，有三种操作，取最小值：
> - **删除** `word1[i-1]`：`dp[i-1][j] + 1`（删掉 word1 的最后一个字符，剩下的 i-1 个字符想办法变成 word2 的前 j 个）
> - **插入** `word2[j-1]`：`dp[i][j-1] + 1`（word1 不变，在末尾插入 word2[j-1]，之后 word1 的前 i 个字符 = word2 的前 j-1 个）
> - **替换**：`dp[i-1][j-1] + 1`（把 word1[i-1] 替换成 word2[j-1]，剩下的前 i-1 和前 j-1 匹配）
>
> 初始化：`dp[i][0] = i`（把 word1 前 i 个字符变成空串，需要删除 i 次），`dp[0][j] = j`（把空串变成 word2 前 j 个字符，需要插入 j 次）。

用到：[[动态规划]]、[[多维DP]]、[[字符串DP]]

## 📝 解题步骤

1. **初始化 dp 表** — `dp = [[0]*(n+1) for _ in range(m+1)]`，其中 m=len(word1), n=len(word2)
2. **初始化边界** — `dp[i][0] = i`（word1 变空串：删 i 次），`dp[0][j] = j`（空串变 word2：插 j 次）
3. **填充 dp 表** — 双重循环 i 从 1 到 m，j 从 1 到 n
   - 字符相等：`dp[i][j] = dp[i-1][j-1]`
   - 字符不等：`dp[i][j] = min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1]) + 1`
4. **返回结果** — `dp[m][n]`

## 💻 解法

> 标准的双字符串 DP，三种操作取最小值。

```python
class Solution:
    def minDistance(self, word1: str, word2: str) -> int:
        # 时间复杂度 O(m × n)，空间复杂度 O(m × n)

        m, n = len(word1), len(word2)

        # dp[i][j] 表示 word1[:i] 转换为 word2[:j] 的最小操作数
        dp = [[0] * (n + 1) for _ in range(m + 1)]

        # 初始化：word1 转换为空字符串，需要删除所有字符
        for i in range(1, m + 1):
            dp[i][0] = i

        # 初始化：空字符串转换为 word2，需要插入所有字符
        for j in range(1, n + 1):
            dp[0][j] = j

        for i in range(1, m + 1):
            for j in range(1, n + 1):
                if word1[i - 1] == word2[j - 1]:
                    # 字符相等，不需要额外操作
                    dp[i][j] = dp[i - 1][j - 1]
                else:
                    # 不相等，取三种操作的最小值
                    # 删除：dp[i-1][j] + 1
                    # 插入：dp[i][j-1] + 1
                    # 替换：dp[i-1][j-1] + 1
                    dp[i][j] = min(
                        dp[i - 1][j],      # 删除 word1[i-1]
                        dp[i][j - 1],      # 插入 word2[j-1]
                        dp[i - 1][j - 1]   # 替换
                    ) + 1

        return dp[m][n]
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(m * n) | 双层循环遍历整个 dp 表 |
| 空间 | O(m * n) | dp 表大小 (m+1) * (n+1)。可优化到 O(min(m,n)) |

## ⚠ 易错点

- [ ] 初始化 `dp[i][0] = i` 和 `dp[0][j] = j`，不是 0。理解为：一个非空串变成空串需要删除该串的所有字符
- [ ] 字符串索引偏移：`word1[i-1]` 和 `word2[j-1]` 不是 `word1[i]` 和 `word2[j]`。dp 表的 i 表示"前 i 个字符"，对应字符串的索引 0 到 i-1
- [ ] 三种操作的含义不要混淆：`dp[i-1][j]` 对应删除（word1 少一个字符，word2 不变），`dp[i][j-1]` 对应插入（word1 不变，word2 少一个字符），`dp[i-1][j-1]` 对应替换
- [ ] 空间优化时（用两行滚动），每一行第一个元素（`j=0`）的初始化不能忘
- [ ] 当两个串完全相同时（如 `word1 = "a"`, `word2 = "a"`），`dp[1][1] = dp[0][0] = 0`，不是 1

## 🎤 面试怎么说

```text
这道题是经典的编辑距离问题，用双字符串动态规划。

核心思路：dp[i][j] 表示 word1 的前 i 个字符变成 word2 的前 j 个字符的最少操作数。

对于每一对字符 (word1[i-1], word2[j-1])：
- 如果相等：dp[i][j] = dp[i-1][j-1]（不用操作）
- 如果不等：考虑三种操作取最小值
  - 删除 word1[i-1]：dp[i-1][j] + 1
  - 插入 word2[j-1]：dp[i][j-1] + 1
  - 替换：dp[i-1][j-1] + 1

初始化：dp[i][0] = i（全删除），dp[0][j] = j（全插入）

时间复杂度 O(m*n)，空间复杂度 O(m*n)，
可以优化到 O(min(m,n)) 用滚动数组。

面试时常问：为什么不能只用两个一维数组交替？
因为 dp[i][j] 依赖 dp[i-1][j]、dp[i][j-1]、dp[i-1][j-1] 三个方向。
```
