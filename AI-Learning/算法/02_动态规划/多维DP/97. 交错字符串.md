# 97. 交错字符串

## 🎯 人话翻译

> 给你三个字符串 s1、s2、s3，判断 s3 是不是由 s1 和 s2 交错拼接而成的。交错的意思是：保持 s1 和 s2 各自的字符顺序，把它们的字符穿插组合起来，形成 s3。

## 💡 核心思想

> **双序列 DP：dp[i][j] 表示 s1 的前 i 个字符和 s2 的前 j 个字符能否交错组成 s3 的前 i+j 个字符。**
>
> 对于位置 `(i, j)`，s3 的当前字符 `s3[i+j-1]` 只能来自 s1 或 s2：
> - 如果来自 s1：需要 `s1[i-1] == s3[i+j-1]` 且 `dp[i-1][j] == True`（s1 的当前字符匹配 s3，且之前的部分能交错组成）
> - 如果来自 s2：需要 `s2[j-1] == s3[i+j-1]` 且 `dp[i][j-1] == True`
>
> 两种来源只要一种可行，`dp[i][j] = True`。
>
> 注意首先检查 `len(s1) + len(s2) != len(s3)`，长度不匹配直接返回 False。

用到：[[动态规划]]、[[多维DP]]、[[字符串DP]]

## 📝 解题步骤

1. **长度检查** — 如果 `len(s1) + len(s2) != len(s3)`，直接返回 False
2. **初始化 dp 表** — `dp = [[False]*(n+1) for _ in range(m+1)]`，`dp[0][0] = True`
3. **初始化第一列**（只用 s1）— 对 i 从 1 到 m：`dp[i][0] = dp[i-1][0] and s1[i-1] == s3[i-1]`
4. **初始化第一行**（只用 s2）— 对 j 从 1 到 n：`dp[0][j] = dp[0][j-1] and s2[j-1] == s3[j-1]`
5. **填充 dp 表** — 双重循环 i,j 从 1 开始：`dp[i][j] = (dp[i-1][j] and s1[i-1]==s3[i+j-1]) or (dp[i][j-1] and s2[j-1]==s3[i+j-1])`
6. **返回结果** — `dp[m][n]`

## 💻 解法

> 标准的双序列 DP，每次决策当前字符来自哪个字符串。

```python
class Solution:
    def isInterleave(self, s1: str, s2: str, s3: str) -> bool:
        # 时间复杂度 O(m × n)，空间复杂度 O(m × n)

        m, n = len(s1), len(s2)
        if m + n != len(s3):
            return False

        # dp[i][j] 表示 s1[:i] 和 s2[:j] 能否交错成 s3[:i+j]
        dp = [[False] * (n + 1) for _ in range(m + 1)]

        # 初始化：空字符串
        dp[0][0] = True

        # 初始化第一列（只用 s1）
        for i in range(1, m + 1):
            dp[i][0] = dp[i - 1][0] and s1[i - 1] == s3[i - 1]

        # 初始化第一行（只用 s2）
        for j in range(1, n + 1):
            dp[0][j] = dp[0][j - 1] and s2[j - 1] == s3[j - 1]

        # 填充 dp 表
        for i in range(1, m + 1):
            for j in range(1, n + 1):
                # 从 s1 取字符
                from_s1 = dp[i - 1][j] and s1[i - 1] == s3[i + j - 1]
                # 从 s2 取字符
                from_s2 = dp[i][j - 1] and s2[j - 1] == s3[i + j - 1]
                dp[i][j] = from_s1 or from_s2

        return dp[m][n]
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(m * n) | 双重循环遍历整个 dp 表 |
| 空间 | O(m * n) | dp 表大小，可优化到 O(n) 用一维滚动 |

## ⚠ 易错点

- [ ] 必须先检查 `len(s1) + len(s2) != len(s3)`，不相等直接返回 False。否则后续索引 `s3[i+j-1]` 可能越界
- [ ] 第一行和第一列的初始化必须单独处理，因为此时只有一种来源（只有 s1 或只有 s2）。初始化公式必须用 `and` 连接前一个状态和当前匹配
- [ ] `s3` 的索引是 `i+j-1`（不是 `i+j` 或 `i+j-2`）。dp[i][j] 对应 s3 的前 `i+j` 个字符，最后一个字符索引是 `i+j-1`
- [ ] `from_s1` 和 `from_s2` 判断时，字符串索引是 `s1[i-1]` 和 `s2[j-1]`（因为 i 表示前 i 个字符），不能与 dp 索引混淆
- [ ] 此题空间可以优化到 O(n)（只保留一行），但需要小心更新顺序

## 🎤 面试怎么说

```text
这道题判断 s3 是否能由 s1 和 s2 交错组成，用双序列 DP。

核心思路：dp[i][j] 表示 s1 前 i 个字符和 s2 前 j 个字符能否交错形成 s3 前 i+j 个字符。

对于状态 dp[i][j]，s3 的最后一个字符 s3[i+j-1] 只能来自：
- s1[i-1]（如果相等且 dp[i-1][j]=True）
- s2[j-1]（如果相等且 dp[i][j-1]=True）
两者满足任一即可。

首先检查总长度是否匹配，不匹配直接 False。
初始化第一行和第一列（单独用 s1 或 s2 的情况）。
然后填充 dp 表。

时间复杂度 O(m*n)，空间 O(m*n)，可以优化到 O(n)。

这个问题也可以用 BFS/DFS 做，但 DP 最直观。
```
