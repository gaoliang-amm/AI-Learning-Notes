# 56. 合并区间

## 🎯 人话翻译

> 给一堆区间（每个区间由 [start, end] 表示），把所有有重叠的区间合并起来。比如 [[1,3], [2,6], [8,10], [15,18]] -> [[1,6], [8,10], [15,18]]。

## 💡 核心思想

> 按左端点排序 + 贪心[[区间合并]]。排序后，重叠的区间会相邻出现。遍历时维护一个结果列表，如果当前区间和结果列表最后一个区间不重叠（`当前start > 上一个的end`），直接加入；如果重叠，更新上一个区间的右端点为两者右端点的较大值。

用到：[[区间合并]]、[[贪心]]

## 📝 解题步骤

1. **按左端点排序** — `intervals.sort()`，默认按第一个元素升序
2. **初始化结果列表** — `res = []`
3. **遍历区间** — 对每个 `interval`：
   - 如果 `res` 为空 或 当前区间与上一个不重叠（`interval[0] > res[-1][1]`）：`res.append(interval)`
   - 否则（重叠）：`res[-1][1] = max(res[-1][1], interval[1])`
4. **返回结果** — `return res`

## 🔢 手动走一遍

输入: `intervals = [[1, 3], [2, 6], [8, 10], [15, 18]]`

排序后: `[[1, 3], [2, 6], [8, 10], [15, 18]]`

| interval | res | res[-1] | interval[0] > res[-1][1]? | 操作 |
|----------|------|---------|---------------------------|------|
| [1,3] | [] | 空 | res 为空 | 直接追加 -> [[1,3]] |
| [2,6] | [[1,3]] | [1,3] | 2 > 3? 否（重叠） | res[-1][1] = max(3,6)=6 -> [[1,6]] |
| [8,10] | [[1,6]] | [1,6] | 8 > 6? 是（不重叠） | 直接追加 -> [[1,6],[8,10]] |
| [15,18] | [[1,6],[8,10]] | [8,10] | 15 > 10? 是（不重叠） | 直接追加 -> [[1,6],[8,10],[15,18]] |

最终返回 `[[1, 6], [8, 10], [15, 18]]`。

## 💻 解法

> 按左端点排序，遍历时检查是否与结果列表最后一个区间重叠，重叠则合并。

```python
from typing import List


class Solution:
    def merge(self, intervals: List[List[int]]) -> List[List[int]]:
        """
        合并所有重叠区间。
        思路：按区间左端点排序，然后贪心合并。
        时间复杂度：O(n log n)，空间复杂度：O(n)
        """
        # 1. 按区间左端点从小到大排序
        intervals.sort()

        res = []
        for interval in intervals:
            # 结果为空 或 当前区间与最后一个区间不重叠 -> 直接添加
            if not res or interval[0] > res[-1][1]:
                res.append(interval)
            else:
                # 重叠了 -> 合并，更新最后一个区间的右端点为最大值
                res[-1][1] = max(res[-1][1], interval[1])

        return res
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(n log n) | 排序 O(n log n)，一次遍历 O(n) |
| 空间 | O(n) | 结果数组可能包含所有区间 |

## ⚠ 易错点

- [ ] 必须按左端点排序：如果不排序，重叠的区间不会相邻出现，合并逻辑会出错
- [ ] 合并时更新右端点用 `max(res[-1][1], interval[1])`：因为有可能当前区间完全被包含在上一个区间里（如 [1,10] 合并 [2,5]），此时右端点应保持 10
- [ ] 判断不重叠的条件是 `interval[0] > res[-1][1]`（严格大于），如果等于 1（如 [1,3] 和 [3,5]），它们共享端点 3，算重叠
- [ ] `res[-1]` 的访问要确保 `res` 不为空（`if not res` 作为前置条件）

这道题是区间合并的模板题。核心分两步：

第一步，按左端点排序。排序后重叠的区间一定相邻。

第二步，遍历并合并。维护一个结果列表 res，对每个区间：
