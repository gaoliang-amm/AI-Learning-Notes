# 228. 汇总区间

## 🎯 人话翻译

> 给一个从小到大排好序、没有重复元素的数组，把连续的数字合并成区间格式。比如 [0,1,2,4,5,7] 变成 ["0->2", "4->5", "7"]。单独一个数字就直接写数字。

## 💡 核心思想

> 类似[[区间合并]]，但数组已经排好序且无重复，不需要真正的合并操作。用双指针/跳跃法：外层 while 确定区间起点，内层 while 沿着连续的数字一直走到底，然后格式化输出。

用到：[[区间合并]]、[[双指针]]

## 📝 解题步骤

1. **初始化** — `res = []`, `i = 0`
2. **外层循环** — `while i < n`：`begin = nums[i]` 为当前区间起点
3. **内层探底** — `while i < n - 1 and nums[i] + 1 == nums[i + 1]`：`i += 1`，越过所有连续数字
4. **记录区间** — `end = nums[i]`，如果 `begin == end` 则追加 `str(begin)`，否则追加 `f"{begin}->{end}"`
5. **跳到下一个区间** — `i += 1`

## 🔢 手动走一遍

输入: `nums = [0, 1, 2, 4, 5, 7]`

| i | begin=nums[i] | 内层 while: i < n-1 and nums[i]+1 == nums[i+1] | end | 格式化 | i += 1 |
|---|---------------|--------------------------------------------------|-----|--------|--------|
| 0 | 0 | i=0: 0+1=1=1 进; i=1: 1+1=2=2 进; i=2: 2+1=3!=4 停 | 2 | "0->2" | 3 |
| 3 | 4 | i=3: 4+1=5=5 进; i=4: 5+1=6!=7 停 | 5 | "4->5" | 5 |
| 5 | 7 | i=5: 5<6 但 7+1!=? (i==n-1=5, 不满足条件不进入) | 7 | "7" | 6 |
| 退出 | 6 >= n=6 | - | - | - | - |

最终返回 `["0->2", "4->5", "7"]`。

## 💻 解法

> 双指针跳跃法：外层确定区间起点，内层沿着连续数字走到底，格式化输出。

```python
from typing import List


class Solution:
    def summaryRanges(self, nums: List[int]) -> List[str]:
        """
        汇总区间：给定无重复元素的有序整数数组，返回最小有序区间列表。
        双指针跳跃法：时间复杂度 O(n)，空间复杂度 O(1)。
        """
        res = []
        n = len(nums)
        i = 0

        while i < n:
            begin = nums[i]  # 当前区间的起点

            # 内层 while：只要下一个数字连续，就继续前进
            while i < n - 1 and nums[i] + 1 == nums[i + 1]:
                i += 1

            end = nums[i]  # 循环结束后 i 指向区间终点

            # 根据起点和终点是否相同，决定格式化方式
            if begin == end:
                res.append(str(begin))
            else:
                res.append(f"{begin}->{end}")

            i += 1  # 移动到下一个区间的起点

        return res
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(n) | 每个元素被访问一次（外层 i 和内层 while 共同推进） |
| 空间 | O(1) | 不计结果数组，只用常数变量 |

## ⚠ 易错点

- [ ] 内层 while 的条件 `i < n - 1 and nums[i] + 1 == nums[i + 1]`：需要先检查 `i < n - 1` 防止访问 `nums[i+1]` 越界
- [ ] 内层循环结束后 `i` 指向区间的终点，外层 `i += 1` 才跳到下一个未处理的元素
- [ ] `begin == end` 时输出单个数字（如 "7"），不要错误地输出 "7->7"
- [ ] 连续判断条件 `nums[i] + 1 == nums[i + 1]`：不是差 1 就不是连续的了
