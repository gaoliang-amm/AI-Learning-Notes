# 57. 插入区间

## 🎯 人话翻译

> 给一个已经按左端点排好序、互不重叠的区间列表，再给你一个新的区间，把这个新区间插进去，合并所有重叠的区间。比如已有 [[1,3], [6,9]]，插入 [2,5] --> [[1,5], [6,9]]。

## 💡 核心思想

> 因为原区间列表已经有序且不重叠，不需要重新排序。分三阶段处理：先加**完全在新区间左边**的区间，再**合并所有与新区间重叠**的区间，最后加**完全在新区间右边**的区间。本质是[[区间合并]]的一个特例。

用到：[[区间合并]]、[[贪心]]

## 📝 解题步骤

1. **初始化** — `res = []`, `i = 0`, `n = len(intervals)`
2. **第一阶段（不重叠的左边区间）** — `while i < n and intervals[i][1] < newInterval[0]`：`res.append(intervals[i])`, `i += 1`
3. **第二阶段（合并重叠区间）** — `while i < n and intervals[i][0] <= newInterval[1]`：`newInterval[0] = min(newInterval[0], intervals[i][0])`, `newInterval[1] = max(newInterval[1], intervals[i][1])`, `i += 1`；合并后 `res.append(newInterval)`
4. **第三阶段（不重叠的右边区间）** — `while i < n`：`res.append(intervals[i])`, `i += 1`
5. **返回结果** — `return res`

## 🔢 手动走一遍

输入: `intervals = [[1, 3], [6, 9]]`, `newInterval = [2, 5]`

| 阶段 | i | 条件 | 操作 |
|------|---|------|------|
| 阶段1 | 0 | intervals[0][1]=3 < newInterval[0]=2? 否 | 不进入，i 保持在 0 |
| 阶段2 | 0 | intervals[0][0]=1 <= newInterval[1]=5? 是 | merge: newInterval = [min(2,1), max(5,3)] = [1,5], i=1 |
| 阶段2 | 1 | intervals[1][0]=6 <= newInterval[1]=5? 否 | 退出阶段2, res.append([1,5]) |
| 阶段3 | 1 | i < 2? 是 | res.append([6,9]), i=2 |
| 阶段3 | 2 | i < 2? 否 | 退出 |

最终返回 `[[1, 5], [6, 9]]`。

## 💻 解法

> 三段式处理：左边无重叠、中间合并、右边无重叠。利用原列表已排序的性质，一次遍历完成。

```python
from typing import List


class Solution:
    def insert(self, intervals: List[List[int]], newInterval: List[int]) -> List[List[int]]:
        """
        插入新区间，合并重叠部分，返回新的无重叠区间列表。
        """
        res = []
        n = len(intervals)
        i = 0

        # 1. 添加所有右端点小于新区间左端点的区间（不重叠，在新区间左边）
        while i < n and intervals[i][1] < newInterval[0]:
            res.append(intervals[i])
            i += 1

        # 2. 合并所有与新区间重叠的区间
        while i < n and intervals[i][0] <= newInterval[1]:
            newInterval[0] = min(newInterval[0], intervals[i][0])
            newInterval[1] = max(newInterval[1], intervals[i][1])
            i += 1
        res.append(newInterval)  # 添加合并后的新区间

        # 3. 添加剩余区间（不重叠，在新区间右边）
        while i < n:
            res.append(intervals[i])
            i += 1

        return res
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(n) | 只做一次线性扫描，每个区间访问一次 |
| 空间 | O(n) | 结果数组存储所有区间 |

## ⚠ 易错点

- [ ] 阶段 1 的判断条件是 `intervals[i][1] < newInterval[0]`：区间完全在新区间**左边**，右端点严格小于新区间的左端点
- [ ] 阶段 2 的判断条件是 `intervals[i][0] <= newInterval[1]`：区间与新区间**有重叠**（左端点 <= 新区间的右端点，注意用了 `<=`，包括恰好接触的情况）
- [ ] 阶段 2 合并时不要忘记 `min` 和 `max`：虽然看上去 newInterval 的左端点已经是最小的了，但如果阶段 1 没有任何区间加入（即 newInterval 左端点很小），阶段 2 需要正确取 min
- [ ] 合并后的新区间要用 `res.append(newInterval)` 加入结果，放在阶段 2 和阶段 3 之间，位置不能错

```text
这道题因为原区间列表已经有序且不重叠，所以比 56 题更简单，
不需要重新排序，一次遍历就够了。

分三阶段处理：
1. 把所有完全在新区间左边的加进去
2. 把所有和新区间重叠的区间一一合并（更新 newInterval 的边界）
3. 把剩余完全在新区间右边的加进去

关键是判断重叠的条件：
- 不重叠在左边：interval右 < newInterval左
- 有重叠：interval左 <= newInterval右

时间 O(n)，空间 O(n)。也可以用 56 题的排序+sweep 解法，
但这里的 O(n) 更优。
```
