# 452. 用最少数量的箭引爆气球

## 🎯 人话翻译

> 二维平面上飘着很多气球，每个气球是一个水平区间 [start, end]。你可以在 x 轴的任意位置垂直射箭，箭能戳破所有穿过它的气球。问最少需要多少支箭才能戳破所有气球。

## 💡 核心思想

> [[贪心]] + [[区间合并]]。和经典的"会议室安排"问题一样，按区间的**右端点排序**，然后贪心：每次在"当前未被覆盖的区间中右端点最小的"那个区间的右端点射一箭，这一箭能覆盖所有和它重叠的区间。

用到：[[贪心]]、[[区间合并]]

## 📝 解题步骤

1. **处理空输入** — 如果 `points` 为空，返回 `0`
2. **按右端点排序** — `points.sort(key=lambda x: x[1])`
3. **初始化第一支箭** — `arrow_pos = points[0][1]`（射在第一个区间的右端点），`count = 1`
4. **遍历剩余区间** — 如果 `points[i][0] > arrow_pos`（区间左端点在箭的右边，不被覆盖），说明需要新的一箭，`arrow_pos = points[i][1]`，`count += 1`
5. **返回 count**

## 🔢 手动走一遍

输入: `points = [[10, 16], [2, 8], [1, 6], [7, 12]]`

按右端点排序后: `[[1, 6], [2, 8], [7, 12], [10, 16]]`

| 步骤 | 当前区间 | arrow_pos | points[i][0] > arrow_pos? | 操作 |
|------|----------|-----------|---------------------------|------|
| 初始 | - | - | - | - |
| 初始化 | [1,6] | 6 | - | count=1 |
| 遍历 [2,8] | [2,8] | 6 | 2 > 6? 否 | 被当前箭覆盖 |
| 遍历 [7,12] | [7,12] | 6 | 7 > 6? 是 | count=2, arrow_pos=12 |
| 遍历 [10,16] | [10,16] | 12 | 10 > 12? 否 | 被当前箭覆盖 |

最终返回 `2`。

## 💻 解法

> 按右端点排序后贪心：每箭射在当前未覆盖区间中右端点最小的位置，能最大化覆盖。

```python
from typing import List


class Solution:
    def findMinArrowShots(self, points: List[List[int]]) -> int:
        """
        返回引爆所有气球所需的最少弓箭数量。
        贪心策略：按右端点排序，每次射在右端点。
        """
        if not points:
            return 0

        # 1. 按右端点升序排序
        points.sort(key=lambda x: x[1])

        # 2. 初始化第一支箭
        arrow_pos = points[0][1]  # 第一支箭射在第一个区间的右端点
        count = 1

        # 3. 遍历剩余区间
        for i in range(1, len(points)):
            if points[i][0] > arrow_pos:  # 当前区间左端点 > 上一箭位置，无法覆盖
                count += 1
                arrow_pos = points[i][1]  # 更新箭的位置为当前区间的右端点

        return count
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(n log n) | 排序 O(n log n)，遍历 O(n) |
| 空间 | O(1) | 只使用常数变量 |

## ⚠ 易错点

- [ ] 排序按**右端点**而不是左端点：想想 [1,100], [2,3], [4,5]——按右端点排，一箭射在 3 能覆盖 [1,100] 和 [2,3]；按左端点排则可能射偏
- [ ] 判断条件 `points[i][0] > arrow_pos`：必须是严格大于（`>`），等于不算——如果 `points[i][0] == arrow_pos`，箭射在相交点，同样能戳破
- [ ] 注意输入可能为空，需要提前返回 0
- [ ] 本题和"最少会议室"（435 题"无重叠区间"）是相反的问题：435 是求最多不重叠的区间数，本质相同
