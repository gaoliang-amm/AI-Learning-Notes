# 125. 验证回文串

## 🎯 人话翻译

> 判断一个字符串是不是回文串。但要先过滤掉非字母数字的字符、忽略大小写。例如 "A man, a plan, a canal: Panama" 去掉空格和标点后正反读一样，是回文。

## 💡 核心思想

> 对撞[[双指针]]：左右各一个指针，同时往中间走。遇到非字母数字就跳过，比较时统一转小写。只要有一次不相等就不是回文。

用到：[[双指针]]

## 📝 解题步骤

1. **初始化双指针** — `left = 0`, `right = len(s) - 1`
2. **跳过非字母数字** — 左指针跳过非字母数字字符，右指针同理（注意 `left < right` 的守卫条件防止越界）
3. **比较字符** — 都转小写后比较，不相等直接返回 `False`
4. **移动指针** — 相等则 `left += 1`, `right -= 1` 继续向中间缩
5. **遍历结束** — 循环结束说明所有字符都匹配，返回 `True`

## 🔢 手动走一遍

输入: `s = "A man, a plan, a canal: Panama"`（简化演示为 `s = "a,b a"`）

处理: 去除逗号和空格后得到 `"aba"`

| 步骤 | left | right | s[left] | s[right] | 比较 | 操作 |
|------|------|-------|---------|----------|------|------|
| 初始 | 0 | 4 | - | - | - | 跳过左逗号 |
| 1 | 1 | 4 | 'a' | - | - | 跳过右空格 |
| 2 | 1 | 3 | 'a' | 'a' | 'a' == 'a' | left=2, right=2 |
| 退出 | 2 | 2 | - | - | - | left == right, 返回 True |

## 💻 解法

> 对撞双指针：左右指针向中间移动，跳过非字母数字，只比较字母数字字符并忽略大小写。

```python
def isPalindrome(s: str) -> bool:
    left = 0
    right = len(s) - 1

    while left < right:
        # 跳过左边非字母数字字符
        while left < right and not s[left].isalnum():
            left += 1
        # 跳过右边非字母数字字符
        while left < right and not s[right].isalnum():
            right -= 1

        # 对比字符（转小写）
        if s[left].lower() != s[right].lower():
            return False

        left += 1
        right -= 1

    return True
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(n) | 每个字符最多访问一次（两次跳过内循环加一次比较） |
| 空间 | O(1) | 只用常数个变量，原地操作 |

## ⚠ 易错点

- [ ] 内层 `while` 跳过时必须加 `left < right` 守卫，否则可能 `left` 越过 `right` 导致索引越界或错误比较
- [ ] 比较前一定要 `.lower()`：题目要求忽略大小写，不转小写会判错（比如 "A" vs "a"）
- [ ] 返回值逻辑：`left < right` 时退出是找到了不匹配；`left >= right` 时退出才是全匹配，直接返回 `True`
- [ ] Python 的 `isalnum()` 只适用于英文/数字，不处理中文等其他 Unicode，但本题输入只有 ASCII

这是双指针的经典入门题。用对撞指针，一个从左一个从右，
同时向中间走。每次跳过非字母数字字符，比较时转小写。
