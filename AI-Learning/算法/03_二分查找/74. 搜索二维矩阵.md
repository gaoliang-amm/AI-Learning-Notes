# 74. 搜索二维矩阵

## 🎯 人话翻译

> 给一个二维矩阵，每行从左到右升序，下一行的第一个元素大于上一行的最后一个元素（相当于把整个矩阵按行展开就是一个有序一维数组）。问目标值在不在矩阵里。

## 💡 核心思想

> 把二维矩阵当作**拉平的一维有序数组**来做[[二分查找]]。索引从 `0` 到 `m*n - 1`，中间索引 `mid` 映射回二维坐标：行号 `row = mid // n`，列号 `col = mid % n`。这就把二维搜索问题转化为标准的一维二分查找。

用到：[[二分查找]]、[[二维数组]]

## 📝 解题步骤

1. **处理边界** — 如果矩阵或第一行为空，返回 `False`
2. **初始化** — `left = 0`, `right = m * n - 1`（把整个矩阵看成长度为 `m*n` 的一维数组）
3. **二分查找** — 循环 `left <= right`，计算 `mid = (left + right) // 2`
4. **坐标映射** — `row = mid // n`, `col = mid % n`，拿到矩阵中对应值
5. **比较与缩区间** — 等于返回 `True`，小于去右边，大于去左边

## 🔢 手动走一遍

输入: `matrix = [[1, 3, 5, 7], [10, 11, 16, 20], [23, 30, 34, 60]]`, `target = 3`

m=3, n=4, 总长度=12

| 步骤 | left | right | mid | row=mid//4 | col=mid%4 | matrix[row][col] | 比较 | 操作 |
|------|------|-------|------|------------|-----------|-------------------|------|------|
| 初始 | 0 | 11 | - | - | - | - | - | - |
| 1 | 0 | 11 | 5 | 1 | 1 | 11 | 11 > 3 | right = 4 |
| 2 | 0 | 4 | 2 | 0 | 2 | 5 | 5 > 3 | right = 1 |
| 3 | 0 | 1 | 0 | 0 | 0 | 1 | 1 < 3 | left = 1 |
| 4 | 1 | 1 | 1 | 0 | 1 | 3 | 3 == 3 | 返回 True |

最终返回 `True`。

## 💻 解法

> 将矩阵按行展开为一维数组，用标准二分查找。关键技巧是 `mid // n` 和 `mid % n` 做坐标映射。

```python
from typing import List


class Solution:
    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:
        # 时间复杂度 O(log(m * n))，空间复杂度 O(1)

        if not matrix or not matrix[0]:
            return False

        m, n = len(matrix), len(matrix[0])
        left, right = 0, m * n - 1

        while left <= right:
            mid = (left + right) // 2
            # 将一维索引映射到二维坐标
            row = mid // n
            col = mid % n
            val = matrix[row][col]

            if val == target:
                return True
            elif val < target:
                left = mid + 1
            else:
                right = mid - 1

        return False
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(log(m * n)) | 等同于对 m*n 个元素做二分查找 |
| 空间 | O(1) | 只使用常数个变量 |

## ⚠ 易错点

- [ ] 空矩阵检查：`if not matrix or not matrix[0]` 两个都要检查，防止访问 `matrix[0]` 越界
- [ ] 坐标映射公式：`row = mid // n`（整除，取行号），`col = mid % n`（取余，取列号），别搞反
- [ ] 右边界初始值是 `m * n - 1` 不是 `m * n`，因为索引从 0 开始
- [ ] 注意区分每行有序和严格递增的关系：本题是"下一行的第一个 > 上一行的最后一个"，所以展开后全局有序；如果只是"每行有序"（LeetCode 240），就不能这么做了
