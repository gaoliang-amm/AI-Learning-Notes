# 35. 搜索插入位置

## 🎯 人话翻译

> 在一个有序数组里找目标值，找到了返回下标，找不到就返回它应该插入的位置。

## 💡 核心思想

> 经典[[二分查找]]模板题。在有序数组中查找目标值，如果不存在则 `left` 指针最终会停在插入位置。
>
> 循环条件用 `left <= right`，这样退出循环时 `left = right + 1`，`left` 正好是第一个大于等于 `target` 的位置。

用到：[[二分查找]]、[[边界搜索]]

## 📝 解题步骤

1. **初始化双指针** — `left = 0`, `right = len(nums) - 1`，表示当前搜索区间 `[left, right]`
2. **二分查找** — 当 `left <= right` 时，算 `mid = (left + right) // 2`
3. **比较与缩区间** — 如果 `nums[mid] == target`，直接返回 `mid`；如果 `nums[mid] < target`，说明目标在右边，`left = mid + 1`；否则在左边，`right = mid - 1`
4. **返回插入位置** — 循环退出时 `left > right`，`left` 就是 `target` 应该插入的位置（第一个大于 `target` 的索引）

## 🔢 手动走一遍

输入: `nums = [1, 3, 5, 6]`, `target = 2`

| 步骤 | left | right | mid | nums[mid] | 比较 | 操作 |
|------|------|-------|------|-----------|------|------|
| 初始 | 0 | 3 | - | - | - | - |
| 1 | 0 | 3 | 1 | 3 | 3 > 2 | right = 0 |
| 2 | 0 | 0 | 0 | 1 | 1 < 2 | left = 1 |
| 退出 | 1 | 0 | - | - | left > right | 返回 left = 1 |

最终返回 `1`，即 2 应该插入在索引 1，数组变成 `[1, 2, 3, 6]`。

## 💻 解法

> 标准二分查找模板，循环条件 `left <= right`，未找到时 `left` 即为插入位置。

```python
from typing import List


def search_insert(nums: List[int], target: int) -> int:
    """
    二分查找目标值的位置或插入位置。
    """
    left, right = 0, len(nums) - 1
    while left <= right:
        mid = (left + right) // 2
        if nums[mid] == target:
            return mid
        elif nums[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    # 未找到时，left 即为应插入的位置
    return left
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(log n) | 每次将搜索区间缩小一半 |
| 空间 | O(1) | 只用常数个变量 |

## ⚠ 易错点

- [ ] 循环条件写 `left < right`（少了等号）会在 `left == right` 时直接跳过，没检查那个元素
- [ ] `mid = (left + right) // 2` 在 Python 中不会溢出，但可以记住 `left + (right - left) // 2` 防溢写法
- [ ] 未找到时返回 `left` 而非 `right`：因为退出循环时 `left > right`，`left` 指向第一个大于 target 的位置，才是正确的插入位置
- [ ] 边界检查：如果 `target` 大于所有元素，`left` 会变成 `len(nums)`，这正是正确的插入位置
