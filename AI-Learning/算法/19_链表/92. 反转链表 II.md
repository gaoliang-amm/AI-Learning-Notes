# 92. 反转链表 II

## 🎯 人话翻译

> 给你一条链表和两个位置 `left`、`right`，把第 `left` 到第 `right` 个节点之间的部分反转，其余保持不变，返回反转后的链表。

## 💡 核心思想

> **头插法（穿针引线）。** 定位到 `left` 的前驱节点 `pre`，然后将 `[left+1, right]` 区间的每个节点依次"摘下"并插入到 `pre` 之后（即反转区间的头部）。这样做了 `right - left` 次头插后，区间就被原地反转了。使用 [[虚拟头节点]] 避免 `left=1` 时的特殊处理。

用到：[[链表]] [[虚拟头节点]] [[头插法]]

## 📝 解题步骤

1. **创建 dummy** — `dummy.next = head`，防止 `left=1` 时头节点变化
2. **定位 pre** — 从 `dummy` 走 `left - 1` 步，到达区间前驱
3. **头插反转** — `cur = pre.next`（区间第一个节点，反转后变尾）。循环 `right - left` 次：
   - 记 `next_node = cur.next`（要摘下的节点）
   - `cur.next = next_node.next`（摘下 next_node）
   - `next_node.next = pre.next`（插到区间头部）
   - `pre.next = next_node`（更新头部）
4. **返回** — `dummy.next`

## 🔢 手动走一遍

输入: `head = 1 -> 2 -> 3 -> 4 -> 5`, `left = 2`, `right = 4`

```
初始: dummy -> 1 -> 2 -> 3 -> 4 -> 5
       pre 在 1 (走 left-1=1步), cur 在 2

区间: [2,3,4] 需要反转成 [4,3,2]
需做 right-left = 2 次头插

=== 第1次头插 (摘 3, 插入到头部) ===
     dummy -> 1 -> 2 -> 3 -> 4 -> 5
                  pre  cur
     next_node = cur.next = 3
     cur.next = next_node.next = 4      (2->4)
     next_node.next = pre.next = 2      (3->2)
     pre.next = next_node = 3           (1->3)
     现在: dummy -> 1 -> 3 -> 2 -> 4 -> 5

=== 第2次头插 (摘 4, 插入到头部) ===
     dummy -> 1 -> 3 -> 2 -> 4 -> 5
                  pre      cur
     next_node = cur.next = 4
     cur.next = next_node.next = 5      (2->5)
     next_node.next = pre.next = 3      (4->3)
     pre.next = next_node = 4           (1->4)
     现在: dummy -> 1 -> 4 -> 3 -> 2 -> 5
```

结果: `1 -> 4 -> 3 -> 2 -> 5`

## 💻 解法

> 头插法原地反转区间，不需要额外空间。

```python
from typing import Optional


# 定义链表节点
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next


class Solution:
    def reverseBetween(self, head: Optional[ListNode], left: int, right: int) -> Optional[ListNode]:
        """
        反转链表中从 left 到 right 位置的节点。
        使用虚拟头节点和头插法。
        """
        if not head or left == right:
            return head

        # 创建虚拟头节点，简化边界处理
        dummy = ListNode(0)
        dummy.next = head

        # 找到 left 的前一个节点 pre
        pre = dummy
        for _ in range(left - 1):
            pre = pre.next

        # 反转区间 [left, right]
        cur = pre.next  # 反转区间的第一个节点
        for _ in range(right - left):
            # 保存当前节点的下一个节点
            next_node = cur.next
            # 将 cur 的 next 指向 next_node 的 next
            cur.next = next_node.next
            # 将 next_node 插入到区间头部
            next_node.next = pre.next
            # 更新 pre.next 指向新的头部
            pre.next = next_node

        return dummy.next
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(n) | 找 pre 走 left 步 O(n)，头插 right-left 次 O(n)，总计 O(n) |
| 空间 | O(1) | 原地操作，只用了几个指针变量 |

## ⚠ 易错点

- [ ] 头插法中三行指针赋值顺序不能乱：必须先 `cur.next = next_node.next`（保存后面的链），再 `next_node.next = pre.next`（指向当前头部），最后 `pre.next = next_node`（更新头部）
- [ ] 循环次数是 `right - left` 次，不是 `right - left + 1`。区间第一个节点 `cur` 始终保持不动（它会自然沉到底部成为新尾部）
- [ ] `cur` 在头插过程中始终指向区间反转后的尾部节点，不要移动 `cur`
- [ ] `left == right` 时不需要反转，直接返回 `head`（否则循环 0 次也能正常工作但提前返回更优）
- [ ] 必须使用 `dummy` 节点，因为 `left = 1` 时头节点会改变

```text
这道题是反转区间链表。核心做法是头插法（也叫穿针引线）。

首先用 dummy 节点统一处理 left=1 的情况。从 dummy 出发走
left-1 步找到 pre（区间前驱节点），cur 指向区间第一个节点。

然后进行 right-left 次操作：每次把 cur 后面的节点（即

然后把 next_node 插入到 pre 之后作为新头部。

最终成为反转后区间的尾部。

时间 O(n)，空间 O(1)，原地反转。
```
