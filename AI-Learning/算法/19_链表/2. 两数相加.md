# 2. 两数相加

## 🎯 人话翻译

> 两个整数用链表逆序存储（个位在头），把它们相加后以同样的逆序链表返回。比如 `342` 表示为 `2 -> 4 -> 3`，`465` 表示为 `5 -> 6 -> 4`，相加得 `807`，返回 `7 -> 0 -> 8`。

## 💡 核心思想

> **模拟竖式加法。** 同时遍历两条链表，逐位相加并加上进位 `carry`。当前位 = `总和 % 10`，进位 = `总和 // 10`。当两条链表都走完且进位为 0 时结束。使用 [[虚拟头节点]] 简化开头处理。

用到：[[链表]] [[虚拟头节点]]

## 📝 解题步骤

1. **初始化** — `dummy` 虚拟头节点，`cur = dummy`，`carry = 0`
2. **逐位相加** — 当 `l1` 或 `l2` 非空或 `carry > 0` 时，取当前位的值（空节点按 0），计算 `total = val1 + val2 + carry`
3. **创建新节点** — `cur.next = ListNode(total % 10)`，更新 `carry = total // 10`
4. **移动指针** — `cur` 前移，`l1` 和 `l2` 若非空则前移
5. **返回** — `dummy.next`

## 🔢 手动走一遍

输入: `l1 = 2 -> 4 -> 3` (342), `l2 = 5 -> 6 -> 4` (465)

```
初始: dummy -> (空), carry=0

步骤1: val1=2, val2=5, total=7, carry=0
       cur.next = 7, l1->4, l2->6
       dummy -> 7

步骤2: val1=4, val2=6, total=10, carry=1
       cur.next = 0, l1->3, l2->4
       dummy -> 7 -> 0

步骤3: val1=3, val2=4, total=3+4+1=8, carry=0
       cur.next = 8, l1=None, l2=None
       dummy -> 7 -> 0 -> 8

循环结束 (l1=None, l2=None, carry=0)
```

结果: `7 -> 0 -> 8`，即 807

## 💻 解法

> 模拟竖式加法，逐位相加并处理进位。

```python
# 定义链表节点
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next


class Solution:
    def addTwoNumbers(self, l1: ListNode, l2: ListNode) -> ListNode:
        """
        将两个逆序存储的链表数字相加，返回逆序链表。
        算法：模拟竖式加法，逐位相加并处理进位。
        """
        dummy = ListNode(0)  # 虚拟头节点
        cur = dummy
        carry = 0

        # 当 l1 或 l2 非空，或还有进位时继续
        while l1 or l2 or carry:
            # 获取当前位的值，若节点为空则视为 0
            val1 = l1.val if l1 else 0
            val2 = l2.val if l2 else 0

            total = val1 + val2 + carry
            cur.next = ListNode(total % 10)  # 当前位
            carry = total // 10  # 进位

            # 移动指针
            cur = cur.next
            if l1:
                l1 = l1.next
            if l2:
                l2 = l2.next

        return dummy.next
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(max(m, n)) | 遍历较长链表的所有节点 |
| 空间 | O(max(m, n)) | 新创建的结果链表节点数等于最长链表长度（最多 +1 进位） |

## ⚠ 易错点

- [ ] 循环条件必须是 `l1 or l2 or carry`，三者都满足才退出。容易漏掉 `carry`：如 `99 + 1`，两条链表都走完但还有进位 1
- [ ] 空节点取值为 0：`val1 = l1.val if l1 else 0`，不能直接 `l1.val`
- [ ] 必须先创建新节点再移动 `cur`，否则尾节点的 next 关系会丢失
- [ ] 移动 `l1` 和 `l2` 前要判断它们是否为空，否则 `l1 = l1.next` 会报错（虽然 while 保证至少一个非空，但某个可能已空）
- [ ] 返回的是 `dummy.next` 而不是 `dummy`

```text
这道题本质就是模拟竖式加法，只不过数字以链表逆序存储——这
恰好对齐了我们做加法时从低位到高位的顺序。

做法是同时遍历两条链表，每次取出当前位的数字（空节点视为
0），加上进位值。当前位的结果 = total % 10，新的进位 =
total // 10。创建结果节点接到尾部。

```
