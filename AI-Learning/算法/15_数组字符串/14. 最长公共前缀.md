# 14. 最长公共前缀

## 🎯 人话翻译

> 给一堆字符串，找出它们共同的最长前缀。

## 💡 核心思想

> **纵向扫描：以第一个字符串为基准，逐列比较所有字符串的同一位置字符。**
>
> 最长公共前缀不可能超过最短字符串的长度。外层遍历第一个字符串的每个字符位置，内层检查其余所有字符串在该位置是否也有相同的字符。
>
> 一旦发现某个字符串长度不够或字符不匹配，立即返回当前已匹配的前缀。

用到：[[字符串]]

## 📝 解题步骤

1. **处理边界** — 若数组为空，直接返回空字符串
2. **选基准** — 以第一个字符串作为参考
3. **逐列扫描** — 遍历第一个字符串的每个字符位置 i
4. **逐行验证** — 检查其余每个字符串：若长度 <= i 或 `other[i] != first[i]`，返回 `first[:i]`
5. **全部匹配** — 若遍历完第一个字符串都没触发返回，则整个 first 就是公共前缀

## 🔢 手动走一遍

输入: `strs = ["flower", "flow", "flight"]`

| i | first[i] | 检查 "flow"         | 检查 "flight"       | 结果          |
|---|----------|---------------------|--------------------|--------------|
| 0 | 'f'      | other[0]='f', 匹配   | other[0]='f', 匹配 | 继续         |
| 1 | 'l'      | other[1]='l', 匹配   | other[1]='l', 匹配 | 继续         |
| 2 | 'o'      | other[2]='o', 匹配   | other[2]='i', 不匹配! | 返回 "fl" |

结果: `"fl"`

## 💻 解法

> 纵向扫描：逐列比较，一列不匹配就停止。

```python
from typing import List


class Solution:
    def longestCommonPrefix(self, strs: List[str]) -> str:
        if not strs:  # 空数组返回空字符串
            return ""

        # 以第一个字符串作为参考
        first = strs[0]

        # 遍历第一个字符串的每个字符
        for i, ch in enumerate(first):
            # 检查其他字符串的相同位置
            for other in strs[1:]:
                # 如果其他字符串长度不足 i，或者字符不相等
                if i >= len(other) or other[i] != ch:
                    return first[:i]  # 返回已匹配的前缀

        # 所有字符都匹配，返回整个第一个字符串
        return first
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(S) | S 为所有字符串的字符总数。最坏情况：所有字符串完全相同，需要检查每个字符 |
| 空间 | O(1) | 只用了几个变量，返回的字符串不计入额外空间 |

## ⚠ 易错点

- [ ] 空数组 `[]` 必须返回 `""`，不是 `None` 或崩溃
- [ ] 判断条件要在检查 `other[i]` 之前先检查 `i >= len(other)`——否则索引越界。顺序很重要：先判断长度，再访问字符
- [ ] 如果用横向扫描（两两比较），需要注意每轮都缩减前缀长度，时间复杂度不变但代码长一些
- [ ] 如果数组只有一个字符串，返回整个字符串即可

```text
这道题我用纵向扫描法。

以第一个字符串作为基准，逐列比较所有字符串在第 i 个位置的字符。

```
