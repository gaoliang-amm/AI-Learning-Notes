# 26. 删除有序数组中的重复项

## 🎯 人话翻译

> 给你一个升序数组，原地去重（每个元素只留一次），返回新长度。

## 💡 核心思想

> **双指针，快慢指针。慢指针指向已去重区域的末尾，快指针在前探路。**
>
> 因为数组有序，重复元素必然相邻。快指针遇到与慢指针不同的值时，说明发现了新元素，将其放到慢指针的下一个位置。
>
> 本质是：用快指针"扫描"，用慢指针"写入"。

用到：[[双指针]] [[原地算法]]

## 📝 解题步骤

1. **处理边界** — 若数组为空，直接返回 0
2. **初始化慢指针** — `slow = 0`，指向第一个元素（已去重区域的最后一个位置）
3. **快指针遍历** — `fast` 从 1 到 n-1，扫描每个元素
4. **发现新元素** — 当 `nums[fast] != nums[slow]` 时，`slow++`，将 `nums[fast]` 赋值到 `nums[slow]`
5. **返回长度** — `slow + 1`，因为 slow 是下标，长度需要 +1

结果: 返回 `slow + 1 = 5`，前 5 个元素为 `[0, 1, 2, 3, 4]`

## 💻 解法

> 双指针：快指针扫描，慢指针写入。

```python
from typing import List


class Solution:
    def removeDuplicates(self, nums: List[int]) -> int:
        # 处理边界情况：空数组直接返回0
        if not nums:
            return 0

        # 慢指针，初始指向第一个元素（第一个唯一元素的位置）
        slow = 0
        # 快指针从第二个元素开始遍历
        for fast in range(1, len(nums)):
            # 若当前元素与慢指针位置元素不同，说明是新的唯一元素
            if nums[fast] != nums[slow]:
                slow += 1  # 慢指针右移，准备存新元素
                nums[slow] = nums[fast]  # 赋值，覆盖原位置（原地修改）
        # 最终slow是最后一个唯一元素的索引，长度需要+1
        return slow + 1
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(n) | 快指针遍历一遍数组，每个元素访问一次 |
| 空间 | O(1) | 只用了两个指针变量，原地修改 |

## ⚠ 易错点

- [ ] 返回的是 `slow + 1`（长度），不是 `slow`（下标），忘记 +1 会少一个元素
- [ ] 空数组要单独处理，否则 `slow = 0` 直接返回 1 就错了（应返回 0）
- [ ] 不要写成 `nums[slow] = nums[fast]` 后又把 `slow` 赋给了错误的值——顺序是：先 `slow += 1`，再赋值
- [ ] 此题要求原地修改，不能新建数组或使用 set，面试官会明确强调

因为是升序数组，重复元素必然相邻，所以可以用快慢指针：

- 快指针 fast 扫描整个数组
