# 68. 文本左右对齐

## 🎯 人话翻译

> 给定单词列表和每行最大宽度 maxWidth。排版成左右对齐的文本：每行尽可能多放单词，单词间均匀分配空格（不能均分时左边多分）。最后一行左对齐。

## 💡 核心思想

> **逐行贪心选词 + 空格均匀分配。先决定每行放哪些单词（长度之和+最少空格数 <= maxWidth），再根据行的类型（最后一行、单单词行、普通行）分配空格。**
>
> 三种行类型：
> 1. **最后一行**：单词间一个空格，剩余空格全部放在行尾（左对齐）
> 2. **单单词行**：单词后面补空格到 maxWidth
> 3. **普通行**：计算总空格数，均匀分配到单词间隙中，多余的空格分给左边间隙

用到：[[字符串]] [[模拟]]

## 📝 解题步骤

1. **贪心选词** — 用 while 循环处理，每次尽可能多地将单词放入当前行：
   - `cur_len + len(word) + len(row_words) <= maxWidth`（行内单词总长度 + 最少空格数）
2. **判断行类型并生成** — 
   - 最后一行：单词间 1 空格，剩余空格补末尾
   - 单单词行：单词 + 剩余空格
   - 普通行：总空格 `total_spaces = maxWidth - cur_len`，间隔数 `gaps = num_words - 1`，每个间隔 `base = total_spaces // gaps`，前 `extra = total_spaces % gaps` 个间隔各多 1 空格
3. **返回结果列表**

## 🔢 手动走一遍

输入: `words = ["This", "is", "an", "example", "of", "text", "justification."], maxWidth = 16`

**选词过程：**

| 行 | 尝试添加的单词 | 总长度+空格 | <=16? | 行单词 | 行类型 |
|----|--------------|-----------|-------|--------|--------|
| 1 | This(4)+is(2)+an(2)+... | 4+2+2=8, +2空格=10 | 是 | ["This","is","an"] | 普通 |
| 2 | example(7)+of(2)+... | 7+2=9, +1空格=10 | 是 | ["example","of","text"] | 普通 |
| 3 | justification.(14) | 14 | 是 | ["justification."] | 最后一行 |

**空格分配：**

行 1: `["This", "is", "an"]`，总长度 8，总空格 16-8=8，2 个间隙
- base = 8/2 = 4，extra = 8%2 = 0
- 每个间隙 4 空格: `"This    is    an"`

行 2: `["example", "of", "text"]`，总长度 13，总空格 16-13=3，2 个间隙
- base = 3/2 = 1，extra = 3%2 = 1
- 前 1 个间隙 2 空格(1+1)，后 1 个间隙 1 空格: `"example  of text"`

行 3: `["justification."]`，总长度 14，最后一行左对齐
- `"justification.  "`（后面补 2 空格）

结果: `["This    is    an", "example  of text", "justification.  "]`

## 💻 解法

> 逐行贪心选词，然后根据行类型分配空格。

```python
from typing import List


class Solution:
    def fullJustify(self, words: List[str], maxWidth: int) -> List[str]:
        res = []
        i, n = 0, len(words)

        while i < n:
            # 1. 贪心选择当前行能容纳的单词区间
            row_words = []
            cur_len = 0  # 当前行单词总长度（不含空格）
            # 持续添加单词，直到再加一个单词会超过宽度
            while i < n and cur_len + len(words[i]) + len(row_words) <= maxWidth:
                row_words.append(words[i])
                cur_len += len(words[i])
                i += 1

            # 2. 判断当前行类型并生成该行文本
            num_words = len(row_words)

            # 情况A：最后一行（行内单词包含最后一个单词）
            if i == n:
                # 最后一行，单词间一个空格，剩余空格全在末尾
                line = ' '.join(row_words)
                line += ' ' * (maxWidth - len(line))
                res.append(line)
                break  # 最后一行处理完直接结束

            # 情况B：只有一个单词的行
            if num_words == 1:
                line = row_words[0] + ' ' * (maxWidth - len(row_words[0]))
                res.append(line)
                continue

            # 情况C：普通行（多单词，非最后一行）
            total_spaces = maxWidth - cur_len
            gaps = num_words - 1
            base_spaces = total_spaces // gaps  # 每个间隔至少分配的空格数
            extra_spaces = total_spaces % gaps  # 前 extra_spaces 个间隔各多一个空格

            parts = []
            for j in range(gaps):
                parts.append(row_words[j])
                # 当前间隔应分配的空格数
                spaces = base_spaces + (1 if j < extra_spaces else 0)
                parts.append(' ' * spaces)
            parts.append(row_words[-1])  # 最后一个单词
            line = ''.join(parts)
            res.append(line)

        return res
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(n) | 每个单词只处理一次 |
| 空间 | O(n) | 结果列表存储所有行 |

## ⚠ 易错点

- [ ] 三种行类型的判断顺序：必须先判断是否为最后一行（`i == n`），再判断是否为单单词行，最后才是普通行
- [ ] 判断是否还能加单词的条件：`cur_len + len(word) + len(row_words) <= maxWidth`。`len(row_words)` 是"至少需要的空格数"（每两个单词间至少一个空格）
- [ ] 多余空格只能分配给左边的间隙——`extra_spaces` 个前面的间隙各多 1 个空格。不能平均分给所有间隙
- [ ] 最后一行用 `' '.join(row_words)` 后还要补齐尾部空格，不是均匀分配空格
