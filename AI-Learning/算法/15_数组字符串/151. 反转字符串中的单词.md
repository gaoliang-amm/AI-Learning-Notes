# 151. 反转字符串中的单词

## 🎯 人话翻译

> 把字符串里的单词顺序反过来，单词间用一个空格分隔，去掉首尾和多余空格。

## 💡 核心思想

> **Python 内置：`split()` 自动按空白字符分割并去除空项，`[::-1]` 反转列表，`' '.join()` 重新拼接。**
>
> `split()` 不带参数时，会按任意连续空白字符分割（空格、tab、换行），并自动去除首尾空格——这一行做了三件事：拆单词、去首尾空格、去多余空格。
>
> 面试官如果不让用内置方法：需要手动实现——先去除首尾空格，再从后往前遍历，提取每个单词并拼接。或者：先整体反转字符串，再逐个反转每个单词。

用到：[[字符串]] [[双指针]]

## 📝 解题步骤

**方法一（Python 内置）：**

1. `s.split()` 自动分割单词，去除所有首尾和多余空格
2. `[::-1]` 切片反转单词列表
3. `' '.join()` 用单个空格拼接

**方法二（手动实现，O(1) 额外空间思路）：**

1. 去除首尾空格
2. 从后往前遍历，找到每个单词的起止位置
3. 逐个追加单词和空格到结果

## 🔢 手动走一遍

输入: `s = "  the sky is  blue "`

**Python 方法：**

| 步骤 | 操作 | 结果 |
|------|------|------|
| 1 | `s.split()` | `['the', 'sky', 'is', 'blue']` |
| 2 | `[::-1]` | `['blue', 'is', 'sky', 'the']` |
| 3 | `' '.join()` | `"blue is sky the"` |

**手动方法（从后往前）：**

| 指针位置 | 动作 | 当前单词 | 结果列表 |
|----------|------|----------|----------|
| 末尾 ' ' | 跳过空格 | - | [] |
| 'e' | 开始扫描 | "blue" | [] |
| ' ' 遇到 | 结束该单词 | - | ["blue"] |
| 继续跳过空格 | - | - | ["blue"] |
| 's' | 开始扫描 | "is" | ["blue"] |
| ' ' 遇到 | 结束该单词 | - | ["blue", "is"] |
| ... | ... | ... | ["blue", "is", "sky", "the"] |

结果: `"blue is sky the"`

## 💻 解法一（Python 内置）

> 利用 `split()` 的特性：默认按任意空白字符分割，自动去多余空格。

```python
class Solution:
    def reverseWords(self, s: str) -> str:
        # s.split() 默认按任意空白字符分割，自动去除多余空格
        # [::-1] 反转列表，' '.join() 用单个空格拼接
        return ' '.join(s.split()[::-1])
```

## 🚀 解法二（手动实现，双指针）

> 手动从后往前找到每个单词，用双指针划定单词边界。

```python
class Solution:
    def reverseWords(self, s: str) -> str:
        # 去除首尾空格
        s = s.strip()
        res = []
        i = len(s) - 1

        while i >= 0:
            # 跳过单词间的空格
            while i >= 0 and s[i] == ' ':
                i -= 1
            if i < 0:
                break

            # 找到单词的起始位置
            j = i
            while j >= 0 and s[j] != ' ':
                j -= 1

            # 提取单词（j+1 到 i）
            res.append(s[j + 1:i + 1])
            i = j

        return ' '.join(res)
```

## ⏱ 复杂度

| 版本 | 时间 | 空间 | 说明 |
|------|------|------|------|
| 内置法 | O(n) | O(n) | split 创建了新列表 |
| 手动法 | O(n) | O(n) | 结果列表占 O(n) 空间 |

## ⚠ 易错点

- [ ] 使用 `split(' ')`（带参数空格）不会合并多个连续空格——必须用无参 `split()` 或 `split()` 才能自动去除多余空白
- [ ] 首尾也可能有空格，`split()` 自动处理，手动实现需要先 `strip()` 或跳过
- [ ] 反转后单词间只能有**一个**空格，`' '.join()` 实现
- [ ] 手动遍历时注意 `j` 指向空格或 -1 的情况，单词范围是 `[j+1, i]`

## 🎤 面试怎么说

```text
这道题的 Python 解法非常简洁：`' '.join(s.split()[::-1])`。

`s.split()` 不带参数时，会按任意连续空白字符分割，自动合并多余空格并去掉首尾空格。
然后 `[::-1]` 反转单词顺序，`' '.join()` 用单个空格拼接。

时间复杂度 O(n)，空间 O(n)——因为 split 创建了新的列表。

如果面试官要求不能使用 split，那就手动实现：从后往前遍历，用双指针找到每个单词的

```
