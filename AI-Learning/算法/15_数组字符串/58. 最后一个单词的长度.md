# 58. 最后一个单词的长度

## 🎯 人话翻译

> 字符串里有单词和空格，找出最后一个单词的长度。

## 💡 核心思想

> **从后往前扫。先跳过末尾空格，再统计非空格字符直到遇到空格或字符串开头。**
>
> 从字符串末尾开始：第一阶段跳过所有空格（处理尾随空格），第二阶段逐字符计数直到遇到下一个空格（最后一个单词结束）。

用到：[[字符串]] [[双指针]]

## 📝 解题步骤

1. **指针初始化** — `i = len(s) - 1`，指向最后一个字符
2. **跳过末尾空格** — while 循环向左移动，直到遇到非空格字符
3. **统计最后一个单词** — 继续向左移动，计数非空格字符，直到遇到空格或 i < 0
4. **返回长度** — 就是计数值

## 🔢 手动走一遍

输入: `s = "   fly me   to   the moon  "`

| 步骤 | i | s[i] | 操作 | length |
|------|---|------|------|--------|
| 初始 | 27 | ' ' | 末尾空格，i-- | 0 |
| 跳过末尾空格 | 26 | ' ' | 末尾空格，i-- | 0 |
| 跳过末尾空格 | 25 | 'n' | 不是空格，停止跳过 | 0 |
| 统计单词 | 25 | 'n' | length++，i-- | 1 |
| 统计单词 | 24 | 'o' | length++，i-- | 2 |
| 统计单词 | 23 | 'o' | length++，i-- | 3 |
| 统计单词 | 22 | 'm' | length++，i-- | 4 |
| 统计单词 | 21 | ' ' | 遇到空格，停止 | 4 |

结果: `4`（单词 "moon"）

## 💻 解法

> 从后往前扫描：先跳过末尾空格，再统计最后一个单词长度。

```python
class Solution:
    def lengthOfLastWord(self, s: str) -> int:
        # 指针从最后一位开始
        i = len(s) - 1
        # 先跳过末尾所有空格
        while i >= 0 and s[i] == ' ':
            i -= 1
        # 开始统计最后一个单词长度
        length = 0
        while i >= 0 and s[i] != ' ':
            length += 1
            i -= 1
        return length
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(n) | 最坏情况遍历整个字符串（全是空格或全无空格） |
| 空间 | O(1) | 只用了两个变量 i 和 length |

## ⚠ 易错点

- [ ] 字符串末尾可能有很多空格（如 `"Hello World   "`），必须**先跳过所有末尾空格**，直接统计会得到 0
- [ ] 不要忘了处理全空格字符串——但题目保证至少有一个单词，所以实际上不会出现
- [ ] 另一种简便写法 `s.split()[-1]` 返回最后一个单词再取 `len()`，面试时用 split 也算 OK，但要说明 O(n) 空间
- [ ] 注意 while 循环的条件顺序：先判断 `i >= 0` 再判断 `s[i]`，防止越界
