# 42. 接雨水

## 🎯 人话翻译

> 二维柱子高度图，每个柱子宽 1。问下雨后柱子之间能积多少水。

## 💡 核心思想

> **双指针法：每个位置的积水量 = min(左边最高, 右边最高) - 当前高度。用双指针从两端向中间收缩，每次移动较矮的一侧。**
>
> 对于位置 i，它能积的水量由"它两边的最高柱子中较矮的那个"决定（短板效应）。
>
> 双指针 `left` 和 `right` 分别指向数组两端，维护 `left_max`（左边最高）和 `right_max`（右边最高）。每次比较 `height[left]` 和 `height[right]`：
> - 如果 `height[left] < height[right]`：左侧更矮，处理左指针。此时 `right_max` >= `height[right]` > `height[left]`，所以 `left_max` 就是短板。积水量 = `left_max - height[left]`。
> - 否则处理右指针，积水量 = `right_max - height[right]`。

用到：[[双指针]] [[单调栈]]

## 📝 解题步骤

**方法一：双指针**

1. 初始化 `left=0, right=n-1, left_max=0, right_max=0, water=0`
2. while `left < right`：
   - 若 `height[left] < height[right]`：处理左侧
     - 若 `height[left] >= left_max`：更新 `left_max = height[left]`
     - 否则：`water += left_max - height[left]`
     - `left++`
   - 否则：处理右侧
     - 若 `height[right] >= right_max`：更新 `right_max = height[right]`
     - 否则：`water += right_max - height[right]`
     - `right--`
3. 返回 water

**方法二：前缀/后缀最大值**

1. 计算每个位置的 `prefix_max[i]` = 左边最高柱高
2. 计算每个位置的 `suffix_max[i]` = 右边最高柱高
3. 遍历：`water += min(prefix_max[i], suffix_max[i]) - height[i]`（若为正）

## 💻 解法一（双指针，O(1) 空间）

> 两端向中间收缩，每次移动较矮的一侧。

```python
from typing import List


class Solution:
    def trap(self, height: List[int]) -> int:
        if not height:
            return 0

        left, right = 0, len(height) - 1
        left_max = right_max = 0
        water = 0

        while left < right:
            # 左侧高度小于右侧，处理左指针
            if height[left] < height[right]:
                if height[left] >= left_max:
                    left_max = height[left]
                else:
                    water += left_max - height[left]
                left += 1
            else:
                # 右侧高度小于等于左侧，处理右指针
                if height[right] >= right_max:
                    right_max = height[right]
                else:
                    water += right_max - height[right]
                right -= 1

        return water
```

## 🚀 解法二（前缀/后缀最大值，O(n) 空间）

> 分别算每个位置的左边最高和右边最高。

```python
class Solution:
    def trap(self, height: List[int]) -> int:
        n = len(height)
        if n == 0:
            return 0

        # 前缀最大值
        prefix = [0] * n
        prefix[0] = height[0]
        for i in range(1, n):
            prefix[i] = max(prefix[i-1], height[i])

        # 后缀最大值
        suffix = [0] * n
        suffix[-1] = height[-1]
        for i in range(n-2, -1, -1):
            suffix[i] = max(suffix[i+1], height[i])

        # 累加每个位置的积水
        water = 0
        for i in range(n):
            water += max(0, min(prefix[i], suffix[i]) - height[i])

        return water
```

## 🚀 解法三（单调栈）

> 遇到上升沿时计算凹槽积水量。

```python
class Solution:
    def trap(self, height: List[int]) -> int:
        stack = []  # 单调递减栈，存放下标
        water = 0

        for i, h in enumerate(height):
            # 当前高度大于栈顶高度：形成了凹槽
            while stack and h > height[stack[-1]]:
                bottom = stack.pop()  # 凹槽底部
                if not stack:
                    break  # 左边没有更高的，无法积水
                # 计算积水：宽度 * 高度
                left = stack[-1]
                width = i - left - 1
                depth = min(height[left], h) - height[bottom]
                water += width * depth
            stack.append(i)

        return water
```

## ⏱ 复杂度

| 版本 | 时间 | 空间 | 说明 |
|------|------|------|------|
| 双指针 | O(n) | O(1) | 面试首选，最优 |
| 前后缀 | O(n) | O(n) | 容易理解和一次性写对 |
| 单调栈 | O(n) | O(n) | 更通用，能处理三维接雨水 |

## ⚠ 易错点

- [ ] 双指针法中，移动较矮一侧的逻辑：如果 `height[left] < height[right]`，处理 left。因为右侧至少有一个更高的柱子，积水由 `left_max` 决定
- [ ] 更新 `left_max` 的时机：必须先判断当前高度是否 >= left_max，如果更大则更新 left_max（成为新边界），否则才计算积水。顺序不能反
- [ ] 双指针循环条件是 `left < right`（严格小于），当两指针相遇时退出。不要用 `<=`，否则会重复计算
- [ ] 单调栈法需要理解"遇到上升沿形成凹槽"——栈存的是递减序列的下标，当遇到一个比栈顶高的柱子时，弹出栈顶作为凹槽底，左边界是新的栈顶
- [ ] 三种方法都能解，面试时先讲前后缀法（最直观），再优化到双指针法（O(1) 空间），有必要时提单调栈
