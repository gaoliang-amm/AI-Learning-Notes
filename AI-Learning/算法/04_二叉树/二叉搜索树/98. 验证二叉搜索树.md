# 98. 验证二叉搜索树

## 🎯 人话翻译

> 给你一棵二叉树，判断它是不是一棵有效的 BST。有效条件：左子树所有节点 < 根 < 右子树所有节点（不只是直接左右孩子），左右子树也必须各自是 BST。

## 💡 核心思想

> **方法一（中序遍历）：BST 的中序遍历一定是严格递增序列。用 `prev` 记录上一个节点的值，如果出现当前值 <= 上一个值，就不是 BST。**
>
> 方法二（上下界传递）：每个节点有一个合法的值范围 `(min_val, max_val)`。根的范围是 `(-inf, +inf)`。往左走，上界变成当前节点值；往右走，下界变成当前节点值。
>
> 中序遍历法的优势：代码简洁，不需要维护区间，只需要一个 `prev` 变量。

用到：[[二叉搜索树]]、[[中序遍历]]、[[二叉树]]、[[递归]]

## 📝 解题步骤

**方法一（中序遍历 + prev）：**

1. **初始化** — `self.prev = float('-inf')`，记录上一个节点的值
2. **中序遍历** — 递归：左 → 当前 → 右
3. **检查递增** — 如果 `node.val <= self.prev`，说明不是严格递增，返回 `False`
4. **更新 prev** — `self.prev = node.val`
5. **遍历结束未发现违规** — 返回 `True`

**方法二（上下界传递）：**

1. **定义递归函数** — `validate(node, min_val, max_val)`，当前节点值必须在 `(min_val, max_val)` 之间
2. **空节点** — 返回 `True`（空树是 BST）
3. **检查当前值** — 如果 `node.val <= min_val` 或 `node.val >= max_val`，返回 `False`
4. **递归检查左右** — 左子树范围 `(min_val, node.val)`，右子树范围 `(node.val, max_val)`

## 🔢 手动走一遍

以下面的树为例（它不是有效的 BST，因为右子树中的 3 小于根 5）：

```
        5
       / \
      1   4
         / \
        3   6
```

**方法一（中序遍历）：**

中序遍历顺序：1 → 5 → 3 → 4 → 6

| 步骤 | 当前值 | prev  | 判断                  | 结果     |
|------|--------|-------|-----------------------|----------|
| 1    | 1      | -inf  | 1 > -inf，通过         | 继续     |
| 2    | 5      | 1     | 5 > 1，通过            | 继续     |
| 3    | 3      | 5     | **3 <= 5，违规！**     | 返回 False |

**方法二（上下界）：**

| 节点 | 当前范围    | 判断                    | 结果     |
|------|-----------|-------------------------|----------|
| 5    | (-inf,+inf) | 合法                    | 继续     |
| 1    | (-inf,5)   | 1 在范围内               | 返回 True |
| 4    | (5,+inf)   | 4 < 5 上界，**违规！**   | 返回 False |

## 💻 解法一（中序遍历 + prev 指针）

```python
from typing import Optional


class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class Solution:
    def isValidBST(self, root: Optional[TreeNode]) -> bool:
        # 时间复杂度 O(n)，空间复杂度 O(h)

        self.prev = float('-inf')  # 记录前一个节点的值

        def inorder(node):
            if not node:
                return True

            # 先遍历左子树
            if not inorder(node.left):
                return False

            # 检查当前节点与前一个节点的关系
            if node.val <= self.prev:
                return False
            self.prev = node.val

            # 再遍历右子树
            return inorder(node.right)

        return inorder(root)
```

## 💻 解法二（上下界传递）

```python
class Solution:
    def isValidBST(self, root: Optional[TreeNode]) -> bool:

        def validate(node: Optional[TreeNode], min_val: float, max_val: float) -> bool:
            if not node:
                return True

            # 当前节点的值必须在 (min_val, max_val) 范围内
            if node.val <= min_val or node.val >= max_val:
                return False

            # 左子树：上界变为当前值；右子树：下界变为当前值
            return (validate(node.left, min_val, node.val) and
                    validate(node.right, node.val, max_val))

        return validate(root, float('-inf'), float('inf'))
```

## ⏱ 复杂度

| 版本     | 时间  | 空间      | 说明                       |
|---------|------|-----------|----------------------------|
| 中序遍历 | O(n) | O(h)      | 每个节点访问一次             |
| 上下界   | O(n) | O(h)      | 递归栈深度，平衡树 O(log n) |

## ⚠ 易错点

- [ ] **必须严格递增不是非递减** — BST 要求 `node.val > prev` 不是 `>=`，相等的值也算无效 BST
- [ ] **不只是比较直接左右孩子** — 错误写法 `if node.left and node.left.val >= node.val` 只能检查一层，检查不了孙子节点
- [ ] **中序法 `prev` 的初始值** — 如果用 `float('-inf')`，节点值可能也是 `-inf`，更好的做法是用 `self.prev_val = None` 然后判断 `is None`
- [ ] **边界值的开闭区间** — 上下界法中，左子树必须严格小于根（开区间），右子树必须严格大于根（开区间）
- [ ] **`return False` 的传播** — 中序法中一旦发现违规，要确保 `False` 沿着递归路径透传上去，不能继续遍历

## 🎤 面试怎么说

```text
验证 BST 有两种主流方法：

方法一（推荐）：利用 BST 的中序遍历产生严格递增序列的性质，
用一个 prev 变量记录上一个访问的节点值，如果出现 "当前值 <= 上一个值"
就说明不是 BST。

方法二：上下界传递法。每个节点有一个合法范围 (min, max)，
根是 (-inf, +inf)。往左走，上界变为当前值；往右走，下界变为当前值。

两种方法时间 O(n)，空间 O(h)。

面试中推荐先说方法一，代码更简洁。方法二作为补充说明。
常见错误是只比较直接左右孩子，忽略了孙子层级的约束。
```
