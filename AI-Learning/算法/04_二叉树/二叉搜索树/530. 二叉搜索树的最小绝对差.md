# 530. 二叉搜索树的最小绝对差

## 🎯 人话翻译

> 给你一个 BST，求任意两个不同节点值之间差的最小绝对值。因为 BST 的中序遍历是严格递增的，所以最小值一定出现在某个相邻的对子里。

## 💡 核心思想

> **BST 中序遍历产生升序序列，最小差值必定出现在相邻的两个节点之间。用一个 `prev` 指针记录上一个访问的节点，在中序遍历中动态计算差值。**
>
> 为什么中序相邻一定包含最小差？
> 升序序列中，任意两个元素的最小差一定出现在相邻位置（非相邻的两个元素之差一定大于或等于它们之间的相邻差之和）。
>
> 所以不需要比较所有节点对，只需在遍历过程中比较当前节点和上一个节点的差值。

用到：[[二叉搜索树]]、[[中序遍历]]、[[二叉树]]、[[递归]]

## 📝 解题步骤

1. **初始化** — `self.prev = None`（上一个被访问的节点），`self.min_diff = float('inf')`
2. **中序遍历** — 递归：左 → 当前 → 右
3. **处理当前节点** — 如果 `self.prev` 不为空（不是第一个节点），计算 `node.val - self.prev.val` 并更新 `self.min_diff`
4. **更新 prev** — `self.prev = node`
5. **遍历完成后返回 `self.min_diff`**

## 🔢 手动走一遍

以下面这棵 BST 为例：

```
        4
       / \
      2   6
     / \
    1   3
```

中序遍历顺序：1 → 2 → 3 → 4 → 6

| 步骤 | 当前节点 val | prev 值 | 差值 | min_diff 更新 |
|------|-------------|---------|------|---------------|
| 1    | 1           | None    | —    | inf（不更新）  |
| 2    | 2           | 1       | 1    | min(inf,1) = 1 |
| 3    | 3           | 2       | 1    | min(1,1) = 1   |
| 4    | 4           | 3       | 1    | min(1,1) = 1   |
| 5    | 6           | 4       | 2    | min(1,2) = 1   |

最小绝对差 = 1。

## 💻 解法

```python
from typing import Optional, List
from collections import deque


# ================== 二叉树节点定义 ==================
class TreeNode:
    """二叉搜索树的节点"""

    def __init__(self, val=0, left=None, right=None):
        self.val = val  # 节点值
        self.left = left  # 左子节点
        self.right = right  # 右子节点


# ================== 题解代码（递归+双指针） ==================
class Solution:
    def getMinimumDifference(self, root: Optional[TreeNode]) -> int:
        """
        返回二叉搜索树中任意两个不同节点值之间的最小绝对差
        """
        self.prev = None  # 记录中序遍历时的前一个节点
        self.min_diff = float('inf')  # 记录当前找到的最小差值，初始为无穷大

        def inorder(node: Optional[TreeNode]) -> None:
            """中序遍历（左-根-右）"""
            if not node:
                return
            inorder(node.left)  # 左子树
            if self.prev is not None:  # 不是第一个节点
                self.min_diff = min(self.min_diff, node.val - self.prev.val)
            self.prev = node  # 更新前一个节点
            inorder(node.right)  # 右子树

        inorder(root)
        return self.min_diff
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明                                      |
|------|--------|-------------------------------------------|
| 时间 | O(n)   | 中序遍历每个节点访问一次                    |
| 空间 | O(h)   | 递归栈深度，最坏 O(n)（链表状 BST），平均 O(log n) |

## ⚠ 易错点

- [ ] **`prev` 初始化为 `None` 而非 0 或 `-inf`** — 第一个节点没有前驱，不能参与差值计算
- [ ] **`self.prev` 和 `self.min_diff` 要用 `self.`** — 嵌套函数里修改外层变量需要 `self.` 或 `nonlocal`
- [ ] **差值用减法 `node.val - prev.val` 即可** — BST 中序遍历是严格递增的，后面一定大于前面，不需要 `abs()`
- [ ] **BST 的性质保证差值在相邻节点间最小** — 这是用中序遍历解决本题的关键理论依据
