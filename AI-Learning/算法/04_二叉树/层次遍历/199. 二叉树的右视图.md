# 199. 二叉树的右视图

## 🎯 人话翻译

> 想象你站在一棵二叉树的右边，从上往下看，返回你能看到的所有节点值（每层最右边的那个节点）。

## 💡 核心思想

> **层序遍历（BFS），每层只取最后一个节点。**
>
> 右视图 = 层序遍历中，每层的最后一个节点。因为从右边看，每层只能看到最靠右的那个节点。
>
> 实现：在层序遍历的模板中，内层 for 循环遍历每一层的所有节点时，用索引 `i` 判断：
> - 如果 `i == level_size - 1`（当前层的最后一个），把该节点的值加入结果。
>
> 也可以用 DFS（根-右-左的前序），记录当前深度，每层第一个访问到的节点就是右视图节点。但 BFS 更直观。

用到：[[二叉树]]、[[BFS]]、[[层序遍历]]、[[队列]]

## 📝 解题步骤

1. **初始化** — 空树返回 `[]`；否则 `queue = deque([root])`，`result = []`
2. **外层循环** — `level_size = len(queue)`
3. **内层遍历** — `for i in range(level_size)`：
   - 出队节点
   - 如果 `i == level_size - 1`（最后一个），加入 `result`
   - 左右孩子按顺序入队（先左后右，因为需要完整的从左到右排列）
4. **返回 `result`**

## 🔢 手动走一遍

以下面的树为例：

```
        1
       / \
      2   3
     / \   \
    4   5   6
         \
          7
```

层序遍历 + 取右视图：

| 层 | level_size | 遍历节点         | i   | 最后一个(i==ls-1)？ | 取的值 |
|----|-----------|-----------------|-----|-------------------|--------|
| 1  | 1         | 1               | 0   | 是                 | 1      |
| 2  | 2         | 2               | 0   | 否                 | —      |
|    |           | 3               | 1   | 是                 | 3      |
| 3  | 3         | 4               | 0   | 否                 | —      |
|    |           | 5               | 1   | 否                 | —      |
|    |           | 6               | 2   | 是                 | 6      |
| 4  | 1         | 7               | 0   | 是                 | 7      |

右视图结果：`[1, 3, 6, 7]`

## 💻 解法

```python
from typing import List, Optional
from collections import deque


class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class Solution:
    def rightSideView(self, root: Optional[TreeNode]) -> List[int]:
        # 时间复杂度 O(n)，空间复杂度 O(w)

        if not root:
            return []

        result = []
        queue = deque([root])

        while queue:
            level_size = len(queue)

            # 遍历当前层
            for i in range(level_size):
                node = queue.popleft()

                # 如果是当前层的最后一个节点，加入结果
                if i == level_size - 1:
                    result.append(node.val)

                # 左右孩子入队（先左后右，顺序不重要，因为我们只取最后一个）
                if node.left:
                    queue.append(node.left)
                if node.right:
                    queue.append(node.right)

        return result
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明                                    |
|------|--------|-----------------------------------------|
| 时间 | O(n)   | 每个节点访问一次                          |
| 空间 | O(w)   | 队列最多存一层的节点数 w，最坏 O(n)        |

## ⚠ 易错点

- [ ] **判断最后一个节点的逻辑** — 用 `i == level_size - 1` 而不是 `node == last_in_queue`
- [ ] **子节点入队顺序不影响结果** — 只要队列中同一层的节点保持从左到右，最后一个就是最右的节点
- [ ] **不要试图只走右子树** — 如果右子树为空但左子树很深，右视图应该看到左子树的节点，所以必须遍历整棵树的每一层
- [ ] **空树返回 `[]`** — 空树的右视图也是空

这道题是层序遍历的直接应用。

站在右边往下看，每层只能看到最右边的节点。
所以用 BFS 层序遍历，每层取最后一个节点的值即可。
