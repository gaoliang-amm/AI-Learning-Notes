# 637. 二叉树的层平均值

## 🎯 人话翻译

> 给你一棵二叉树，求每层节点值的平均值，按层返回一个浮点数列表。

## 💡 核心思想

> **层序遍历（BFS），每层求和后除以节点数。**
>
> 在 [[102. 二叉树的层序遍历]] 的模板基础上，只需要把"收集节点值"改为"累加求和"，遍历完一层后计算 `level_sum / level_size` 加入结果。
>
> 注意：`level_size` 可能是 0（当前层没有节点），但题目说输入是非空二叉树，所以不会出现除零情况。

用到：[[二叉树]]、[[BFS]]、[[层序遍历]]、[[队列]]

## 📝 解题步骤

1. **初始化** — `res = []`，`queue = deque([root])`
2. **外层循环** — `level_size = len(queue)`，`level_sum = 0`
3. **内层循环** — `for _ in range(level_size)`：
   - 出队节点，`level_sum += node.val`
   - 左右孩子非空则入队
4. **计算平均值** — `res.append(level_sum / level_size)`
5. **返回 `res`**

## 🔢 手动走一遍

以下面的树为例：

```
        3
       / \
      9  20
         / \
        15  7
```

| 层 | level_size | 遍历节点 | level_sum | 平均值     |
|----|-----------|---------|-----------|-----------|
| 1  | 1         | 3       | 3         | 3.0       |
| 2  | 2         | 9, 20   | 9+20=29   | 29/2=14.5 |
| 3  | 2         | 15, 7   | 15+7=22   | 22/2=11.0 |

结果：`[3.0, 14.5, 11.0]`

## 💻 解法

```python
from collections import deque


# Definition for a binary tree node.
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class Solution:
    def averageOfLevels(self, root: TreeNode) -> list[float]:
        res = []
        if not root:
            return res

        q = deque([root])

        while q:
            level_size = len(q)
            level_sum = 0

            # 求当前一层的总和
            for _ in range(level_size):
                node = q.popleft()
                level_sum += node.val

                if node.left:
                    q.append(node.left)
                if node.right:
                    q.append(node.right)

            # 计算平均值并加入结果
            res.append(level_sum / level_size)

        return res
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明                                    |
|------|--------|-----------------------------------------|
| 时间 | O(n)   | 每个节点访问一次                          |
| 空间 | O(w)   | 队列最多存一层的节点数 w，最坏 O(n)        |

## ⚠ 易错点

- [ ] **返回类型是浮点数** — Python 中 `/` 自动得到 float，如果是整数除法用 `/` 不是 `//`
- [ ] **`level_size` 不会为 0** — 因为题目输入非空二叉树，且 while 循环保证队列非空才进入
- [ ] **数据类型注意** — 如果用了 `typing.List[float]` 需要 `from typing import List`
- [ ] **`level_sum` 可能很大** — 但如果节点值在 32 位整数范围内，层总和不会溢出 Python 的 int

这道题是层序遍历的简单变种。

用 BFS 逐层遍历，每层用一个变量累加节点值的总和，
遍历完该层后，总和除以节点数就是该层的平均值，
加入结果即可。

时间 O(n)，空间 O(w)。

这是层序遍历模板的最简单应用，102 题的基础上改两行就行。
