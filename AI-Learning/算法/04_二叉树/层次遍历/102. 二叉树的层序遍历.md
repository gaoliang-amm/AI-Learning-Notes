# 102. 二叉树的层序遍历

## 🎯 人话翻译

> 给你一棵二叉树，按层返回节点值，每一层是一个列表，从根开始从上到下逐层输出。

## 💡 核心思想

> **BFS + 队列。用一个队列存放"待访问的节点"，每轮处理一整层。**
>
> 核心技巧：
> 1. 在外层 while 循环开始前，用 `level_size = len(queue)` 固定当前层的节点数
> 2. 内层 for 循环恰好执行 `level_size` 次，把当前层全部出队
> 3. 出队时把该节点的左右孩子加入队列（下一层）
> 4. 每次出队 `level_size` 次后，队列里剩下的全是下一层的节点
>
> 这个模板是层序遍历的通用骨架，[[199. 二叉树的右视图]]、[[637. 二叉树的层平均值]]、[[103. 二叉树的锯齿形层序遍历]] 都是在这个骨架上稍微改几行。

用到：[[二叉树]]、[[BFS]]、[[层序遍历]]、[[队列]]

## 📝 解题步骤

1. **初始化** — 如果根为空，返回 `[]`；否则 `queue = deque([root])`，`result = []`
2. **外层循环** — 当 `queue` 非空时，记录 `level_size = len(queue)`
3. **创建当前层列表** — `level_values = []`
4. **内层循环** — `for _ in range(level_size)`：
   - 出队节点 `node = queue.popleft()`
   - 将 `node.val` 加入 `level_values`
   - 如果 `node.left` 非空，入队
   - 如果 `node.right` 非空，入队
5. **当前层处理完毕** — 将 `level_values` 加入 `result`
6. **循环直到队列为空** — 返回 `result`

## 🔢 手动走一遍

以下面的树为例：

```
        3
       / \
      9  20
         / \
        15  7
```

层序遍历过程：

| 外层轮次 | level_size | 队列初始        | 内层出队 | level_values | 子节点入队            | 出队后队列   |
|---------|-----------|----------------|---------|-------------|---------------------|------------|
| 1       | 1         | [3]            | 3       | [3]         | 9, 20               | [9, 20]    |
| 2       | 2         | [9, 20]        | 9       | [9, 20]     | —                    | [20, 15, 17] |
|         |           |                | 20      |             | 15, 17              | [15, 17]   |
| 3       | 2         | [15, 7]        | 15      | [15, 7]     | —                    | [7]        |
|         |           |                | 7       |             | —                    | []         |

结果：`[[3], [9, 20], [15, 7]]`

## 💻 解法

```python
from typing import List, Optional
from collections import deque


class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class Solution:
    def levelOrder(self, root: Optional[TreeNode]) -> List[List[int]]:
        # 时间复杂度 O(n)，空间复杂度 O(n)

        if not root:
            return []

        result = []
        queue = deque([root])

        while queue:
            # 记录当前层的节点数（一定要先固定，因为循环中 queue 长度会变）
            level_size = len(queue)
            level_values = []

            # 处理当前层的所有节点
            for _ in range(level_size):
                node = queue.popleft()
                level_values.append(node.val)

                # 将下一层的节点加入队列（先左后右）
                if node.left:
                    queue.append(node.left)
                if node.right:
                    queue.append(node.right)

            # 当前层处理完毕，加入结果
            result.append(level_values)

        return result
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明                                    |
|------|--------|-----------------------------------------|
| 时间 | O(n)   | 每个节点入队出队各一次                    |
| 空间 | O(w)   | 队列最多存一层的节点数 w（最宽层），最坏 O(n) |

## ⚠ 易错点

- [ ] **`level_size = len(queue)` 在 for 循环前取值** — 如果写在 `range(len(queue))` 里，会因为子节点不断入队变成无限循环
- [ ] **子节点入队顺序是"先左后右"** — 保证同一层从左到右排列
- [ ] **空节点不要入队** — 入队前判断 `if node.left` 和 `if node.right`，否则队列里会混入 `None`
- [ ] **`queue` 用 `collections.deque`** — `list.pop(0)` 是 O(n)，`deque.popleft()` 是 O(1)

## 🎤 面试怎么说

```text
这道题是层序遍历的标准模板题，BFS + 队列。

核心思路：用一个队列做 BFS。每轮循环开始前，
先记录当前层的节点数 level_size，
然后恰好出队 level_size 次，这一层的节点就都处理完了。
出队时把左右孩子入队准备下一轮。

这个模板可以轻松变种，比如：
- 右视图: 取每层最后一个
- 锯齿形: 偶数层反转结果
- 层平均值: 每层求和除以 level_size

时间 O(n)，空间 O(w)。
```
