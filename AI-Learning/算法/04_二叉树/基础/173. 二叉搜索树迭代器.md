# 173. 二叉搜索树迭代器

## 🎯 人话翻译

> 实现一个 BST 的迭代器，`next()` 返回当前最小的元素（中序遍历的顺序），`hasNext()` 检查是否还有元素。要求 O(h) 空间（h 是树高）。

## 💡 核心思想

> **用栈模拟中序遍历，把递归转成迭代。初始化时和每次 `next()` 后，把当前节点的所有左子节点（左链）压入栈。**
>
> BST 的中序遍历（左-根-右）产生的序列是递增的，第 k 次调用 `next()` 返回的就是第 k 小的元素。
>
> 关键操作：
> - `_push_left_chain(node)`：从 `node` 开始，沿着左子树一路压栈到底。栈顶就是当前最小值。
> - `next()`：弹出栈顶（当前最小值），如果这个节点有右子树，把右子树的左链压栈（因为右子树的元素介于当前节点和父节点之间）。
>
> 核心洞察：栈始终保存着"下一个要访问的节点"的候选。

用到：[[二叉树]]、[[二叉搜索树]]、[[中序遍历]]、[[栈]]、[[迭代器]]

## 📝 解题步骤

1. **初始化 `_push_left_chain(root)`** — 构造函数中把根节点的整个左链全部压入栈
2. **`hasNext()`** — 判断栈是否非空
3. **`next()`**：
   - 弹出栈顶节点 `node`（当前最小值）
   - 如果 `node` 有右子树，调用 `_push_left_chain(node.right)`，把右子树的左链压栈
   - 返回 `node.val`
4. **`_push_left_chain(node)`** — while 循环，`stack.append(node)` 然后 `node = node.left`

## 🔢 手动走一遍

以下面这棵 BST 为例：

```
        7
       / \
      3  15
         / \
        9  20
```

初始化：压入左链 7 → 3，栈 = [7, 3]（栈顶是 3）

| 操作          | 栈（顶在右）        | 弹出 | 解释                           | 返回值 |
|--------------|-------------------|------|--------------------------------|--------|
| 初始化        | [7, 3]            | —    | 压入 7 的左链：7,3              | —      |
| next() #1    | [7]               | 3    | 弹出 3，3 无右子树               | 3      |
| next() #2    | [15, 9]           | 7    | 弹出 7，7 有右子树(15)，压左链：15,9 | 7      |
| next() #3    | [15]              | 9    | 弹出 9，9 无右子树               | 9      |
| next() #4    | [20]              | 15   | 弹出 15，15 有右子树(20)，压左链：20 | 15     |
| next() #5    | []                | 20   | 弹出 20，20 无右子树             | 20     |

输出顺序：3, 7, 9, 15, 20，正是升序的中序遍历！

## 💻 解法

```python
from typing import Optional


class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class BSTIterator:
    # 时间复杂度：初始化为 O(h)，next() 均摊 O(1)，hasNext() 为 O(1)
    # 空间复杂度：O(h)

    def __init__(self, root: Optional[TreeNode]):
        self.stack = []
        # 将根节点的左链全部压入栈
        self._push_left_chain(root)

    def _push_left_chain(self, node: Optional[TreeNode]):
        """将 node 及其所有左子节点压入栈"""
        while node:
            self.stack.append(node)
            node = node.left

    def next(self) -> int:
        # 弹出栈顶节点（当前最小值）
        node = self.stack.pop()
        # 如果该节点有右子树，将右子树的左链压入栈
        if node.right:
            self._push_left_chain(node.right)
        return node.val

    def hasNext(self) -> bool:
        return len(self.stack) > 0
```

## ⏱ 复杂度

| 项目 | 复杂度           | 说明                                                         |
|------|-----------------|--------------------------------------------------------------|
| 时间 | 构造 O(h)       | 初始压入左链，最多压 h 个节点                                   |
|      | next() 均摊 O(1) | 每个节点恰好入栈一次、出栈一次，n 次 next 总共 O(n)，均摊 O(1) |
|      | hasNext() O(1)  | 仅判断栈是否为空                                               |
| 空间 | O(h)            | 栈中最多保存 h 个节点（树的高度）                               |

## ⚠ 易错点

- [ ] **`next()` 弹出后要处理右子树** — 弹出节点后，如果它有右子树，要把右子树的左链压入栈，否则会丢失右子树的数据
- [ ] **均摊分析** — `next()` 单次可能不是 O(1)（压左链可能压多个），但均摊是 O(1)，面试要说清楚
- [ ] **`_push_left_chain` 要在 `__init__` 和 `next()` 中都调用** — 初始化时压根节点的左链，每次 `next()` 后可能要压右子树的左链
- [ ] **栈顶是当前最小值** — 因为左链压栈保证了栈中的顺序符合中序遍历的"左-根-右"
