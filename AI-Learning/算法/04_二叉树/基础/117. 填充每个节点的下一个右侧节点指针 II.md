# 117. 填充每个节点的下一个右侧节点指针 II

## 🎯 人话翻译

> 给一棵二叉树（不一定是满二叉树），每个节点多一个 `next` 指针，让你把同一层中的节点从左到右用 `next` 串起来。

## 💡 核心思想

> **BFS 层序遍历：同一层的节点依次出队时，前一个出队的节点的 next 指向队列头部（同层的下一个节点）。**
>
> 关键是 `level_size` 变量：在处理每一层之前，先用 `len(queue)` 固定当前层的节点数。
> 对于当前层的每个节点：
> - 如果不是该层最后一个（`i < level_size - 1`），`next` 指向队列头部 `q[0]`
> - 将左右子节点（如果存在）加入队列，等待下一层处理
>
> 与 [[116. 填充每个节点的下一个右侧节点指针]]（完美二叉树）的区别：这题不要求树是完美的，可能有节点缺失。

用到：[[二叉树]]、[[BFS]]、[[层序遍历]]、[[队列]]

## 📝 解题步骤

1. **空树处理** — 如果 `root` 为空，直接返回 `None`
2. **初始化队列** — `queue = deque([root])`
3. **按层循环** — 当队列非空时，记录 `level_size = len(queue)`
4. **遍历当前层** — `for i in range(level_size)`：出队一个节点，如果不是最后一个（`i < level_size - 1`），设 `node.next = q[0]`
5. **将子节点入队** — 如果当前节点有左/右孩子，加入队列尾部
6. **最后返回 `root`** — 整个树的 `next` 指针已原地修改

## 🔢 手动走一遍

以下面的树为例：

```
        1
       / \
      2   3
     / \   \
    4   5   7
```

层序遍历过程：

| 层 | level_size | 处理节点 | i   | i==最后？ | next 指向   | 入队子节点  | 队列状态         |
|----|-----------|---------|-----|----------|-------------|------------|------------------|
| 1  | 1         | 1       | 0   | 是       | None        | 2,3         | [2,3]            |
| 2  | 2         | 2       | 0   | 否       | q[0]=3      | 4,5         | [3,4,5]          |
| 2  | 2         | 3       | 1   | 是       | None        | 7           | [4,5,7]          |
| 3  | 3         | 4       | 0   | 否       | q[0]=5      | None        | [5,7]            |
| 3  | 3         | 5       | 1   | 否       | q[0]=7      | None        | [7]              |
| 3  | 3         | 7       | 2   | 是       | None        | None        | []               |

最终 next 连接：
- 1 → NULL
- 2 → 3 → NULL
- 4 → 5 → 7 → NULL

## 💻 解法

```python
from collections import deque


# Definition for a Node.
class Node:
    def __init__(self, val: int = 0, left: 'Node' = None, right: 'Node' = None, next: 'Node' = None):
        self.val = val
        self.left = left
        self.right = right
        self.next = next


class Solution:
    def connect(self, root: 'Node') -> 'Node':
        """
        填充每个节点的 next 指针，指向右侧节点
        思路：层序遍历（BFS），同一层节点依次相连
        时间 O(n)，空间 O(n)
        """
        if not root:
            return root

        q = deque([root])

        while q:
            level_size = len(q)

            # 遍历当前层所有节点
            for i in range(level_size):
                node = q.popleft()

                # 不是最后一个，next 指向队列下一个
                if i < level_size - 1:
                    node.next = q[0]

                # 把子节点加入下一层
                if node.left:
                    q.append(node.left)
                if node.right:
                    q.append(node.right)

        return root
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明                           |
|------|--------|--------------------------------|
| 时间 | O(n)   | 每个节点入队出队各一次          |
| 空间 | O(w)   | 队列最多存储一层的节点数 w（最宽层的宽度），最坏 O(n) |

进阶空间优化：可以用 O(1) 空间，方法是维护一个哑节点 `dummy`，利用上层已连接的 `next` 指针来串联下一层。面试中提一下即可。

## ⚠ 易错点

- [ ] **`level_size` 必须提前固定** — 循环内 `len(q)` 会因为子节点入队而变化，必须在 for 循环之前取值
- [ ] **`i < level_size - 1` 判断是否为最后** — 最后一个节点的 `next` 已经是 `None`（初始化时），不需要额外处理
- [ ] **节点定义有 `next` 字段** — 不是标准 `TreeNode`，是专门的 `Node` 类，注意构造函数
- [ ] **子节点入队顺序** — 先左后右，保证同一层从左到右排列

这道题用 BFS 层序遍历解决。标准做法是用队列逐层遍历。

每一层开始前，记录当前层的大小 level_size = len(q)。
对层内每个节点，只要它不是本层最后一个，
