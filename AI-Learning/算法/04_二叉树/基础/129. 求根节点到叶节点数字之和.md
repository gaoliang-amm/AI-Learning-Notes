# 129. 求根节点到叶节点数字之和

## 🎯 人话翻译

> 从根到叶子，每一段路径上的节点值排成一个数字（比如 1→2→3 就是 123）。求所有根到叶子路径组成的数字的总和。

## 💡 核心思想

> **DFS 自顶向下，用一个变量 `cur_sum` 记录从根到当前节点的数字。到叶子返回这个数字；非叶子则向左向右继续拼。**
>
> 核心公式：`cur_sum = cur_sum * 10 + node.val`，相当于每次往下走一层，把之前的数字左移一位（乘 10）再拼上新数字。
>
> 递归函数返回的是"以当前节点为根的子树中，所有叶子路径数字的累加和"。
> - 叶子节点：直接返回 `cur_sum`（已经拼好的完整数字）
> - 非叶子：返回左子树的累加和 + 右子树的累加和

用到：[[二叉树]]、[[递归]]、[[DFS]]

## 📝 解题步骤

1. **定义递归函数** — `dfs(node, cur_sum)`：返回以 `node` 为根的子树中所有叶路径数字的累加和
2. **空节点处理** — 空节点返回 0（不贡献任何数字）
3. **拼接当前数字** — `cur_sum = cur_sum * 10 + node.val`
4. **叶子节点** — 如果左右都为空，返回 `cur_sum`（这条路径拼好了）
5. **非叶子** — 返回 `dfs(node.left, cur_sum) + dfs(node.right, cur_sum)`

## 🔢 手动走一遍

以下面的树为例：

```
        1
       / \
      2   3
     / \
    4   5
```

DFS 过程：

| 节点 | 进入时 cur_sum | 计算 cur_sum*10+val | 叶子？ | 动作                                | 返回值 |
|------|---------------|---------------------|--------|-------------------------------------|--------|
| 1    | 0             | 0*10+1 = 1          | 否     | dfs(左,1) + dfs(右,1)               | 124+125+13=262|
| 2    | 1             | 1*10+2 = 12         | 否     | dfs(4,12) + dfs(5,12)               | 124+125=249   |
| 4    | 12            | 12*10+4 = 124       | 是     | 返回 124                            | 124           |
| 5    | 12            | 12*10+5 = 125       | 是     | 返回 125                            | 125           |
| 3    | 1             | 1*10+3 = 13         | 是     | 返回 13                             | 13            |

最终总和 = 124 + 125 + 13 = 262。

三条路径对应的数字：
- 1 → 2 → 4 = 124
- 1 → 2 → 5 = 125
- 1 → 3 = 13

## 💻 解法

```python
from typing import Optional


class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class Solution:
    def sumNumbers(self, root: Optional[TreeNode]) -> int:
        # 时间复杂度 O(n)，空间复杂度 O(h)

        def dfs(node: Optional[TreeNode], cur_sum: int) -> int:
            # 空节点，不贡献任何数字
            if not node:
                return 0

            # 当前路径数字 = 父路径 * 10 + 当前节点值
            cur_sum = cur_sum * 10 + node.val

            # 叶子节点：返回当前路径数字
            if not node.left and not node.right:
                return cur_sum

            # 非叶子节点：递归左右子树，累加结果
            return dfs(node.left, cur_sum) + dfs(node.right, cur_sum)

        return dfs(root, 0)
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明                                      |
|------|--------|-------------------------------------------|
| 时间 | O(n)   | 每个节点访问一次                           |
| 空间 | O(h)   | 递归栈深度，最坏 O(n)，平均 O(log n)       |

## ⚠ 易错点

- [ ] **空节点返回 0 不是返回 `cur_sum`** — 空节点不构成一条路径，不能把现有的 `cur_sum` 加进去
- [ ] **叶子节点返回 `cur_sum` 后不要再累加** — 叶子的 `cur_sum` 已经包含了它自己的值，直接返回即可
- [ ] **`cur_sum` 是传值不是传引用** — Python int 不可变，每次递归创建新的值，不需要回溯
- [ ] **DFS 内部的 `cur_sum` 是局部变量** — 不需要像全局变量那样重置
