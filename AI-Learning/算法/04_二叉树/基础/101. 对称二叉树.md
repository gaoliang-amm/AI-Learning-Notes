# 101. 对称二叉树

## 🎯 人话翻译

> 给你一棵二叉树，判断它是不是轴对称的（从根节点折过来，左子树和右子树完全镜像对称）。

## 💡 核心思想

> **根本身不需要比较，把左右子树当成两棵独立的树，用镜像方式同步比较。**
>
> 镜像比较的规则：对于两个节点 t1 和 t2：
> - t1 的左子树 应该镜像等于 t2 的右子树（"外侧"）
> - t1 的右子树 应该镜像等于 t2 的左子树（"内侧"）
>
> 这比 [[100. 相同的树]] 多了一步"交叉比较"，因为对称要求左右颠倒。

用到：[[二叉树]]、[[递归]]、[[DFS]]

## 📝 解题步骤

1. **根节点特判** — 空树直接算对称，返回 `True`
2. **启动镜像比较** — 把 `root.left` 和 `root.right` 传入辅助函数 `_isMirror`
3. **空节点处理** — 两个都为空说明对称到底了，返回 `True`；一个空一个不空则不对称
4. **值比较 + 交叉递归** — `t1.val == t2.val` 且 `_isMirror(t1.left, t2.right)` 且 `_isMirror(t1.right, t2.left)`
5. **返回结果** — 三个条件取 AND

## 🔢 手动走一遍

以下面这棵树为例：

```
        1
       / \
      2   2
     / \ / \
    3  4 4  3
```

镜像比较过程（`_isMirror(左2, 右2)`）：

| 步骤 | t1 (左) | t2 (右) | 都非空？ | 值相等？ | 动作                                               |
|------|---------|---------|----------|----------|----------------------------------------------------|
| 1    | 2       | 2       | 是       | 2==2     | 交叉递归：外侧 t1.left(3) vs t2.right(3)；内侧 t1.right(4) vs t2.left(4) |
| 2    | 3       | 3       | 是       | 3==3     | 左右都为空，返回 True                               |
| 3    | 4       | 4       | 是       | 4==4     | 左右都为空，返回 True                               |
| 结果 |         |         |          |          | 三个条件全部 True，对称                             |

假如换成一个不对称的树：

```
        1
       / \
      2   2
       \   \
        3   3
```

| 步骤 | t1 (左) | t2 (右) | 动作                                                   |
|------|---------|---------|--------------------------------------------------------|
| 1    | 2       | 2       | 交叉递归：外侧 t1.left(None) vs t2.right(3)            |
| 2    | None    | 3       | 一个空一个不空 → False，不对称                         |

## 💻 解法

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class Solution:
    def isSymmetric(self, root: TreeNode) -> bool:
        """
        判断二叉树是否轴对称
        思路：根不用比，直接比较左右子树是否镜像
        时间复杂度：O(n), 空间复杂度：O(h)
        """
        if not root:
            return True
        # 比较左子树和右子树是否镜像
        return self._isMirror(root.left, root.right)

    def _isMirror(self, t1: TreeNode, t2: TreeNode) -> bool:
        # 都空：对称
        if not t1 and not t2:
            return True
        # 一个空一个不空：不对称
        if not t1 or not t2:
            return False
        # 值相等 + 外侧对称(t1左 vs t2右) + 内侧对称(t1右 vs t2左)
        return (t1.val == t2.val and
                self._isMirror(t1.left, t2.right) and
                self._isMirror(t1.right, t2.left))
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明                                      |
|------|--------|-------------------------------------------|
| 时间 | O(n)   | 每个节点访问一次                           |
| 空间 | O(h)   | 递归栈深度，最坏 O(n)（链表），最好 O(log n) |

## ⚠ 易错点

- [ ] **交叉比较方向不能写反** — 是 `t1.left vs t2.right`（外侧）和 `t1.right vs t2.left`（内侧），写反了就变成了"相同的树"的判断
- [ ] **根节点不需要比较值** — 直接传 `root.left` 和 `root.right` 进入辅助函数即可，因为对称轴就是根节点
- [ ] **空节点要同时判** — `not t1 and not t2` 返回 True，`not t1 or not t2` 返回 False，顺序不可颠倒
- [ ] **切忌把"镜像"和"相同"搞混** — 镜像要求左右交叉，相同是左右对齐

## 🎤 面试怎么说

```text
这道题判断二叉树是否轴对称，本质是把左右子树按镜像方式做同步比较。

方法：定义一个辅助函数 isMirror(t1, t2)，判断两棵树是否镜像。
镜像的条件是 t1 的左子树等于 t2 的右子树（外侧），t1 的右子树等于 t2 的左子树（内侧）。

递归终止条件：都空→对称，一空一非空→不对称，值不同→不对称。

时间 O(n) 遍历所有节点，空间 O(h) 递归栈。
```
