# 236. 二叉树的最近公共祖先

## 🎯 人话翻译

> 在一棵普通的二叉树中找到两个节点 p 和 q 的最近公共祖先（LCA）。就是你往上走，第一个同时是 p 和 q 祖先的节点。

## 💡 核心思想

> **自底向上的递归（后序遍历）。每个节点问它的左右子树：你们那里有 p 或 q 吗？**
>
> 四种情况分析（当前节点为 `root`）：
> 1. 左子树和右子树都返回了非空结果 → p 和 q 分居两侧，`root` 就是最近公共祖先
> 2. 只有左子树返回非空 → p 和 q 都在左边，左子树的返回值就是 LCA
> 3. 只有右子树返回非空 → p 和 q 都在右边，右子树的返回值就是 LCA
> 4. 左右都返回空 → 这棵子树里没有 p 也没有 q
>
> 递归终止条件：空节点返回 `None`；遇到 p 或 q 直接返回该节点（因为它可能是自身的 LCA，或者作为找到的标志往上传递）。

用到：[[二叉树]]、[[递归]]、[[DFS]]、[[分治]]

## 📝 解题步骤

1. **递归终止条件** — 如果 `root` 为空，或 `root` 是 p，或 `root` 是 q，直接返回 `root`
2. **递归搜索左右子树** — `left = lca(root.left, p, q)`，`right = lca(root.right, p, q)`
3. **情况 1**：`left` 和 `right` 都不空 → p 和 q 分居两侧，返回 `root`
4. **情况 2**：只有 `left` 不空 → 返回 `left`
5. **情况 3**：只有 `right` 不空 → 返回 `right`
6. **情况 4**：都不空 → 返回 `None`

## 🔢 手动走一遍

以下面的树为例，找 p=5, q=1 的 LCA：

```
          3
        /   \
       5      1
      / \    / \
     6   2  0   8
        / \
       7   4
```

后序遍历过程：

| 节点 | 左子树返回 | 右子树返回 | 判断                              | 返回值 |
|------|-----------|-----------|-----------------------------------|--------|
| 6    | —         | —         | 空节点，返回 None                  | None   |
| 7    | —         | —         | 空节点，返回 None                  | None   |
| 4    | —         | —         | 空节点，返回 None                  | None   |
| 2    | None      | None      | 2 != p,q → 递归左右。7 和 4 都返 None | None   |
| 5    | None(6)   | None(2)   | 5 == p！直接返回 5                  | 5      |
| 0    | —         | —         | 0 != p,q → 左右 None，返回 None     | None   |
| 8    | —         | —         | 8 != p,q → 左右 None，返回 None     | None   |
| 1    | None(0)   | None(8)   | 1 == q！直接返回 1                  | 1      |
| 3    | 5         | 1         | 左右都非空！p 和 q 在两侧！          | **3**  |

最近公共祖先 = 3。

## 💻 解法

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class Solution:
    def lowestCommonAncestor(self, root: TreeNode, p: TreeNode, q: TreeNode) -> TreeNode:
        # 时间复杂度 O(n)，空间复杂度 O(h)

        # 1. 终止条件：空节点返回 None，找到 p 或 q 返回该节点
        if not root or root == p or root == q:
            return root

        # 2. 递归查找左子树和右子树
        left = self.lowestCommonAncestor(root.left, p, q)
        right = self.lowestCommonAncestor(root.right, p, q)

        # 3. 根据左右结果判断
        # 情况1：p 和 q 分别在左右子树，当前节点就是 LCA
        if left and right:
            return root

        # 情况2：都在左子树，返回左结果
        if left:
            return left

        # 情况3：都在右子树，返回右结果
        if right:
            return right

        # 情况4：找不到，返回 None
        return None
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明                                      |
|------|--------|-------------------------------------------|
| 时间 | O(n)   | 最坏情况遍历所有节点一次                    |
| 空间 | O(h)   | 递归栈深度，最坏 O(n)（退化成链表）          |

## ⚠ 易错点

- [ ] **顺序不能错：先判断 `if left and right`** — 必须先判断"左右都非空"的情况，因为它意味着当前节点就是 LCA，优先级最高
- [ ] **遇到 p 或 q 直接返回** — 不需要等到左右子树都搜完，遇到 p 或 q 就可以返回，因为"自己可以是自己的祖先"
- [ ] **返回值含义理解清楚** — 递归返回的不是 bool，而是节点本身。非空节点表示"找到了 p/q/LCA"
- [ ] **不要和 BST 的 LCA 混淆** — BST 可以通过值的大小关系判断走向（O(log n)），但普通二叉树必须遍历整棵树（O(n)）
- [ ] **p 可能不在树中** — LeetCode 保证 p 和 q 一定在树中，所以不需要额外判断
