# 100. 相同的树

## 🎯 人话翻译

> 给你两棵二叉树的根节点 `p` 和 `q`，判断这两棵树是否完全相同（结构一样，对应位置的值也一样）。

## 💡 核心思想

> **同步递归遍历两棵树，在每个对应位置逐一比较。**
>
> 对于每一对节点，需要满足三个条件才算相同：
> 1. 两个节点都为空（同时到底，结构一致）
> 2. 两个节点都不为空且值相等
> 3. 它们的左子树相同 AND 右子树相同
>
> 一旦某个位置出现"一个空一个不空"或"值不同"，直接返回 False。

用到：[[二叉树]]、[[递归]]、[[DFS]]

## 📝 解题步骤

1. **空节点判断** — 如果 `p` 和 `q` 都为空，说明同步到达了叶子下面，返回 `True`
2. **结构差异判断** — 如果一个为空另一个不为空，说明树的结构不同，返回 `False`
3. **值判断** — 如果当前两个节点的值不同，返回 `False`
4. **递归比较子树** — 同时递归比较 `p.left vs q.left` 和 `p.right vs q.right`，两个结果取 AND

## 🔢 手动走一遍

以下面两棵树为例：

```
树 p:         树 q:
    1             1
   / \           / \
  2   3         2   3
```

比较过程：

| 步骤 | p 节点 | q 节点 | 都非空？ | 值相等？ | 动作                       |
|------|--------|--------|----------|----------|----------------------------|
| 1    | 1      | 1      | 是       | 1==1     | 递归比较左右子树            |
| 2    | 2      | 2      | 是       | 2==2     | 左右均为空，返回 True       |
| 3    | 3      | 3      | 是       | 3==3     | 左右均为空，返回 True       |
| 结果 |        |        |          |          | 左 True AND 右 True = True  |

假如树 p 的右子节点是 `3`，树 q 的右子节点是 `4`：

| 步骤 | p 节点 | q 节点 | 都非空？ | 值相等？ | 动作            |
|------|--------|--------|----------|----------|-----------------|
| ...  | 3      | 4      | 是       | 3!=4     | 直接返回 False  |

## 💻 解法

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class Solution:
    def isSameTree(self, p: TreeNode, q: TreeNode) -> bool:
        """
        判断两棵二叉树是否完全相同（结构相同、节点值相同）
        思路：递归同步遍历两棵树，逐一比较节点
        时间复杂度：O(n)，空间复杂度：O(h)
        """
        # 两个都为空 → 相同
        if not p and not q:
            return True
        # 一个空一个不空 → 不相同
        if not p or not q:
            return False
        # 值不同 → 不相同
        if p.val != q.val:
            return False

        # 递归比较左孩子和右孩子
        return self.isSameTree(p.left, q.left) and self.isSameTree(p.right, q.right)
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明                                      |
|------|--------|-------------------------------------------|
| 时间 | O(n)   | 每个节点访问一次，n 为两棵树中较小的节点数  |
| 空间 | O(h)   | 递归栈深度，最坏 O(n)（链表），平均 O(log n) |

## ⚠ 易错点

- [ ] **空节点的判断顺序不能乱** — 必须先判断 `not p and not q` 再判断 `not p or not q`，否则会误判
- [ ] **不要忘记比较值** — 只比较结构不够，还要比较 `p.val == q.val`
- [ ] **递归结果用 AND 不是 OR** — 左子树和右子树都必须相同，是 `and` 不是 `or`
- [ ] **函数签名用 `Optional[TreeNode]`** — 在 LeetCode 中需要 `from typing import Optional`

## 🎤 面试怎么说

```text
这道题考察二叉树的递归遍历，本质是同步 DFS。

核心思路是同时递归遍历两棵树，在每个对应位置逐一比较：
先判断结构（都为空 or 一个空一个不空），再判断值，最后递归左右子树。

三个条件缺一不可：都为空返回 True，一空一非空返回 False，
值不同返回 False，否则继续递归且左右都用 AND 连接。

时间 O(n) 每个节点走一次，空间 O(h) 递归栈深度。
```
