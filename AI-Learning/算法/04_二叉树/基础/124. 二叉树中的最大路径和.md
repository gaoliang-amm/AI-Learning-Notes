# 124. 二叉树中的最大路径和

## 🎯 人话翻译

> 在二叉树中找一条路径（可以是任意起点、任意终点，不一定要经过根），让路径上所有节点值的和最大。返回这个最大值。

## 💡 核心思想

> **自底向上 DFS，每个节点返回"单边最大贡献"给父节点，同时在当前节点合并左右贡献来更新全局最大值。**
>
> 关键概念区分：
> - **单边贡献**：从当前节点出发，向下只能走左或右一条路，给父节点用的值。公式：`node.val + max(left_gain, right_gain, 0)`
> - **路径和**：当前节点作为路径的"拐点"，左右都走，形成一条经过当前节点的完整路径。公式：`left_gain + node.val + right_gain`
> - **负值截断**：如果子树的贡献是负数，就不如不走（`max(gain, 0)`），相当于从当前节点重新开始
>
> 用一个全局变量 `self.max_sum` 不断用"路径和"去更新最大值。

用到：[[二叉树]]、[[递归]]、[[DFS]]、[[分治]]

## 📝 解题步骤

1. **初始化全局最大** — `self.max_sum = float('-inf')`，因为节点值可能是负数
2. **定义递归函数 `dfs(node)`** — 返回以 `node` 为起点的**单边最大贡献**
3. **递归终止** — 空节点返回 0
4. **计算左右子树的单边贡献** — 调用 `dfs(node.left)` 和 `dfs(node.right)`，与 0 取 max（负数就舍弃）
5. **更新全局最大** — `self.max_sum = max(self.max_sum, left_gain + node.val + right_gain)`
6. **返回单边贡献** — `node.val + max(left_gain, right_gain)`

## 🔢 手动走一遍

以下面的树为例：

```
        -10
        /  \
       9   20
           / \
          15  7
```

DFS 自底向上：

| 节点 | 左贡献  | 右贡献  | left_gain (>=0) | right_gain (>=0) | 经过当前节点的路径和 | 更新全局 max | 单边返回给父节点 |
|------|--------|--------|-----------------|------------------|---------------------|-------------|-----------------|
| 9    | 0 (空) | 0 (空) | 0               | 0                | 0+9+0 = 9           | 9           | 9+max(0,0)=9    |
| 15   | 0 (空) | 0 (空) | 0               | 0                | 0+15+0 = 15         | 15          | 15+max(0,0)=15  |
| 7    | 0 (空) | 0 (空) | 0               | 0                | 0+7+0 = 7           | 15          | 7+max(0,0)=7    |
| 20   | 15     | 7      | 15              | 7                | 15+20+7 = 42        | 42          | 20+max(15,7)=35 |
| -10  | 9      | 35     | 9               | 35               | 9+(-10)+35 = 34     | 42          | -10+max(9,35)=25|

最终 `self.max_sum = 42`，对应路径 15 → 20 → 7。

## 💻 解法

```python
from typing import Optional


class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class Solution:
    def maxPathSum(self, root: Optional[TreeNode]) -> int:
        # 时间复杂度 O(n)，空间复杂度 O(h)

        # 全局最大值，初始化为最小值（因为节点值可能为负数）
        self.max_sum = float('-inf')

        def dfs(node: Optional[TreeNode]) -> int:
            # 空节点贡献为 0
            if not node:
                return 0

            # 递归计算左右子树的最大单边贡献
            # 如果子树贡献为负数，就选择不经过它（取 0）
            left_gain = max(dfs(node.left), 0)
            right_gain = max(dfs(node.right), 0)

            # 经过当前节点的最大路径和 = 左贡献 + 当前值 + 右贡献
            current_path_sum = left_gain + node.val + right_gain

            # 更新全局最大值
            self.max_sum = max(self.max_sum, current_path_sum)

            # 返回给父节点的单边最大贡献 = 当前值 + max(左贡献, 右贡献)
            return node.val + max(left_gain, right_gain)

        dfs(root)
        return self.max_sum
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明                                      |
|------|--------|-------------------------------------------|
| 时间 | O(n)   | 每个节点访问一次，后序遍历                  |
| 空间 | O(h)   | 递归栈深度，最坏 O(n)（退化成链表）          |

## ⚠ 易错点

- [ ] **负值节点的处理** — `max(dfs(child), 0)` 很关键，如果子树贡献是负的，直接不走它（设为 0）
- [ ] **全局最大要用 `float('-inf')`** — 不能用 0，因为节点值可能全为负数（如 [-3]），最大路径和应该是 -3
- [ ] **`self.max_sum` 必须用 `self.`** — 在嵌套函数中修改外层的 int 变量需要使用 `self.` 或 `nonlocal`
- [ ] **单边返回和路径和是两个概念** — 返回给父节点的只能选左或右一条路（`max(left, right)`），但在当前节点合并时左右都要（`left + right`）
- [ ] **后序遍历顺序** — 必须先拿到左右子树的结果，再计算当前节点，不能颠倒

## 🎤 面试怎么说

```text
这道题是二叉树中的 Hard 题，但它本质是自底向上的 DFS。

核心思想：把问题拆成两个层面——
1. 返回值（给父节点用）：从当前节点出发向下的最大单边贡献，
   只能走左或右一边，负值直接弃用。
2. 全局更新（当前节点为拐点）：左贡献 + 当前值 + 右贡献，
   这是一条完整的路径，用它更新全局最大值。

用后序遍历自底向上，每个节点在向上返回之前，
先用"左右合并"的路径和去更新全局最大。

时间 O(n) 遍历所有节点，空间 O(h) 递归栈。
```
