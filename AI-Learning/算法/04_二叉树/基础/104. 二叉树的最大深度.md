# 104. 二叉树的最大深度

## 🎯 人话翻译

> 给定一棵二叉树，求它从根节点到最远叶子节点的层数（最大深度）。

## 💡 核心思想

> **自底向上递归：当前节点的深度 = 1 + max(左子树深度, 右子树深度)。**
>
> 把问题拆解：想知道整棵树多深，先问左子树多深、右子树多深，取大的那个再加 1（当前节点自己的一层）。
>
> 空节点深度为 0（没有节点，不算一层）。

用到：[[二叉树]]、[[递归]]、[[DFS]]

## 📝 解题步骤

1. **递归终止条件** — 空节点返回 0，表示不贡献任何深度
2. **递归计算左子树深度** — `left_depth = self.maxDepth(root.left)`
3. **递归计算右子树深度** — `right_depth = self.maxDepth(root.right)`
4. **返回** — `1 + max(left_depth, right_depth)`

## 🔢 手动走一遍

以下面这棵树为例：

```
        3
       / \
      9  20
         / \
        15  7
```

自底向上计算：

| 节点 | 左子树深度 | 右子树深度 | 计算过程       | 返回深度 |
|------|-----------|-----------|----------------|----------|
| 9    | 0 (空)    | 0 (空)    | 1 + max(0,0)   | 1        |
| 15   | 0 (空)    | 0 (空)    | 1 + max(0,0)   | 1        |
| 7    | 0 (空)    | 0 (空)    | 1 + max(0,0)   | 1        |
| 20   | 1 (15)    | 1 (7)     | 1 + max(1,1)   | 2        |
| 3    | 1 (9)     | 2 (20)    | 1 + max(1,2)   | 3        |

最终返回 3。

## 💻 解法

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


class Solution:
    def maxDepth(self, root: TreeNode) -> int:
        """
        求二叉树的最大深度
        思路：递归——当前深度 = 1 + max(左子树深度, 右子树深度)
        时间复杂度：O(n)，空间复杂度：O(h) h为树高
        """
        # 空节点深度为 0
        if not root:
            return 0

        # 递归计算左右子树深度
        left_depth = self.maxDepth(root.left)
        right_depth = self.maxDepth(root.right)

        # 取大的那个 + 1（当前节点）
        return 1 + max(left_depth, right_depth)
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明                                          |
|------|--------|-----------------------------------------------|
| 时间 | O(n)   | 每个节点访问一次                               |
| 空间 | O(h)   | 递归栈深度，最坏 O(n)（退化成链表），平均 O(log n) |

## ⚠ 易错点

- [ ] **空节点返回 0 不是 -1** — 如果返回 -1 会导致最终深度少一层
- [ ] **叶子节点的深度是 1 不是 0** — 叶子节点左右都空，`1 + max(0,0) = 1`
- [ ] **不要和"平衡二叉树判断"混淆** — 求深度只要 max，判断平衡需要比较左右深度差
- [ ] **不要忘记加 1** — 递归返回的是子树深度，当前节点自身贡献 1 层深度

## 🎤 面试怎么说

```text
这道题是二叉树 DFS 最基础的题目，自底向上递归。

核心公式：当前节点深度 = 1 + max(左子树深度, 右子树深度)。

递归终止条件是空节点返回 0。从叶子开始往上算，每个节点取左右子树中较深的加 1。

时间 O(n) 遍历所有节点，空间 O(h) 递归栈深度。

如果面试官问 BFS 怎么做：用层序遍历，每遍历完一层计数器加 1。
```
