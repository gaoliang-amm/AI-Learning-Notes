# 205. 同构字符串

## 🎯 人话翻译

> 两个字符串 s 和 t，问能不能通过一种"一对一替换字典"把 s 变成 t。比如 "egg" 和 "add"：e->a, g->d，可以。但 "foo" 和 "bar"：f->b, o->a, 第二个 o 又要变成 r，一个字符映射了两个不同的字符，不行。

## 💡 核心思想

> [[哈希表]]建立**双向映射**。用两个字典 `s2t` 和 `t2s` 同时记录映射关系。遍历时检查：
> - c1 有没有已经映射到别的字符？
> - c2 有没有已经被别的字符映射了？
>
> 双向检查确保了一对一的映射要求。

用到：[[哈希表]]

## 📝 解题步骤

1. **长度检查** — 若 `len(s) != len(t)`，直接返回 `False`
2. **初始化双向映射** — `s2t = {}`（s 字符 -> t 字符），`t2s = {}`（t 字符 -> s 字符）
3. **遍历并检查冲突** — 对于每个 `(c1, c2)`：
   - 若 `c1 in s2t and s2t[c1] != c2`：c1 已映射到别的字符，冲突
   - 若 `c2 in t2s and t2s[c2] != c1`：c2 已被别的字符映射，冲突
4. **建立映射** — `s2t[c1] = c2`, `t2s[c2] = c1`
5. **全部通过** — 返回 `True`

## 🔢 手动走一遍

输入: `s = "egg"`, `t = "add"`

| 索引 | c1 | c2 | s2t | t2s | c1已在s2t? | c2已在t2s? | 冲突? | 操作 |
|------|----|----|------|------|-----------|-----------|-------|------|
| 0 | 'e' | 'a' | {} | {} | 否 | 否 | 否 | s2t['e']='a', t2s['a']='e' |
| 1 | 'g' | 'd' | {e:a} | {a:e} | 否 | 否 | 否 | s2t['g']='d', t2s['d']='g' |
| 2 | 'g' | 'd' | {e:a,g:d} | {a:e,d:g} | 是(s2t['g']='d'==c2) | 是(t2s['d']='g'==c1) | 否(都一致) | 无冲突 |

全部通过，返回 `True`。

反例: `s = "foo"`, `t = "bar"`
| 索引 | c1 | c2 | 冲突? |
|------|----|----|-------|
| 0 | 'f' | 'b' | 否 |
| 1 | 'o' | 'a' | 否 |
| 2 | 'o' | 'r' | s2t['o']='a' != 'r'，冲突！返回 False |

## 💻 解法

> 双向哈希表：s->t 和 t->s 分别映射，确保一对一关系不被破坏。

```python
class Solution:
    def isIsomorphic(self, s: str, t: str) -> bool:
        # 长度不同直接返回False
        if len(s) != len(t):
            return False

        # 建立双向映射：s->t 和 t->s
        s2t = {}
        t2s = {}

        for c1, c2 in zip(s, t):
            # 情况1：c1已经映射过，但不是当前c2，冲突
            if c1 in s2t and s2t[c1] != c2:
                return False
            # 情况2：c2已经被映射过，但不是当前c1，冲突
            if c2 in t2s and t2s[c2] != c1:
                return False
            # 建立双向映射
            s2t[c1] = c2
            t2s[c2] = c1

        # 所有字符都满足双向唯一映射
        return True
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(n) | 每个字符访问一次 |
| 空间 | O(C) | C 为不同字符的个数，最多 128（ASCII） |

## ⚠ 易错点

- [ ] 必须**双向映射**：只用单向映射会漏掉"两个 s 字符映射到同一个 t 字符"的情况（如 "ab" -> "cc"）
- [ ] 检查顺序：先检查 `c1 in s2t and s2t[c1] != c2`，再检查 `c2 in t2s and t2s[c2] != c1`，两者都要检查
- [ ] 映射建立要放在检查之后：如果检查不通过就 `return False`，通过了才建映射
- [ ] 长度不同直接返回 `False`，是个前置剪枝

```text
这道题要求判断两个字符串是否同构，核心是建立双向映射。

用两个字典：s2t 记录 s 到 t 的映射，t2s 记录 t 到 s 的映射。
遍历时同时检查两个方向：如果 c1 已经映射到了别的 c2，

两个字典的检查缺一不可，

时间 O(n)，空间 O(字符种类数)。
```
