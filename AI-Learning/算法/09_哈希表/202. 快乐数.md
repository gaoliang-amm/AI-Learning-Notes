# 202. 快乐数

## 🎯 人话翻译

> 判断一个数是不是"快乐数"。规则：把一个数的每一位数字平方后相加，得到一个新数，不断重复这个过程。如果最终变成 1，就是快乐数；如果陷入无限循环（永远到不了 1），就不是快乐数。比如 19 -> 1^2+9^2=82 -> 68 -> 100 -> 1，是快乐数。

## 💡 核心思想

> 本质是**检测循环**的问题。如果计算过程中出现了重复的数字，说明进入了循环，永远不会到 1。可以用[[哈希表]] `set` 记录出现过的数字；也可以用快慢指针（Floyd 判圈法）将空间优化到 O(1)。

用到：[[哈希表]]、[[快慢指针]]

## 📝 解题步骤

1. **定义辅助函数** — `get_next(n)`：计算 n 的每位数字平方和
2. **初始化 set** — `seen = set()` 记录出现过的数字
3. **循环判断** — `while n != 1`：
   - 如果 `n in seen`，说明进入循环，返回 `False`
   - 否则 `seen.add(n)`，然后 `n = get_next(n)`
4. **到达 1** — 返回 `True`

（本解法用快慢指针替代 set，空间 O(1)）

## 🔢 手动走一遍

输入: `n = 19`

| 步骤 | n | 数字计算 | 结果 | 判断 |
|------|---|----------|------|------|
| 初始 | 19 | - | - | - |
| 1 | 19 | 1^2 + 9^2 = 1 + 81 | 82 | n != 1 |
| 2 | 82 | 8^2 + 2^2 = 64 + 4 | 68 | n != 1 |
| 3 | 68 | 6^2 + 8^2 = 36 + 64 | 100 | n != 1 |
| 4 | 100 | 1^2 + 0^2 + 0^2 = 1 | 1 | n == 1，返回 True |

输入: `n = 2`（非快乐数）

| 步骤 | n | 结果 | 是否重复? |
|------|---|------|-----------|
| 1 | 2 | 4 | 否 |
| 2 | 4 | 16 | 否 |
| 3 | 16 | 37 | 否 |
| 4 | 37 | 58 | 否 |
| 5 | 58 | 89 | 否 |
| 6 | 89 | 145 | 否 |
| 7 | 145 | 42 | 否 |
| 8 | 42 | 20 | 否 |
| 9 | 20 | 4 | 是！4 已出现过，循环，返回 False |

## 💻 解法

> 快慢指针（Floyd 判圈法）：慢指针走一步，快指针走两步，相遇则说明有环（不是快乐数）；快指针到 1 则是快乐数。

```python
class Solution:
    def isHappy(self, n: int) -> bool:
        """
        快慢指针法判断快乐数
        时间复杂度：O(log n)
        空间复杂度：O(1)
        """

        def get_next(num: int) -> int:
            """计算一个数的各位数字平方和"""
            total = 0
            while num > 0:
                digit = num % 10
                total += digit * digit
                num //= 10
            return total

        # 快慢指针初始化
        slow = n
        fast = get_next(n)

        # 终止条件：快指针到1 或 快慢指针相遇
        while fast != 1 and slow != fast:
            slow = get_next(slow)  # 慢指针走1步
            fast = get_next(get_next(fast))  # 快指针走2步

        return fast == 1
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(log n) | get_next 每次处理 log10(n) 位数；循环次数取决于快乐数序列长度，是常数级 |
| 空间 | O(1) | 快慢指针只用两个变量 |

## ⚠ 易错点

- [ ] `get_next` 函数要正确提取每一位：`num % 10` 得个位，`num //= 10` 去掉个位，顺序不能错
- [ ] 快慢指针初始化：`slow = n`（原地），`fast = get_next(n)`（走了一步），不要两个都从 n 开始
- [ ] `while` 条件 `fast != 1 and slow != fast`：如果 fast 到 1 了直接退出，说明是快乐数；如果 slow == fast 了退出，说明有环
- [ ] 返回值 `fast == 1`：fast 到 1 说明是快乐数，slow 和 fast 相遇说明不是快乐数（fast 不会和 slow 同时在 1 相遇，因为 1 的下一个数还是 1）

## 🎤 面试怎么说

```text
这道题的核心是检测循环。如果一个数不是快乐数，
它在平方和计算过程中一定会陷入循环。

可以用 set 记录出现过的数字，空间 O(该序列长度)。
但更优雅的做法是用快慢指针（Floyd 判圈法）：
慢指针每次走一步，快指针每次走两步。
如果快指针走到了 1，说明是快乐数；
如果快慢指针相遇（且不在 1），说明有循环。

时间 O(log n)（每次计算位数平方和），空间 O(1)。
```
