# 128. 最长连续序列

## 🎯 人话翻译

> 给一个未排序的整数数组，找出最长的连续数字序列的长度（不要求它们在原数组中也连续）。比如 [100, 4, 200, 1, 3, 2]，最长连续序列是 [1, 2, 3, 4]，长度是 4。要求 O(n) 时间。

## 💡 核心思想

> 先用 `set()` 去重并实现 O(1) 查找。然后关键优化是：**只从连续序列的起点开始计数**。如何判断是不是起点？如果 `num - 1` 不在 set 中，说明 `num` 是一个序列的起点。对于每个起点，用 while 循环往后数，统计这条序列有多长。每个元素最多被"往后数"访问一次，总体 O(n)。

用到：[[哈希表]]、[[Set]]

## 📝 解题步骤

1. **转集合** — `num_set = set(nums)`，去重并提供 O(1) 查找
2. **遍历集合** — 对于每个 `num`：
   - 如果 `num - 1 not in num_set`，说明 `num` 是一个序列的起点
   - 从 `num` 开始往后数：`while current_num + 1 in num_set`，`current_num += 1`, `current_length += 1`
3. **更新最大值** — `max_length = max(max_length, current_length)`
4. **返回最大值** — 返回 `max_length`

## 💻 解法

> Set + 只从序列起点开始计数：`num-1 not in set` 判断起点，while 往后扩展。

```python
from typing import List

class Solution:
    def longestConsecutive(self, nums: List[int]) -> int:
        """
        找出数组中最长连续序列的长度。
        时间复杂度：O(n)，空间复杂度：O(n)
        """
        # 1. 将数组转为集合，实现O(1)时间复杂度的查找，同时去重
        num_set = set(nums)
        # 记录最长连续序列的长度
        max_length = 0

        # 2. 遍历集合中的每个数字
        for num in num_set:
            # 3. 只有当当前数字是序列的起点时（前一个数字不在集合中），才开始统计
            if num - 1 not in num_set:
                current_num = num
                current_length = 1

                # 4. 向后查找连续的数字，统计当前序列的长度
                while current_num + 1 in num_set:
                    current_num += 1
                    current_length += 1

                # 5. 更新最长序列长度
                max_length = max(max_length, current_length)

        return max_length
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(n) | 每个元素最多被"从起点往后数"访问一次，while 循环不计重复 |
| 空间 | O(n) | set 存储所有元素 |

## ⚠ 易错点

- [ ] 必须用 `set` 而不是 `list` 来查找，因为 `in` 操作在 list 中是 O(n)，在 set 中是 O(1)
- [ ] 必须遍历 `num_set`（集合中的元素），而不是原数组 `nums`。遍历 set 可以对重复元素只处理一次
- [ ] 起点判断 `num - 1 not in num_set` 是核心优化：如果没有这个判断，每个元素都尝试往后数，while 循环的累计代价会变成 O(n^2)
- [ ] while 循环内部不要重复更新 `max_length`，在 while 循环结束后更新一次即可

```text
这道题要求在 O(n) 时间内找最长连续序列。

核心思路：先把数组变成 set（去重 + O(1) 查找）。

然后遍历 set 中的每个元素，但有个关键的优化：

```
