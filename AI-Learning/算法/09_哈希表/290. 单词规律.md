# 290. 单词规律

## 🎯 人话翻译

> 给你一个模式串 pattern（比如 "abba"）和一个由空格分隔的字符串 s（比如 "dog cat cat dog"）。判断 s 中的单词是否和 pattern 中的字符按顺序一一对应。每个字符对应唯一一个单词，每个单词也只能对应唯一一个字符。

## 💡 核心思想

> 本质和 [[205. 同构字符串]] 完全一样，只是把"字符映射到字符"变成了"字符映射到单词"。用[[哈希表]]建立双向映射：`char2word` 和 `word2char`，遍历时检查两个方向的映射是否一致。

用到：[[哈希表]]

## 📝 解题步骤

1. **拆分字符串** — `words = s.split()`，将 s 按空格拆成单词列表
2. **长度检查** — 如果 `len(pattern) != len(words)`，直接返回 `False`
3. **初始化双向映射** — `char2word = {}`, `word2char = {}`
4. **遍历检查冲突** — 对于每对 `(ch, word)`：
   - 若 `ch in char2word and char2word[ch] != word`：字符已映射到别的单词，冲突
   - 若 `word in word2char and word2char[word] != ch`：单词已被别的字符绑定，冲突
5. **建立映射** — `char2word[ch] = word`, `word2char[word] = ch`
6. **全部匹配** — 返回 `True`

## 🔢 手动走一遍

输入: `pattern = "abba"`, `s = "dog cat cat dog"`

拆分: `words = ["dog", "cat", "cat", "dog"]`，长度都是 4。

| 索引 | ch | word | char2word | word2char | ch已映射? | word已映射? | 冲突? | 操作 |
|------|----|------|-----------|-----------|-----------|------------|-------|------|
| 0 | 'a' | "dog" | {} | {} | 否 | 否 | 否 | a->dog |
| 1 | 'b' | "cat" | {a:dog} | {dog:a} | 否 | 否 | 否 | b->cat |
| 2 | 'b' | "cat" | {a:dog,b:cat} | {dog:a,cat:b} | char2word['b']=='cat'==word | word2char['cat']=='b'==ch | 否 | 一致，通过 |
| 3 | 'a' | "dog" | {a:dog,b:cat} | {dog:a,cat:b} | char2word['a']=='dog'==word | word2char['dog']=='a'==ch | 否 | 一致，通过 |

全部通过，返回 `True`。

## 💻 解法

> 双向映射哈希表：字符到单词、单词到字符，检查两个方向的一致性。

```python
class Solution:
    def wordPattern(self, pattern: str, s: str) -> bool:
        # 把字符串按空格切分成单词列表
        words = s.split()
        # 字符个数和单词个数不一样，直接不匹配
        if len(pattern) != len(words):
            return False

        # 双向映射：字符 -> 单词，单词 -> 字符
        char2word = {}
        word2char = {}

        # 同时遍历 pattern 字符和单词
        for ch, word in zip(pattern, words):
            # 1. 字符已经绑定过，但绑定的不是当前单词 -> 冲突
            if ch in char2word and char2word[ch] != word:
                return False
            # 2. 单词已经绑定过，但绑定的不是当前字符 -> 冲突
            if word in word2char and word2char[word] != ch:
                return False
            # 建立双向绑定
            char2word[ch] = word
            word2char[word] = ch

        # 全部匹配成功
        return True
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(n) | n 为 pattern 长度，每个字符/单词访问一次 |
| 空间 | O(k) | k 为不同字符/单词的数量 |

## ⚠ 易错点

- [ ] 必须先 `s.split()` 再比较长度：split 后才能知道单词数量是否和 pattern 长度一致
- [ ] 双向映射不可省略：只用单向映射会漏判——比如 pattern="ab", s="dog dog"；a->dog, b->dog，单向映射无法发现两个字符映射到了同一个单词
- [ ] `zip(pattern, words)` 要求两者长度一致（前置检查保证了这一点）
- [ ] 检查冲突的逻辑：先检查 `char2word[ch] != word`，再检查 `word2char[word] != ch`，然后才建立映射
