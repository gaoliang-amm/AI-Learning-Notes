# 155. 最小栈

## 🎯 人话翻译

> 设计一个普通的栈，额外支持一个 `getMin()` 方法，能在 O(1) 时间内返回栈中当前的最小元素。

## 💡 核心思想

> **用两个栈同步操作：一个存数据，一个存"当前位置的最小值"。**
>
> 每次 `push(x)` 时，辅助栈压入 `min(x, 当前辅助栈栈顶)`，这样辅助栈的栈顶始终是当前主栈的最小元素。`pop` 时两个栈同步弹出。

用到：[[栈]]

## 📝 解题步骤

1. **初始化** — `stack` 存所有元素，`min_stack` 存每个位置对应的最小值
2. **push(x)** — 主栈压入 `x`；辅助栈若为空则压入 `x`，否则压入 `min(x, min_stack[-1])`
3. **pop()** — 主栈和辅助栈同时弹出栈顶
4. **top()** — 返回主栈栈顶 `stack[-1]`
5. **getMin()** — 返回辅助栈栈顶 `min_stack[-1]`

## 🔢 手动走一遍

操作序列: `push(-2) -> push(0) -> push(-3) -> getMin() -> pop() -> top() -> getMin()`

| 操作       | 主栈         | 辅助栈（最小值） | 当前最小 | 说明                      |
|------------|--------------|------------------|----------|---------------------------|
| push(-2)   | `[-2]`       | `[-2]`           | -2       | 第一个元素，直接入栈      |
| push(0)    | `[-2, 0]`    | `[-2, -2]`       | -2       | min(0, -2) = -2           |
| push(-3)   | `[-2, 0, -3]`| `[-2, -2, -3]`   | -3       | min(-3, -2) = -3          |
| getMin()   | 查询返回 -3  |                  |          | 辅助栈栈顶 = -3           |
| pop()      | `[-2, 0]`    | `[-2, -2]`       | -2       | 两个栈同步弹出            |
| top()      | 查询返回 0   |                  |          | 主栈栈顶 = 0              |
| getMin()   | 查询返回 -2  |                  |          | 辅助栈栈顶 = -2           |

## 💻 解法

> 双栈同步，辅助栈始终记录当前位置的最小值。

```python
class MinStack:
    def __init__(self):
        """
        初始化两个栈：
        - stack: 主栈，存储所有元素
        - min_stack: 辅助栈，存储每个位置对应的最小值
        """
        self.stack = []
        self.min_stack = []

    def push(self, val: int) -> None:
        """将元素压入栈。"""
        self.stack.append(val)
        # 辅助栈压入当前最小值
        if not self.min_stack:
            self.min_stack.append(val)
        else:
            self.min_stack.append(min(val, self.min_stack[-1]))

    def pop(self) -> None:
        """删除栈顶元素。"""
        self.stack.pop()
        self.min_stack.pop()

    def top(self) -> int:
        """获取栈顶元素。"""
        return self.stack[-1]

    def getMin(self) -> int:
        """获取栈中的最小元素。"""
        return self.min_stack[-1]
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(1) | 所有操作均为常数时间 |
| 空间 | O(n) | 两个栈各存 n 个元素，额外空间 O(n) |

## ⚠ 易错点

- [ ] `pop()` 时两个栈必须同步弹出，若只弹主栈会破坏辅助栈的对应关系
- [ ] `push()` 时每次都要更新辅助栈，不能只在值更小时才压入——否则 `pop()` 时无法知道该不该弹辅助栈
- [ ] `getMin()` 是取 `min_stack[-1]`，不是 `min(stack)`，后者是 O(n) 不满足题意
- [ ] 辅助栈为空（首次 push）时的处理：直接压入而非调用 `min()`
- [ ] 不要用单个变量存最小值——`pop()` 后最小值可能失效，必须用辅助栈同步维护
