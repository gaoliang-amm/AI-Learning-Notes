# 71. 简化路径

## 🎯 人话翻译

> 给定一个 Unix 绝对路径（可能有 `..`、`.`、连续斜杠），把它化简成最短的规范路径。

## 💡 核心思想

> **用栈模拟目录的进出。**
>
> 按 `/` 分割路径后，遇到普通目录名就入栈，遇到 `..` 就出栈（回到上级），遇到 `.` 或空字符串就忽略。最后栈中剩下的就是有效路径，用 `/` 拼回即可。

用到：[[栈]]

## 📝 解题步骤

1. **分割路径** — 用 `split('/')` 将路径按斜杠分割，得到每一段
2. **初始化栈** — 空栈用来存放有效目录名
3. **遍历每一段** — 遇到 `''` 或 `'.'` 跳过；遇到 `'..'` 若栈非空则弹出栈顶；其他情况（普通目录名）压入栈
4. **组装结果** — 返回 `'/' + '/'.join(stack)`

## 🔢 手动走一遍

输入: `path = "/a/./b/../c/"`

| 步骤 | 当前段   | 操作   | 栈内容    | 说明                          |
|------|----------|--------|-----------|-------------------------------|
| 1    | `""`     | 跳过   | `[]`      | `/a/` 开头斜杠产生空字符串    |
| 2    | `"a"`    | push   | `["a"]`   | 普通目录，入栈                |
| 3    | `"."`    | 跳过   | `["a"]`   | 当前目录，无变化              |
| 4    | `"b"`    | push   | `["a","b"]`| 普通目录，入栈                |
| 5    | `".."`   | pop    | `["a"]`   | 回上级，弹出 `"b"`            |
| 6    | `"c"`    | push   | `["a","c"]`| 普通目录，入栈                |
| 7    | `""`     | 跳过   | `["a","c"]`| 末尾斜杠产生空字符串          |

结果: `"/" + "a/c" = "/a/c"`

## 💻 解法

> 用栈维护当前有效路径，`..` 回退时弹出栈顶。

```python
class Solution:
    def simplifyPath(self, path: str) -> str:
        """
        简化 Unix 风格路径，返回规范路径。
        使用栈模拟目录切换。
        """
        # 按 '/' 分割路径
        parts = path.split('/')
        stack = []

        for part in parts:
            if part == '' or part == '.':
                # 空字符串（连续斜杠）或当前目录，忽略
                continue
            elif part == '..':
                # 上级目录，弹出栈顶（如果栈非空）
                if stack:
                    stack.pop()
            else:
                # 实际目录名，入栈
                stack.append(part)

        # 用 '/' 连接栈中目录，并在开头加上 '/'
        return '/' + '/'.join(stack)
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(n) | 遍历分割后的每一段，`split` 和 `join` 各 O(n) |
| 空间 | O(n) | 最坏情况路径全为有效目录，栈存储所有段 |

## ⚠ 易错点

- [ ] `split('/')` 会因连续斜杠产生空字符串，必须处理 `part == ''` 的情况（如 `"/a//b"` -> `["", "a", "", "b"]`）
- [ ] `..` 弹出前要检查栈是否非空：根目录的 `..` 仍然是根目录，不能弹空栈
- [ ] 当前目录 `.` 需要显式跳过（`".."` 和 `"."` 都是单点/双点，容易漏掉 `.` 的处理）
- [ ] 最终结果必须以 `'/'` 开头：`'/' + '/'.join(stack)`，空栈会返回 `"/"` 即根目录
- [ ] 不要把多级目录名拼接成不带 `/` 的路径

这道题是栈的一个经典应用场景。Unix 路径的 `..` 表示返回上
级目录，天然对应栈的 pop 操作。
