# 52. N皇后 II

## 🎯 人话翻译

> N×N 的棋盘上放 N 个皇后，任意两个皇后不能在同一行、同一列、同一对角线上。问有多少种不同的摆法。这题只求数量，不需要输出具体摆放方案（区别于 51 题）。

## 💡 核心思想

> **回溯法逐行放置皇后，用三个集合记录已被攻击的列、对角线。**
>
> 逐行放置保证"行"不冲突，只需检查：
> 1. **列冲突**：`col` 是否已被占用
> 2. **主对角线冲突**（左上→右下）：同一条主对角线上 `row - col` 为定值
> 3. **副对角线冲突**（右上→左下）：同一条副对角线上 `row + col` 为定值
>
> 每行在可选列中尝试放置，放置后递归下一行，递归返回时撤销（回溯）。

用到：[[回溯]]、[[DFS]]

## 📝 解题步骤

1. **初始化** — 三个 `set`：`cols`（已占列）、`diag1`（已占主对角线，key=row-col）、`diag2`（已占副对角线，key=row+col）；计数器 `count = 0`
2. **定义回溯函数 `backtrack(row)`**：
   - **终止条件**：`row == n`，所有行都放置成功，`count += 1`，返回
   - **遍历当前行的每一列** `col in 0..n-1`：
     - 如果 `col` 在 `cols` 中，或 `row-col` 在 `diag1` 中，或 `row+col` 在 `diag2` 中 → 跳过
     - **放置皇后**：将 `col`, `row-col`, `row+col` 加入三个集合
     - **递归**：`backtrack(row + 1)`
     - **回溯**：从三个集合中移除 `col`, `row-col`, `row+col`
3. **启动** — `backtrack(0)`
4. **返回 count**

## 💻 解法

> 逐行回溯 + 三个集合 O(1) 判断冲突。

```python
class Solution:
    def totalNQueens(self, n: int) -> int:
        # 时间复杂度 O(n!)，空间复杂度 O(n)
        cols = set()       # 已占用的列
        diag1 = set()      # 主对角线（左上→右下），key = row - col
        diag2 = set()      # 副对角线（右上→左下），key = row + col
        count = 0

        def backtrack(row: int):
            nonlocal count

            # 终止条件：所有行都放置完毕
            if row == n:
                count += 1
                return

            # 尝试当前行的每一列
            for col in range(n):
                d1 = row - col
                d2 = row + col

                # 冲突检查
                if col in cols or d1 in diag1 or d2 in diag2:
                    continue

                # 放置皇后
                cols.add(col)
                diag1.add(d1)
                diag2.add(d2)

                # 递归下一行
                backtrack(row + 1)

                # 回溯：撤销放置
                cols.remove(col)
                diag1.remove(d1)
                diag2.remove(d2)

        backtrack(0)
        return count
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(n!) | 第一行 n 种选择，第二行 ~n-2 种，实际复杂度低于 n! |
| 空间 | O(n) | 递归栈深度 n，三个集合各最多 n 个元素 |

## ⚠ 易错点

- [ ] **对角线 key 的计算**：主对角线 `row - col`（同一条上差值恒定），副对角线 `row + col`（同一条上和值恒定）。搞反了会导致约束失效
- [ ] **使用 set 而非 list** — 三个集合的 add/remove 都是 O(1)，不要用 `cols[:i] + cols[i+1:]` 这种 O(n) 操作
- [ ] **Python 的 nonlocal** — 在嵌套函数中修改外部变量 `count` 需要 `nonlocal count` 声明，否则会报 `UnboundLocalError`
- [ ] **回溯时记得 remove** — `cols.add(col)` 后递归，递归返回后必须 `cols.remove(col)`，否则状态污染同行的后续列尝试
- [ ] **N=1 的边界** — 1×1 棋盘有一个解（放 (0,0)），不要返回 0
- [ ] 区别于 51. N 皇后 — 51 要输出具体方案（需要维护棋盘状态），52 只输出数量（只需计数）

## 🎤 面试怎么说

```text
这道题是经典的回溯问题，用逐行放置的方法解决。

核心思路：按行放置皇后，保证每行只有一个皇后。
然后维护三个集合来快速判断冲突：
1. cols：已占用的列
2. diag1：主对角线，用 row - col 做 key（同一条对角线上差值恒定）
3. diag2：副对角线，用 row + col 做 key（和值恒定）

回溯过程：
- 在当前行的每一列尝试放置
- 如果三个集合都不冲突，就放置、递归到下一行
- 如果某行所有列都冲突，自动回溯到上一行换位置
- 当 row == n 时，说明找到一种有效摆放，计数 +1

时间复杂度约 O(n!)，空间 O(n)。
```
