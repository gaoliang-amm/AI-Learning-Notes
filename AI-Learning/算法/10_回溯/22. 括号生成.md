# 22. 括号生成

## 🎯 人话翻译

> 给定 n 对括号，生成所有合法的括号组合。合法 = 每个右括号前面都有对应的左括号，且左右括号数量相等。

## 💡 核心思想

> **回溯法：在每一步有两种选择——加左括号或加右括号，但受两个约束：**
> 1. 左括号数量 < n 时，才能加左括号
> 2. 右括号数量 < 左括号数量时，才能加右括号（保证不会出现 `())` 这种右括号比左括号多的情况）
>
> 本质是在构建一个长度为 2n 的字符串，每个位置做选择。

用到：[[回溯]]、[[DFS]]

## 📝 解题步骤

1. **初始化** — `result = []`，回溯函数参数：当前字符串 `path`，已用左括号数 `left`，已用右括号数 `right`
2. **终止条件** — `len(path) == 2 * n` 时，括号用完了，将 `path` 加入结果
3. **选择左括号** — 如果 `left < n`：`path + '('`，递归 `backtrack(path+'(', left+1, right)`
4. **选择右括号** — 如果 `right < left`：`path + ')'`，递归 `backtrack(path+')', left, right+1)`
5. **返回结果**

## 💻 解法

> 回溯：在每一步根据 left 和 right 数量判断可做的选择。

```python
from typing import List


class Solution:
    def generateParenthesis(self, n: int) -> List[str]:
        # 时间复杂度 O(4^n / sqrt(n))（第 n 个卡特兰数），空间复杂度 O(n)
        result = []

        def backtrack(path: str, left: int, right: int):
            # 终止条件：已经用了 n 对括号
            if len(path) == 2 * n:
                result.append(path)
                return

            # 选择 1：加左括号（只要还没用完）
            if left < n:
                backtrack(path + "(", left + 1, right)

            # 选择 2：加右括号（必须比左括号少才有效）
            if right < left:
                backtrack(path + ")", left, right + 1)

        backtrack("", 0, 0)
        return result
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(4^n / sqrt(n)) | 第 n 个卡特兰数，合法括号组合数 ~ 4^n/(n^(3/2)*sqrt(pi)) |
| 空间 | O(n) | 递归栈深度最多 2n |

## ⚠ 易错点

- [ ] `right < left` 是加右括号的条件，不是 `right < n`。写成后者会产生 `())...` 这样的非法序列
- [ ] 递归传参用 `path + "("` 而非 `path.append("(")` —— 字符串不可变，天然实现回溯，不需要显式 pop
- [ ] 终止条件是 `len(path) == 2 * n`（2n 个字符，n 个左 + n 个右），不能写成 `left == n and right == n`（虽然等价，但前者更简洁）
- [ ] 如果用列表 `path.append / path.pop` 实现，最终 `result.append("".join(path))` 必须在终止条件中先 join 再 append，否则后续 pop 会影响结果
- [ ] 这道题可以用"按长度 DP"（闭合法）做，但回溯是面试最直观的做法
