# 79. 单词搜索

## 🎯 人话翻译

> 给一个字母网格和一个单词，判断单词是否能在网格中找到——沿着上下左右相邻的格子拼出来，每个格子只能用一次。

## 💡 核心思想

> **二维回溯（DFS + 标记访问）。** 从网格中每个格子出发，尝试匹配单词的每个字符。每走一步标记当前格子为已访问（防止重复使用），向四个方向 DFS，递归返回时恢复标记（回溯）。
>
> 本质是图的 DFS，网格的每个格子是一个节点，相邻格子是边。

用到：[[回溯]]、[[DFS]]、[[图]]

## 📝 解题步骤

1. **遍历起点** — 对网格中每个位置 `(i, j)`，如果 `board[i][j] == word[0]`，从该位置开始回溯
2. **定义回溯函数 `backtrack(i, j, index)`**：
   - **终止条件**：`index == len(word)`，所有字符匹配完成，返回 `True`
   - **越界/不匹配检查**：如果出界 或 `board[i][j] != word[index]`，返回 `False`
   - **标记访问**：将当前格子暂存，并将 `board[i][j]` 改为特殊字符（如 `#`）
   - **四方向搜索**：递归搜索上下左右 `(i+1,j), (i-1,j), (i,j+1), (i,j-1)`，传递 `index + 1`
   - **回溯**：恢复 `board[i][j]` 为原字符
   - 返回四个方向中任一为 `True` 的结果
3. **任一起点返回 True 则整体返回 True**

## 🔢 手动走一遍

输入: `board = [["A","B","C","E"],["S","F","C","S"],["A","D","E","E"]]`, `word = "ABCCED"`

网格：
```
  j=0 1 2 3
i=0 A  B  C  E
i=1 S  F  C  S
i=2 A  D  E  E
```

从 (0,0) = 'A' 出发：

| 步骤 | 位置(i,j) | 当前字符 | index | word[index] | 操作 | 网格状态 |
|------|-----------|----------|-------|-------------|------|----------|
| 1 | (0,0) | A | 0 | A | 匹配，标记 #，向下 | A→# |
| 2 | (1,0) | S | 1 | B | S≠B，返回 False | - |
| 3 | (0,1) | B | 1 | B | 匹配，标记 #，向右 | B→# |
| 4 | (0,2) | C | 2 | C | 匹配，标记 #，向下 | C→# |
| 5 | (1,2) | C | 3 | C | 匹配，标记 #，向下 | C→# |
| 6 | (2,2) | E | 4 | E | 匹配，标记 #，向左 | E→# |
| 7 | (2,1) | D | 5 | D | 匹配，标记 # | D→# |

`index = 6 == len(word)` → 匹配完成，返回 `True`！

访问路径：`A(0,0) → B(0,1) → C(0,2) → C(1,2) → E(2,2) → D(2,1)`

## 💻 解法

> 二维回溯：每个格子尝试作为起点，四方向 DFS + 原地标记访问。

```python
from typing import List


class Solution:
    def exist(self, board: List[List[str]], word: str) -> bool:
        # 时间复杂度 O(m*n*4^L)（L=len(word)），空间复杂度 O(L)（递归栈）
        m, n = len(board), len(board[0])

        def backtrack(i: int, j: int, index: int) -> bool:
            # 终止条件：所有字符都匹配完成
            if index == len(word):
                return True

            # 越界检查 + 字符不匹配（包括已被标记的 #）
            if i < 0 or i >= m or j < 0 or j >= n or board[i][j] != word[index]:
                return False

            # 标记当前格子为已访问（原地修改，省去 visited 矩阵）
            temp = board[i][j]
            board[i][j] = "#"

            # 四个方向搜索
            found = (
                backtrack(i + 1, j, index + 1) or  # 下
                backtrack(i - 1, j, index + 1) or  # 上
                backtrack(i, j + 1, index + 1) or  # 右
                backtrack(i, j - 1, index + 1)     # 左
            )

            # 回溯：恢复格子
            board[i][j] = temp

            return found

        # 遍历每个格子作为起点
        for i in range(m):
            for j in range(n):
                if board[i][j] == word[0] and backtrack(i, j, 0):
                    return True

        return False
```

## ⏱ 复杂度

| 项目 | 复杂度 | 说明 |
|------|--------|------|
| 时间 | O(m × n × 4^L) | 最多 m×n 个起点，每个起点 DFS 最多走 4^L 条路径（L=单词长度） |
| 空间 | O(L) | 递归栈深度最多 L 层，原地标记无需 visited 矩阵 |

## ⚠ 易错点

- [ ] **必须原地标记后再恢复** — 用 `board[i][j] = '#'` 标记已访问，回溯后恢复。不能只用 visited 矩阵判断（重复格子场景会出错），也不能忘记恢复（后续搜索找不到这个格子）
- [ ] **四方向的或运算短路** — `backtrack(下) or backtrack(上) or ...` 利用了 Python 的短路求值，一旦某个方向返回 True 就不再搜索其他方向，节省时间
- [ ] **起点遍历** — 不能只从一个格子开始，必须遍历整个网格找匹配首字符的格子作为起点
- [ ] **提前剪枝** — 在遍历起点时先检查 `board[i][j] == word[0]`，避免不必要的 DFS 调用
- [ ] **越界检查顺序** — 越界检查要在字符匹配检查之前（或同一条），否则 `board[-1][0]` 可能取到最后一行导致错误

## 🎤 面试怎么说

```text
这道题是二维网格上的回溯/DFS 问题。

核心思路：从每个格子出发，尝试匹配单词的每个字符。
回溯函数检查当前位置是否越界、字符是否匹配。
匹配后标记当前格子为已访问（可以用原地修改 board[i][j] = '#'），
然后向上下左右四个方向递归搜索下一个字符。
递归返回后恢复标记（回溯）。

如果某条路径成功匹配到单词末尾，返回 True。
所有起点都尝试失败则返回 False。

时间复杂度 O(m×n×4^L)，L 是单词长度。
空间 O(L) 递归栈。

可以优化的点：
- 先统计 board 和 word 的字符频率，如果 word 中有 board 不存在的字符直接返回 False
- 如果 word 的最后一个字符在 board 中出现的次数少于第一个字符，可以反向搜索（从尾到头）
```
