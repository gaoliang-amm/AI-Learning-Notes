## 一、高等数学

### 1.1 极限

**概念**：函数在自变量无限趋近某值时，函数值趋近的值。极限是微积分的基石——连续性和导数都通过极限来定义。

$$\lim_{x \to a} f(x) = L$$

**例子**：$f(x) = x^2$，当 $x \to 2$ 时，$f(x) \to 4$，所以 $\lim_{x \to 2} x^2 = 4$。

**极限运算法则**（设 $\lim f(x) = A$，$\lim g(x) = B$）：
- 加减：$\lim[f(x) \pm g(x)] = A \pm B$
- 乘积：$\lim[f(x) \cdot g(x)] = A \cdot B$
- 商：$\lim \frac{f(x)}{g(x)} = \frac{A}{B}$（$B \neq 0$）

### 1.2 导数

**概念**：导数（derivative）是函数在某一点附近的变化率，本质是**局部线性逼近**。几何上是切线斜率。

$$f'(x_0) = \lim_{\Delta x \to 0} \frac{f(x_0 + \Delta x) - f(x_0)}{\Delta x}$$

> **直觉**：运动学中，位移对时间的导数 = 瞬时速度。

**常用函数导数表**：

| 函数 | 导数 |
|------|------|
| 常数 $c$ | $0$ |
| 幂函数 $x^n$ | $nx^{n-1}$ |
| 指数函数 $e^x$ | $e^x$ |
| 指数函数 $a^x$ | $a^x \ln a$ |
| 对数函数 $\ln x$ | $\frac{1}{x}$ |
| 对数函数 $\log_a x$ | $\frac{1}{x \ln a}$ |
| $\sin x$ | $\cos x$ |
| $\cos x$ | $-\sin x$ |
| $\tan x$ | $\sec^2 x$ |
| $\cot x$ | $-\csc^2 x$ |

**求导法则**：

| 法则 | 公式 |
|------|------|
| 和差 | $(f \pm g)' = f' \pm g'$ |
| 乘积 | $(fg)' = f'g + fg'$ |
| 商 | $\left(\frac{f}{g}\right)' = \frac{f'g - fg'}{g^2}$ |
| 链式法则（复合函数） | $\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx}$ |

> **商求导口诀**：上导下不导，减去上不导下导，分母平方照抄。

**复合函数求导示例**：$y = \sin(2x)$，令 $u = 2x$，则 $y' = \cos(2x) \cdot 2 = 2\cos(2x)$。

**利用导数求极值**：
- 导数为零的点称为**驻点**（极值可疑点）
- 驻点不一定是极值点：如 $f(x) = x^3$ 在 $x=0$ 处导数为0，但无极值

### 1.3 二阶导数

**概念**：二阶导数是导数的导数，描述**变化率的变化快慢**。

$$f''(x) = \frac{d^2y}{dx^2}$$

> **直觉**：位移对时间的二阶导数 = 瞬时加速度（速度随时间的变化率）。

**二阶导数与凹凸性**：

| 条件 | 图像特征 | 名称 |
|------|---------|------|
| $f''(x) > 0$ | 向上弯（开口向上） | 下凸函数（凸函数） |
| $f''(x) < 0$ | 向下弯（开口向下） | 上凸函数（凹函数） |

**拐点**：二阶导数在某点左右异号，曲线弯曲方向改变的点。若二阶导数连续，则拐点处 $f''(x) = 0$，但反之不成立（如 $f(x) = x^4$ 在 $x=0$ 处 $f''=0$ 但无拐点）。

**极值判定（二阶导数测试）**：

| 条件 | 结论 |
|------|------|
| $f'(x_0) = 0$，$f''(x_0) < 0$ | $x_0$ 处取得**极大值** |
| $f'(x_0) = 0$，$f''(x_0) > 0$ | $x_0$ 处取得**极小值** |
| $f'(x_0) = 0$，$f''(x_0) = 0$ | 可能是拐点，需进一步判断 |

> **注意区分**：函数值 $f(x)$ 决定曲线在 x 轴上/下方；二阶导数 $f''(x)$ 决定曲线弯曲方向，两者无关。例如 $y = -x^2 + 10$ 在 $(-2,2)$ 区间内 $y > 0$（上方），但 $f'' = -2 < 0$（向下弯）。

### 1.4 偏导与梯度

**偏导数**：多元函数中，将其他变量视为常数，对某一个变量求导。

$$\frac{\partial f}{\partial x_1} = \lim_{\Delta x_1 \to 0} \frac{f(x_1 + \Delta x_1, x_2, \ldots, x_n) - f(x_1, x_2, \ldots, x_n)}{\Delta x_1}$$

**梯度**：由所有偏导数组成的向量，方向是函数增长最快的方向。

$$\nabla f = \left(\frac{\partial f}{\partial x_1}, \frac{\partial f}{\partial x_2}, \ldots, \frac{\partial f}{\partial x_n}\right)$$

> **直觉**：梯度方向 = 最陡上坡方向；负梯度方向 = 最陡下坡方向（**梯度下降法**的核心原理）。

## 二、线性代数

### 2.1 标量、向量与范数

| 维度 | 名称 | 直觉 | 示例 |
|------|------|------|------|
| 0维 | 标量（Scalar） | 单个数值 | 温度 25°C |
| 1维 | 向量（Vector） | 一维数组，有大小有方向 | 特征向量 $[1.2, 3.5, 0.8]$ |
| 2维 | 矩阵（Matrix） | 二维表格 | 数据集矩阵 |
| N维 | 张量（Tensor） | N维数组 | 批次图像 $[B, C, H, W]$ |

**向量转置**：列向量变行向量，记为 $x^T$。

**向量运算**：
- **加法**：同维度对应元素相加
- **标量乘**：标量乘每个元素
- **内积（点积）**：$\langle x, y \rangle = \sum_{i=1}^n x_i y_i$，结果为标量

**内积与夹角关系**：
- $\langle x, y \rangle > 0$：锐角
- $\langle x, y \rangle = 0$：正交（垂直）
- $\langle x, y \rangle < 0$：钝角

#### 向量范数

范数是将向量映射为非负实数的函数，衡量向量的"大小"。

**三大范数**：

| 范数 | 定义 | 直觉 | 应用 |
|------|------|------|------|
| L0 | 非零元素个数 | 稀疏度 | 特征选择 |
| L1 | $\sum \|x_i\|$（绝对值之和） | 曼哈顿距离（只能横竖走） | 稀疏正则化，对异常值鲁棒 |
| L2 | $\sqrt{\sum x_i^2}$（平方和开根） | 欧几里得距离（直线距离） | 最常用，正则化 |
| L∞ | $\max\|x_i\|$（最大绝对值） | 切比雪夫距离 | 强化学习中常用 |

**Lp范数（统一形式）**：

$$\|x\|_p = \left(\sum_{i=1}^n |x_i|^p\right)^{1/p}$$

**numpy 计算**：

```python
import numpy as np
x = np.array([0, 2, -1])
np.linalg.norm(x, ord=0)   # L0 = 2
np.linalg.norm(x, ord=1)   # L1 = 3
np.linalg.norm(x)           # L2 = √5 ≈ 2.236
np.linalg.norm(x, ord=np.inf)  # L∞ = 2
```

### 2.2 矩阵

**特殊矩阵**：
- **方阵**：行数 = 列数
- **对角矩阵**：主对角线以外全为0
- **单位矩阵 $I$**：主对角线全为1的对角矩阵，$AI = IA = A$

**矩阵转置**：按主对角线翻转，行变列、列变行，记为 $A^T$。

性质：
- $(A^T)^T = A$
- $(A + B)^T = A^T + B^T$
- $(kA)^T = kA^T$
- $(AB)^T = B^T A^T$

**矩阵乘法**：$A_{m \times n} \cdot B_{n \times p} = C_{m \times p}$，要求 A 的列数 = B 的行数。

性质：满足**结合律**和**分配律**，**不满足交换律**（$AB \neq BA$）。

**矩阵的逆**：对于方阵 $A$，若存在 $A^{-1}$ 使得 $AA^{-1} = A^{-1}A = I$，则 $A^{-1}$ 为 $A$ 的逆矩阵。

### 2.3 其他矩阵运算

**向量化**：将矩阵元素按列排成 $mn \times 1$ 的向量，记为 $\text{vec}(A)$。

**内积**：$\langle A, B \rangle = \sum_{i,j} a_{ij} b_{ij}$，结果为标量。

**Hadamard积（逐元素积）**：$A \odot B$，对应元素相乘，结果仍为 $m \times n$ 矩阵。

**Kronecker积（直积/张量积）**：$A \otimes B$，A 中每个元素乘以整个矩阵 B，结果为 $mp \times nq$ 矩阵。

$$A \otimes B = \begin{pmatrix} a_{11}B & a_{12}B \\ a_{21}B & a_{22}B \end{pmatrix}$$

### 2.4 矩阵求导

**本质**：对多个自变量逐个求偏导，结果排列成向量或矩阵形式。

**符号约定**：
- $x \in \mathbb{R}^m$：m维列向量
- $X \in \mathbb{R}^{m \times n}$：m行n列矩阵
- $f(x)$：标量函数（输入向量，输出标量）
- $f(X)$：标量函数（输入矩阵，输出标量）
- $f(x) \in \mathbb{R}^p$：向量函数

#### 四种基本场景

| 场景 | 自变量 | 函数 | 结果形状 |
|------|--------|------|---------|
| 标量对向量求导 | $x \in \mathbb{R}^m$ | 标量 $f(x)$ | $m \times 1$（列向量） |
| 标量对矩阵求导 | $X \in \mathbb{R}^{m \times n}$ | 标量 $f(X)$ | $m \times n$（同 X） |
| 向量对向量求导 | $x \in \mathbb{R}^m$ | 向量 $f(x) \in \mathbb{R}^p$ | $p \times m$（矩阵） |
| 向量对标量求导 | 标量 $x$ | 向量 $f(x) \in \mathbb{R}^p$ | $p \times 1$（列向量） |

#### 梯度向量与梯度矩阵

**梯度向量** = 标量函数对向量求导的结果：

$$\nabla_x f = \frac{\partial f}{\partial x} = \begin{pmatrix} \frac{\partial f}{\partial x_1} \\ \vdots \\ \frac{\partial f}{\partial x_m} \end{pmatrix}$$

**梯度矩阵** = 标量函数对矩阵求导的结果，形状与矩阵相同。

#### 常用求导公式速查表

**向量对向量求导**（$x$ 为 $n$ 维列向量）：

| 公式 | 含义 | 结果 |
|------|------|------|
| $\frac{\partial (a^T x)}{\partial x} = a$ | 线性函数 | 与 $a$ 同维度的列向量 |
| $\frac{\partial \|x\|^2}{\partial x} = 2x$ | L2范数平方 | $2x$ |
| $\frac{\partial (Ax)}{\partial x} = A$ | 线性变换 | 矩阵 $A$ |
| $\frac{\partial (x^T A)}{\partial x} = A^T$ | 行向量形式 | $A^T$ |
| $\frac{\partial (x^T A x)}{\partial x} = (A + A^T)x$ | 二次型 | 若 $A$ 对称则 $= 2Ax$ |

**标量对矩阵求导**（$X$ 为 $m \times n$ 矩阵）：

| 公式 | 含义 | 结果形状 |
|------|------|---------|
| $\frac{\partial (a^T X b)}{\partial X} = ab^T$ | 双线性形式 | $m \times n$ |
| $\frac{\partial (b^T X^T a)}{\partial X} = ab^T$ | 转置形式 | $m \times n$ |
| $\frac{\partial (a^T X^T X b)}{\partial X} = X(ab^T + ba^T)$ | 二次型 | $m \times n$ |
| $\frac{\partial (a^T X^T X a)}{\partial X} = 2Xaa^T$ | 对称简化 | $m \times n$ |

### 2.5 黑塞矩阵（Hessian Matrix）

**概念**：由函数所有二阶偏导数构成的矩阵，描述函数的**曲率（凹凸性）**。

$$H = \begin{pmatrix} \frac{\partial^2 f}{\partial x_1^2} & \frac{\partial^2 f}{\partial x_1 \partial x_2} \\ \frac{\partial^2 f}{\partial x_2 \partial x_1} & \frac{\partial^2 f}{\partial x_2^2} \end{pmatrix}$$

**凹凸性判定**：

| 黑塞矩阵性质 | 函数性质 | 优化意义 |
|-------------|---------|---------|
| 半正定 | 凸函数 | 局部极小 = 全局最小 |
| 负半正定 | 凹函数 | 局部极大 = 全局最大 |
| 不定 | 非凸 | 存在鞍点 |

**示例**：$f(x_1, x_2) = x_1^2 + x_1 x_2 - x_2^2$

- 一阶偏导：$\frac{\partial f}{\partial x_1} = 2x_1 + x_2$，$\frac{\partial f}{\partial x_2} = x_1 - 2x_2$
- 二阶偏导：$H = \begin{pmatrix} 2 & 1 \\ 1 & -2 \end{pmatrix}$
- 1阶主子式：$2 > 0$；2阶行列式：$2 \times (-2) - 1 \times 1 = -5 < 0$
- 黑塞矩阵**不定**，函数非凸，存在鞍点

## 三、概率论

### 3.1 积分基础（概率论前置）

**不定积分**：求导的逆运算。若 $F'(x) = f(x)$，则 $F(x)$ 是 $f(x)$ 的一个原函数，记为 $\int f(x)dx = F(x) + C$。

**定积分**：计算函数在区间 $[a, b]$ 上的"面积"，记为 $\int_a^b f(x)dx$。

> **直觉**：定积分就是求曲线下方的面积。概率密度函数（PDF）在某个区间上的定积分 = 该区间内事件发生的概率。

**常用积分公式**：

| 函数 | 不定积分 |
|------|---------|
| $x^n$ | $\frac{x^{n+1}}{n+1} + C$（$n \neq -1$） |
| $e^x$ | $e^x + C$ |
| $\frac{1}{x}$ | $\ln\|x\| + C$ |

**PDF 与 CDF 的关系**：

- **PDF（概率密度函数）**$f(x)$：描述概率分布的"形状"，$f(x) \geq 0$，$\int_{-\infty}^{+\infty} f(x)dx = 1$
- **CDF（累积分布函数）**$F(x)$：$F(x) = P(X \leq x) = \int_{-\infty}^{x} f(t)dt$

$$F(x) = \int_{-\infty}^{x} f(t)dt, \quad f(x) = F'(x)$$

> **直觉**：PDF 是"概率密度"（某个点的概率为 0，但区间有概率），CDF 是"累积概率"（从负无穷累加到 x）。知道一个就能求另一个。

### 3.2 方差、标准差、协方差

**方差**：衡量单组数据离散程度，是各数据到均值的平方距离的平均值。

$$\text{Var}(X) = \frac{1}{n} \sum_{i=1}^n (x_i - \bar{x})^2$$

- 方差越小 → 数据越集中，波动越小
- 方差越大 → 数据越分散，波动越大

**标准差**：方差的算术平方根，**统一量纲**（方差单位是原数据单位的平方）。

$$\sigma = \sqrt{\text{Var}(X)}$$

**协方差**：衡量两个变量之间的**线性相关程度**。

$$\text{Cov}(X, Y) = \frac{1}{n} \sum_{i=1}^n (x_i - \bar{x})(y_i - \bar{y})$$

| 符号 | 含义 |
|------|------|
| $\text{Cov} > 0$ | 正相关（$X$ 增大，$Y$ 增大） |
| $\text{Cov} < 0$ | 负相关（$X$ 增大，$Y$ 减小） |
| $\text{Cov} = 0$ | 无线性相关 |

### 3.3 概率基础

| 事件 | 公式 |
|------|------|
| A 的概率 | $P(A)$ |
| 非 A | $P(\bar{A}) = 1 - P(A)$ |
| 联合概率 | $P(A \cap B) = P(A)P(B \mid A)$ |
| 并集 | $P(A \cup B) = P(A) + P(B) - P(A \cap B)$ |
| 条件概率 | $P(A \mid B) = \frac{P(A \cap B)}{P(B)}$ |

### 3.4 概率分布

**概率分布**描述随机变量取值的概率规律。

#### 均匀分布

$U(a, b)$：在 $[a, b]$ 区间上等可能取值。

$$f(x) = \begin{cases} \frac{1}{b-a} & a \leq x \leq b \\ 0 & \text{其他} \end{cases}$$

#### 正态分布（高斯分布）

最重要的连续概率分布，钟形曲线。记为 $X \sim N(\mu, \sigma^2)$。

$$f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$$

- $\mu$（均值）：位置参数，决定分布中心
- $\sigma^2$（方差）：尺度参数，决定分布宽度
- **标准正态分布**：$\mu = 0$，$\sigma = 1$

> **中心极限定理**：大量独立同分布随机变量的均值，随样本量增加收敛于正态分布。这是正态分布无处不在的根本原因。

### 3.5 全概率公式

将复杂事件分解为一组互斥且完备的简单事件，再合并求解。

设 $A_1, A_2, \ldots, A_n$ 为完备事件组（互斥且 $\sum P(A_i) = 1$），则：

$$P(B) = \sum_{i=1}^n P(A_i) P(B \mid A_i)$$

**工厂案例**：3条生产线，产量占比 40%/35%/25%，次品率 2%/1%/3%。

$$P(\text{次品}) = 0.4 \times 0.02 + 0.35 \times 0.01 + 0.25 \times 0.03 = 0.019 = 1.9\%$$

### 3.6 贝叶斯公式

**核心思想**：从结果反推原因——在已知 $B$ 发生的条件下，更新对 $A$ 发生概率的估计。

$$P(A \mid B) = \frac{P(B \mid A) \cdot P(A)}{P(B)} = \frac{P(B \mid A) \cdot P(A)}{\sum_{i} P(A_i) P(B \mid A_i)}$$

| 术语 | 符号 | 含义 |
|------|------|------|
| 后验概率 | $P(A \mid B)$ | 观测到 B 后更新的 A 的概率 |
| 似然概率 | $P(B \mid A)$ | A 成立时观测到 B 的概率 |
| 先验概率 | $P(A)$ | 观测前对 A 的认知 |
| 证据 | $P(B)$ | B 发生的总概率（全概率公式计算） |

**疾病检测案例**：
- 发病率 $P(\text{病}) = 1\%$（先验）
- 有病时检测阳性率 $P(\text{阳性} \mid \text{病}) = 95\%$
- 无病时检测阳性率 $P(\text{阳性} \mid \text{无病}) = 5\%$

$$P(\text{病} \mid \text{阳性}) = \frac{0.95 \times 0.01}{0.95 \times 0.01 + 0.05 \times 0.99} = \frac{0.0095}{0.0095 + 0.0495} \approx 16.1\%$$

> **直觉**：即使检测呈阳性，真正患病概率仅 16.1%。因为基数效应——健康人占 99%，即使 5% 误报，绝对人数也远超真正的患者。

### 3.7 似然函数

**概率 vs 似然**：
- **概率**：已知原因，猜结果（参数 → 数据）
- **似然**：已知结果，猜原因（数据 → 参数）

> **硬币案例**：
> - 概率：已知 $p=0.5$，问抛出正面的可能性多大？
> - 似然：已知抛出正面，问 $p=0.1$ 和 $p=0.9$ 哪个更合理？

**似然函数定义**：给定观测数据 $D$，似然函数是关于参数 $\theta$ 的函数：

$$L(\theta) = P(D \mid \theta)$$

对于独立同分布数据 $x_1, x_2, \ldots, x_n$：

$$L(\theta) = \prod_{i=1}^n P(x_i \mid \theta)$$

取对数（连乘变连加，计算更稳定）：

$$\log L(\theta) = \sum_{i=1}^n \log P(x_i \mid \theta)$$

### 3.8 极大似然估计（MLE）

**目标**：找到使似然函数最大化的参数 $\hat{\theta}$——即最可能生成观测数据的参数值。

$$\hat{\theta}_{MLE} = \arg\max_\theta L(\theta) = \arg\max_\theta \log L(\theta)$$

**掷硬币案例**：3次投掷，2正1反，求正面概率 $p$ 的 MLE。

似然函数：$L(p) = p^2(1-p)^1$

取对数：$\log L = 2\log p + \log(1-p)$

求导令其为0：

$$\frac{d \log L}{dp} = \frac{2}{p} - \frac{1}{1-p} = 0 \implies \hat{p} = \frac{2}{3}$$

> **直觉**：3次中2次正面，最可能的正面概率就是 $\frac{2}{3}$——MLE 本质上就是"用观测频率估计概率"。

## 四、面试高频问题

### Q1：梯度下降的数学原理是什么？

梯度方向是函数增长最快的方向，负梯度方向是下降最快的方向。参数更新公式 $\theta = \theta - \alpha \nabla f(\theta)$，其中 $\alpha$ 为学习率。迭代执行直到收敛到（局部）极小值。

### Q2：L1 和 L2 正则化有什么区别？

- **L1 正则化**（Lasso）：$\|w\|_1$，倾向于产生**稀疏**权重（部分权重为0），可做特征选择
- **L2 正则化**（Ridge）：$\|w\|_2^2$，倾向于让权重**均匀缩小**，防止过拟合但不会产生稀疏解

直觉：L1 的等高线是菱形，容易与损失函数等高线在坐标轴上相交（稀疏）；L2 的等高线是圆形，交点一般不在坐标轴上。

### Q3：贝叶斯定理在机器学习中有哪些应用？

- **朴素贝叶斯分类器**：利用贝叶斯公式计算后验概率进行分类
- **贝叶斯优化**：超参数搜索，用先验+观测更新对目标函数的估计
- **贝叶斯神经网络**：对权重做概率建模，量化不确定性
- **MAP 估计**：极大后验估计 = MLE + 先验

### Q4：什么是极大似然估计？和最小二乘法的关系？

MLE 找最可能生成数据的参数。当假设噪声服从高斯分布时，MLE 等价于最小二乘法（对数似然最大化 $\iff$ 均方误差最小化）。两者在高斯假设下数学上等价。

### Q5：黑塞矩阵在优化中的作用？

黑塞矩阵描述函数曲率，用于：
- **牛顿法**：利用黑塞矩阵加速收敛（比梯度下降更快）
- **判断凸性**：半正定 → 凸函数（局部极小 = 全局最小）
- **鞍点检测**：不定矩阵 → 存在鞍点

### Q6：为什么深度学习中常用随机梯度下降（SGD）而非牛顿法？

牛顿法需要计算和求逆黑塞矩阵（$O(n^2)$ 存储，$O(n^3)$ 求逆），参数量达百万级时不可行。SGD 每次只需一个（或一小批）样本的梯度，计算量小，且噪声有助于跳出局部极小。Adam 等自适应学习率方法是对 SGD 的改进。

### Q7：中心极限定理的实际意义是什么？

无论原始数据服从什么分布，大量样本的均值趋近正态分布。这使得我们可以用正态分布来构造置信区间、做假设检验，也是很多统计方法的理论基础。

## 五、背诵版总结

**高等数学**：极限是微积分基石；导数是局部变化率，用于求极值；二阶导数判断凹凸性；偏导+梯度指向函数增长最快方向，负梯度驱动梯度下降。

**线性代数**：向量/矩阵/张量是数据的基本载体；范数衡量向量大小（L1稀疏、L2常用）；矩阵乘法不满足交换律；矩阵求导本质是逐元素求偏导并排列成矩阵；黑塞矩阵判断函数凸性。

**概率论**：积分是微积分的另一基石，定积分求曲线下面积，PDF 的积分 = CDF；方差衡量离散度，协方差衡量相关性；全概率公式分解复杂事件，贝叶斯公式从结果反推原因；似然函数是"已知结果猜原因"，MLE 找最可能的参数值。
