# 扩散模型与文生图

## 一、扩散模型通俗讲解

### 1.1 正向扩散：逐步加噪

将图像逐步添加高斯噪声，直到变成纯噪声。每一步从均值为0、方差较小的高斯分布中采样，加到每个像素上。经过数百步迭代后，图像无法与纯噪声区分。

关键特性：**容易**，确定性的马尔可夫链过程。

### 1.2 反向扩散：逐步去噪

训练神经网络预测每一步添加的噪声（而非预测去噪后的图像），从 $x_t$ 中减去预测噪声即可去噪。预测噪声比重构图像更简单，是实际采用的方案。

### 1.3 DDPM核心思想

- **共享权重**：所有时间步使用同一个 U-Net，通过时间步编码区分不同步骤
- **迭代次数**：通常 50~1000 步，越多相邻步差异越小、学习越容易，但计算成本更高
- **方差计划**：$\beta_{start}=0.0001$，$\beta_{end}=0.02$，线性递增

## 二、扩散模型数学理论

### 2.1 前向过程的解析解

无需逐步迭代，可直接通过闭式公式从 $x_0$ 采样任意时间步 $x_t$：

$$x_t = \sqrt{\bar{\alpha}_t} x_0 + \sqrt{1 - \bar{\alpha}_t} \epsilon, \quad \epsilon \sim \mathcal{N}(0, I)$$

其中 $\bar{\alpha}_t = \prod_{s=1}^{t} \alpha_s$，$\alpha_t = 1 - \beta_t$。推导利用了重参数化技巧和两个独立正态分布之和仍为正态分布的性质。

### 2.2 变分推断与ELBO

目标：最大化 $\log p_\theta(x_0)$。利用 Jensen 不等式得到证据下界（ELBO）：

$$\log p_\theta(x_0) \geq \mathbb{E}_{q(x_0 \cdots x_T)} \left[ \log \frac{p_\theta(x_0 \cdots x_T)}{q(x_1 \cdots x_T | x_0)} \right]$$

展开后分解为三项：$L_T$（常数可忽略）+ $\sum L_{t-1}$（KL散度项）+ $L_0$（重建项）。由于前向转移概率是高斯分布且 $\beta_t$ 充分小，反向转移概率也是高斯分布（Feller定理）。

$$q(x_{t-1} | x_t, x_0) = \mathcal{N}(x_{t-1}; \tilde{\mu}_t(x_t, x_0), \tilde{\beta}_t I)$$

### 2.3 简化损失函数

忽略权重系数，最终简化为：

$$L_{simple} = \mathbb{E}_{t, x_0, \epsilon} \left[ \| \epsilon - \epsilon_\theta(x_t, t) \|^2 \right]$$

即神经网络预测噪声与真实噪声的 MSE 损失。反向采样迭代公式：

$$x_{t-1} = \frac{1}{\sqrt{\alpha_t}} \left( x_t - \frac{1-\alpha_t}{\sqrt{1-\bar{\alpha}_t}} \epsilon_\theta(x_t, t) \right) + \sigma_t z$$

## 三、U-Net与实现

### 3.1 U-Net架构

输入输出分辨率相同，三部分组成：

| 组件 | 操作 | 作用 |
|------|------|------|
| 下采样 | Conv + MaxPool | 逐步缩小特征图，提取高层特征 |
| 瓶颈层 | Conv | 压缩到潜在空间 |
| 上采样 | Upsample + Conv | 逐步恢复分辨率 |
| 跳跃连接 | `torch.cat` | 保留空间细节信息 |

在扩散模型中，输入输出通道数均为 C（与输入图像一致）。

### 3.2 时间步编码（正弦位置编码）

将整数时间步 $t$ 编码为向量，偶数维度用 sin，奇数维度用 cos：

$$v_i = \begin{cases} \sin(t / 10000^{i/D}), & i \text{ 为偶数} \\ \cos(t / 10000^{i/D}), & i \text{ 为奇数} \end{cases}$$

编码后通过 MLP 变换为 $(N, C, 1, 1)$ 形状，与特征图逐元素相加。

### 3.3 训练与采样流程

**训练**：随机采样时间步 $t$ -> 对 $x_0$ 加噪得到 $x_t$ -> U-Net 预测噪声 -> MSE 反向传播

**采样**：从 $x_T \sim \mathcal{N}(0, I)$ 出发，循环 $t = T \to 1$，每步用 U-Net 预测噪声并去噪，最终输出 $x_0$

## 四、条件扩散模型

### 4.1 得分函数与Classifier Guidance

得分函数定义为 $s(x) = \nabla_x \log p(x)$，表示概率密度增长最快的方向。噪声与得分的关系：

$$\epsilon \approx -\sqrt{1-\bar{\alpha}_t} \nabla_{x_t} \log p(x_t)$$

**Classifier Guidance**：用预训练分类器引导生成，条件得分分解为：

$$\nabla_{x_t} \log p(x_t | y) = \nabla_{x_t} \log p(x_t) + \gamma \nabla_{x_t} \log p(y | x_t)$$

$\gamma$ 控制条件引导强度。缺点：需要单独训练分类器。

### 4.2 Classifier-Free Guidance（无分类器指引）

不需要额外分类器，使用同一模型同时学习有条件和无条件得分。训练时以一定概率（如10%）将条件置为 $\emptyset$（零向量），推理时：

$$\epsilon_\theta(x_t, t, y) \approx \epsilon_\theta(x_t, t, \emptyset) + \gamma \left[ \epsilon_\theta(x_t, t, y) - \epsilon_\theta(x_t, t, \emptyset) \right]$$

$\gamma$ 越大，生成结果越符合提示词。`$\emptyset$` 位置也可填入反向提示词（negative prompt）。

### 4.3 条件注入方式

条件 $y$（如类别标签）通过 Embedding 层转为向量，与时间步编码相加后送入 U-Net：

```python
# 条件嵌入与时间步嵌入相加
t = pos_encoding(timesteps, time_embed_dim)
if labels is not None:
    t += self.label_emb(labels)  # nn.Embedding(num_labels, time_embed_dim)
```

### 4.4 Stable Diffusion简介

Stable Diffusion 是在**潜在空间（Latent Space）**中运行的扩散模型。

**核心思想**：传统扩散模型直接在像素空间做加噪/去噪，计算量与图像分辨率的平方成正比（如 512x512 图像有 262,144 个像素）。Stable Diffusion 引入 VAE（变分自编码器）将图像压缩到低维的潜在表示（如 64x64x4），再在潜在空间中进行扩散过程，最后用 VAE 解码器还原为高分辨率图像。这使得计算量大幅降低，同时保持生成质量。

```
编码器(VAE)          潜在空间扩散          解码器(VAE)
原图 512x512x3 → 压缩 → 潜在表示 64x64x4 → 扩散 → 解码 → 生成图 512x512x3
```

代码和预训练权重开源，可生成高清图像。

## 五、DALL-E2：文生图三模型

DALL-E2 (unCLIP) 通过三个模型协同实现文本到图像生成：

### 5.1 CLIP模型（冻结）

用图文对训练，包含图像编码器（ViT）和文本编码器（Transformer），输出同一嵌入空间的向量。训练完成后**冻结**，后续模型复用。

### 5.2 先验模型 Prior（Decoder-Only Transformer）

**目的**：给定文本，预测对应的 CLIP 图像嵌入，生成高质量条件。

**训练输入**：(文本ID, CLIP文本嵌入, 随机时间步嵌入, 加噪的CLIP图像嵌入, 可学习嵌入)，加上因果注意力掩码。

**关键点**：噪声加在 **CLIP图像嵌入** 上，不是原图。使用余弦方差计划：

$$\beta_t = 1 - \frac{\alpha_t}{\alpha_{t-1}}, \quad \alpha_t = \frac{f(t)}{f(0)}, \quad f(t) = \cos\left(\frac{t/T + s}{1+s} \cdot \frac{\pi}{2}\right)^2$$

**训练细节**：
1. CLIP 图像嵌入加噪得到 noisy image embedding
2. 6 个输入拼接为序列送入 Decoder-Only Transformer（因果注意力掩码）
3. 取最后一个 learned embedding 送入 MLP，输出预测的 CLIP 图像嵌入
4. 损失为预测嵌入与真实 CLIP 图像嵌入的 MSE

**推理时**：从纯噪声嵌入出发，两次采样取与文本嵌入点积更高的结果。直接预测嵌入而非迭代去噪。

### 5.3 条件扩散模型 UNet

以 CLIP 图像嵌入（经先验模型生成）为条件，生成最终图像。

**架构增强**：
- **残差块**：GroupNorm + SiLU + Conv2d，条件嵌入通过 scale-shift 调制（乘以 scale 加上 shift，优于简单加法）
- **注意力块**：混合注意力，K/V 由输入和条件信息拼接而成
- **文本编码**：可选的 Transformer 编码器处理提示词

**训练**：冻结 CLIP + 冻结 Prior，仅训练 UNet。输入 $(x_t, t, \text{CLIP图像嵌入})$，预测噪声，MSE 损失。

**采样**：从白噪声出发，以先验模型预测的 CLIP 图像嵌入为条件，逐步去噪生成图像。

### 5.4 三模型协同流程

```
文本提示词
    ↓
[CLIP文本编码器] → CLIP文本嵌入 (冻结)
    ↓
[Prior模型] → CLIP图像嵌入 (冻结)
    ↓
[条件UNet扩散] → 生成图像 (训练)
```

三个模型依次冻结：CLIP先训先冻，Prior再训再冻，最后训UNet。

## 六、关键要点总结

| 概念 | 核心要点 |
|------|----------|
| DDPM | 正向加噪用闭式解，反向去噪训练U-Net预测噪声 |
| 损失函数 | $L_{simple} = \| \epsilon - \epsilon_\theta(x_t, t) \|^2$ |
| U-Net | 下采样+瓶颈+上采样+跳跃连接，所有时间步共享权重 |
| 时间编码 | 正弦位置编码将整数 $t$ 映射为向量 |
| Classifier Guidance | 条件得分 = 无条件得分 + $\gamma$ × 分类器梯度 |
| Classifier-Free | 同一模型学有条件/无条件，推理时插值，无需分类器 |
| DALL-E2 | CLIP(冻结) → Prior(冻结) → UNet扩散(训练) |
| Prior模型 | 对CLIP嵌入空间加噪→Transformer预测→MSE损失，推理时两次采样取点积更大者 |
| Stable Diffusion | VAE压缩图像到潜在空间 → 潜在空间做扩散 → 大幅降低计算量 |
