# 📦 review_analyze_bert — 电商评论情感分析

> 基于 **BERT 预训练模型**对电商评论做**正/负情感分类**的完整 PyTorch 项目。

---

## 🗂️ 项目结构

```
review_analyze_bert/
├── src/
│   ├── config.py       # 全局配置（路径、超参数）
│   ├── preprocess.py   # 数据预处理（清洗、编码、切分）
│   ├── dataset.py      # 数据集加载器
│   ├── model.py        # 模型定义（BERT + 分类头）
│   ├── train.py        # 训练脚本
│   ├── evaluate.py     # 评估脚本（计算准确率）
│   └── predict.py      # 预测脚本（单条 + 批量 + 交互应用）
├── data/
│   ├── raw/            # 原始 CSV 数据
│   └── processed/      # 预处理后的 HuggingFace Dataset
├── models/             # 保存训练好的模型权重
└── logs/               # TensorBoard 日志
```

---

## 🎯 任务目标

| 输入 | 输出 |
|------|------|
| 一段中文电商评论文字 | 正面（1）/ 负面（0）及置信度 |

示例：
- 输入：`"质量很好，物流也快，非常满意！"` → **正面 0.97**
- 输入：`"收到货发现破损，客服不处理"` → **负面 0.93**

## 🚀 快速运行流程

```bash
# 第一步：数据预处理（只需运行一次）
python src/preprocess.py

# 第二步：训练模型
python src/train.py

# 第三步：评估模型
python src/evaluate.py

# 第四步：交互预测
python src/predict.py
```
